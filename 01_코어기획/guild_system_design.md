# 에테르나 크로니클 — 길드 시스템

> **작성일:** 2026-03-10 | **버전:** v1.0 | **분류:** 코어기획 — 소셜/길드

---

## 1. 개요

길드는 에테르나 크로니클의 핵심 사회 시스템이다. 플레이어 간 협력, 경쟁, 공동 목표 달성을 위한 조직 프레임워크를 제공한다.

### 1.1 설계 목표

| 목표 | 설명 |
|------|------|
| **사회적 유대** | 플레이어 간 지속적 관계 형성 유도 |
| **협력 콘텐츠 관문** | 길드 전용 던전/레이드로 참여 동기 부여 |
| **경쟁 구조** | 길드 전쟁을 통한 서버 내 경쟁 생태계 |
| **성장 보상** | 길드 레벨/스킬로 개인 성장에 실질적 이득 |

## 2. 길드 생성 · 가입 · 탈퇴

### 2.1 길드 생성

| 항목 | 조건/값 |
|------|---------|
| 최소 레벨 | 30 이상 |
| 창설 비용 | 500,000 골드 |
| 길드명 규칙 | 2~16자, 한글/영문/숫자, 비속어 필터 적용 |
| 길드 태그 | 2~5자 영문 대문자 (유니크) |
| 길드 엠블럼 | 사전 정의 아이콘 50종 + 색상 조합 (추후 커스텀 업로드) |
| 길드 소개 | 최대 200자 |
| 서버당 길드 수 상한 | 없음 (자연 조절) |

**생성 플로우:**
1. 캐릭터 레벨 30+ 확인
2. 골드 500,000 차감
3. 길드명/태그 중복 검사 + 비속어 필터
4. 길드 생성 → 생성자가 자동으로 길드마스터 취임
5. 24시간 내 최소 5인 가입 실패 시 경고 (해산은 없음)

### 2.2 길드 가입

| 방식 | 설명 |
|------|------|
| **공개 가입** | 길드 설정이 "공개"일 때 즉시 가입 |
| **승인 가입** | 가입 신청 → 장교 이상 승인 |
| **초대 가입** | 장교 이상이 직접 초대 → 대상 수락 |

- **동시 가입 제한:** 1캐릭터 = 1길드
- **가입 쿨타임:** 길드 탈퇴/추방 후 **24시간** 동안 다른 길드 가입 불가
- **최대 인원:** 길드 레벨에 따라 30~100명

### 2.3 길드 탈퇴 · 추방 · 해산

| 행동 | 조건 | 페널티 |
|------|------|--------|
| **자발적 탈퇴** | 아무 때나 가능 | 24시간 가입 쿨타임, 기부 골드 미반환 |
| **추방** | 장교 이상 권한. 본인보다 낮은 직급만 가능 | 대상에게 24시간 쿨타임 |
| **길드 해산** | 길드마스터만 가능. 72시간 해산 대기 | 길드 자산 전액 소멸. 전 길드원 24시간 쿨타임 |
| **길드마스터 위임** | 길드마스터 → 부마스터 지정 | 즉시 적용 |
| **자동 위임** | 길드마스터 30일 미접속 | 부마스터 중 최고 기여도 순 자동 위임 |

## 5. 길드 전쟁

### 5.1 기본 규칙

| 항목 | 값 |
|------|-----|
| 인원 | 20 vs 20 |
| 시간 | 매주 토요일 20:00~21:00 (KST) |
| 방식 | 거점 점령전 |
| 참가 조건 | 길드 레벨 10+, 참가 신청 인원 15명 이상 |
| 매칭 | 길드 전투력 기반 (±20% 범위) |

### 5.2 전쟁 맵 · 거점

```
┌─────────────────────────────────────────┐
│                                         │
│   [A 기지]                [B 기지]       │
│      ▲                      ▲           │
│      │                      │           │
│   [거점 A1]  ←─ [중앙 거점] ─→ [거점 B1]  │
│      │              │          │        │
│   [거점 A2]  ←──────┘───────→ [거점 B2]  │
│                                         │
└─────────────────────────────────────────┘
```

- **기지 (2개):** 각 길드 리스폰 포인트. 파괴 불가
- **측면 거점 (4개):** 점령 시 버프 제공 (공격력+10% 또는 방어력+10%)
- **중앙 거점 (1개):** 점령 시 분당 +100 전쟁 포인트

### 5.3 점령 메카닉

| 항목 | 값 |
|------|-----|
| 점령 시간 | 30초 (거점 범위 내 인원 수에 따라 가속: 2인 ×1.5, 3인+ ×2.0) |
| 점령 해제 | 적 진입 시 점령 중단, 5초 후 중립화 시작 |
| 거점 체력 | 없음 (인원 기반 점령) |
| 부활 시간 | 사망 후 10초 (기지에서 부활) |

### 5.4 승리 조건 · 보상

**승리 조건:** 60분 종료 시 **전쟁 포인트** 높은 길드 승리

| 포인트 획득 | 수치 |
|------------|------|
| 적 처치 | +10 / kill |
| 거점 점령 중 | +100 / 분 (중앙), +50 / 분 (측면) |
| 전 거점 점령 (올킬) | 즉시 승리 |

**보상:**

| 구분 | 승리 | 패배 |
|------|------|------|
| 골드 | 200,000 | 50,000 |
| 길드 경험치 | 5,000 | 1,000 |
| 길드 전쟁 코인 | 100 | 30 |
| 개인 명예 포인트 | 50~200 (기여도 비례) | 10~50 |

**길드 전쟁 코인 상점:** 전쟁 전용 코스메틱, 소비 아이템, 영웅 등급 장비 구매 가능

### 5.5 PvP 밸런스 적용

길드 전쟁에도 PvP 장비 스코어 정규화가 동일하게 적용된다:
- `normalizedStat = baseStat + (gearStat × 0.3)`
- CC 지속 -50%, 힐량 -30%, 즉사기 무효

## 6. 길드 던전 · 레이드

### 6.1 길드 던전

| 항목 | 값 |
|------|-----|
| 참가 인원 | 4인 |
| 주기 | 주 1회 (월요일 초기화) |
| 입장 조건 | 길드 레벨 8+, 캐릭터 레벨 40+ |
| 난이도 | 일반 / 영웅 (길드 레벨 15+) |
| 소요 시간 | 30~45분 |

**던전 구조:**
- 3개 스테이지 + 최종 보스
- 스테이지마다 길드 고유 메카닉: "기억 수집" (스테이지 내 기억 파편 수집 → 최종 보스 약화)
- 파편 수집률에 따른 보상 등급: S(100%) / A(80%+) / B(60%+) / C(60% 미만)

**보상:**

| 등급 | 골드 | 장비 상자 | 길드 경험치 |
|------|------|-----------|------------|
| S | 100,000 | 영웅 장비 상자 ×2 | 3,000 |
| A | 70,000 | 영웅 장비 상자 ×1 | 2,000 |
| B | 40,000 | 희귀 장비 상자 ×2 | 1,500 |
| C | 20,000 | 희귀 장비 상자 ×1 | 1,000 |

### 6.2 길드 레이드

| 항목 | 값 |
|------|-----|
| 참가 인원 | 8인 |
| 주기 | 주 1회 (토요일 초기화, 길드전과 다른 시간대) |
| 입장 조건 | 길드 레벨 13+, 캐릭터 레벨 60+ |
| 난이도 | 일반 / 영웅 / 전설 (길드 레벨 18+) |
| 소요 시간 | 45~90분 |

**레이드 보스:** 지역별 순환 (6주 로테이션)

| 주차 | 보스 | 지역 | 특수 메카닉 |
|------|------|------|------------|
| 1주 | 고대의 기억 수호자 | 에레보스 | 기억 흡수 강화판 |
| 2주 | 숲의 수호정령 | 실반헤임 | 덩굴 속박 + 치유 차단 |
| 3주 | 사막의 대군주 | 솔라리스 | 지형 변환 확장판 |
| 4주 | 제국의 철기병단장 | 아르겐티움 | 다수 소환 + 진형 전투 |
| 5주 | 빙하의 감시자 | 북방영원빙원 | 빙결 + 시야 제한 |
| 6주 | 자유항의 해적왕 | 브리탈리아 | 함선 전투 + 수중 페이즈 |

**보상 (전설 난이도 기준):**
- 전설 등급 장비 재료 (세트 장비 제작용)
- 신화 등급 에테르 결정 (주간 1개 한정)
- 길드 경험치 8,000
- 골드 300,000 (8인 분배)

## 부록 A. 데이터 모델 (Prisma 스키마 초안)

```prisma
model Guild {
  id          String   @id @default(cuid())
  name        String   @unique
  tag         String   @unique
  emblem      Json     // { icon: string, color1: string, color2: string }
  description String?
  level       Int      @default(1)
  exp         Int      @default(0)
  gold        Int      @default(0)
  setting     Json     // { joinType: "open"|"approval"|"invite", ... }
  disbandAt   DateTime?
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt

  members     GuildMember[]
  skills      GuildSkill[]
  storage     GuildStorageItem[]
  quests      GuildQuest[]
  warHistory  GuildWarRecord[]
}

model GuildMember {
  id        String   @id @default(cuid())
  guildId   String
  userId    String
  rank      String   @default("recruit") // master/vice/officer/member/recruit
  joinedAt  DateTime @default(now())
  donation  Int      @default(0)

  guild     Guild    @relation(fields: [guildId], references: [id])
  @@unique([guildId, userId])
}

model GuildSkill {
  id       String @id @default(cuid())
  guildId  String
  skillKey String
  level    Int    @default(0)

  guild    Guild  @relation(fields: [guildId], references: [id])
  @@unique([guildId, skillKey])
}
```
