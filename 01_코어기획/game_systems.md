## 1. 캐릭터 스탯 시스템

### 1.1 기본 스탯 종류

#### 주요 스탯 (Primary Stats)

| 스탯 | 약어 | 설명 | 기본값 (Lv.1) |
|------|------|------|--------------|
| 생명력 | HP | 캐릭터가 받을 수 있는 총 피해량. 0이 되면 사망 | 200 |
| 마나 | MP | 스킬 사용에 필요한 자원. 일부 스킬은 HP를 소모 | 100 |
| 힘 | STR | 물리 공격력 및 무거운 장비 착용 조건에 영향 | 10 |
| 민첩 | DEX | 공격 속도, 회피율, 크리티컬 확률에 영향 | 10 |
| 지능 | INT | 마법 공격력, MP 최대치, 마법 크리티컬에 영향 | 10 |
| 체력 | VIT | HP 최대치, 물리 방어력, 상태이상 저항에 영향 | 10 |
| 정신력 | WIS | MP 최대치, 마법 방어력, MP 회복 속도에 영향 | 10 |
| 행운 | LUK | 크리티컬 확률, 아이템 드롭률, 특수 이벤트 발생률 | 5 |

#### 파생 스탯 (Secondary Stats)

| 스탯 | 계산 공식 | 설명 |
|------|----------|------|
| 최대 HP | `200 + (VIT × 20) + (Lv × 15)` | 최대 생명력 |
| 최대 MP | `100 + (WIS × 10) + (INT × 5) + (Lv × 8)` | 최대 마나 |
| 물리 공격력 | `STR × 2.5 + 장비 공격력` | 기본 물리 데미지 |
| 마법 공격력 | `INT × 2.8 + 장비 마법 공격력` | 기본 마법 데미지 |
| 물리 방어력 | `VIT × 1.5 + 장비 방어력` | 물리 피해 감소량 |
| 마법 방어력 | `WIS × 1.5 + 장비 마법 방어력` | 마법 피해 감소량 |
| 공격 속도 | `1.0 + (DEX × 0.005)` | 초당 공격 횟수 배율 (최대 2.5) |
| 크리티컬 확률 | `5% + (DEX × 0.2%) + (LUK × 0.3%)` | 크리티컬 히트 확률 (최대 75%) |
| 크리티컬 배율 | `150% + (DEX × 0.1%) + (LUK × 0.2%)` | 크리티컬 시 데미지 배율 |
| 회피율 | `(DEX × 0.15%) + (LUK × 0.1%)` | 공격 회피 확률 (최대 50%) |
| HP 재생 | `VIT × 0.5 / 초` | 전투 외 초당 HP 회복량 |
| MP 재생 | `WIS × 0.3 + INT × 0.1 / 초` | 초당 MP 회복량 |

### 1.2 레벨업 시스템

#### 최대 레벨
- **최대 레벨: 100**
- 레벨 100 달성 후 초월 시스템(Transcendence) 개방: 별도의 초월 레벨 1~20 추가

#### 경험치 공식

레벨 N에서 레벨 N+1로 올라가는 데 필요한 경험치:

```
Required_EXP(N) = 100 × N^1.8 × Growth_Factor(N)

Growth_Factor(N):
  - Lv 1~30:   1.0  (초반, 빠른 성장)
  - Lv 31~60:  1.3  (중반, 안정적 성장)
  - Lv 61~80:  1.6  (후반, 도전적 성장)
  - Lv 81~99:  2.0  (최후반, 장기 목표)
```

**레벨별 필요 경험치 예시:**

| 레벨 | 필요 경험치 | 누적 경험치 |
|------|------------|------------|
| 1 -> 2 | 100 | 100 |
| 5 -> 6 | 1,741 | 5,539 |
| 10 -> 11 | 6,310 | 28,927 |
| 20 -> 21 | 21,275 | 180,644 |
| 30 -> 31 | 44,924 | 622,800 |
| 50 -> 51 | 192,890 | 4,210,000 |
| 70 -> 71 | 611,200 | 20,500,000 |
| 99 -> 100 | 2,940,000 | 112,000,000 |

#### 경험치 획득 방법

| 방법 | 경험치 | 보너스 조건 |
|------|--------|------------|
| 일반 몬스터 처치 | 레벨 × 10 ~ 레벨 × 15 | 몬스터 레벨이 플레이어보다 높으면 +50% |
| 엘리트 몬스터 처치 | 레벨 × 50 ~ 레벨 × 80 | - |
| 보스 처치 | 레벨 × 200 ~ 레벨 × 500 | 최초 처치 보너스 ×3 |
| 퀘스트 완료 | 퀘스트 등급 × 500 | 일일 퀘스트 +20%, 스토리 퀘스트 +100% |
| 던전 클리어 | 던전 레벨 × 100 | 파티 보너스 최대 +30% |
| 제작/채집 | 아이템 레벨 × 5 | 처음 만드는 아이템 +100% |

#### 파티 경험치 규칙
- 파티원 2명: 각 +10% 경험치
- 파티원 3명: 각 +20% 경험치
- 파티원 4명: 각 +30% 경험치
- 레벨 차이 10 이상 파티원 존재 시: 하위 파티원은 경험치 50% 감소

### 1.3 스탯 성장 공식

#### 레벨업 시 자동 성장

레벨 N에서 N+1로 레벨업할 때 자동 증가 스탯 (클래스별 상이):

```
자동 성장 공식:
  HP  += VIT_Growth × (1 + 0.01 × N)
  MP  += WIS_Growth × (1 + 0.008 × N)
  STR += STR_Growth
  DEX += DEX_Growth
  INT += INT_Growth
  VIT += VIT_Growth_Base
  WIS += WIS_Growth_Base
  LUK += 0.2 (전 클래스 동일)
```

**클래스별 기본 성장치:**

| 스탯 | 에테르 기사 (Ether Knight) | 기억술사 (Mnemonist) | 그림자 직조사 (Shadow Weaver) |
|------|--------------|--------------|--------------|
| HP 성장 | +25 per Lv | +12 per Lv | +18 per Lv |
| MP 성장 | +5 per Lv | +20 per Lv | +10 per Lv |
| STR | +2.0 | +0.5 | +1.2 |
| DEX | +0.8 | +0.8 | +2.5 |
| INT | +0.3 | +2.8 | +0.7 |
| VIT | +2.5 | +0.8 | +1.5 |
| WIS | +0.5 | +2.0 | +1.0 |
| LUK | +0.2 | +0.2 | +0.2 |

#### 스탯 포인트 시스템

레벨업 시 **스탯 포인트 5점** 획득 (매 10레벨마다 추가 10점 보너스).
스탯 1포인트 투자 시 해당 스탯 +1 증가.

**스탯 투자 비용 (하드캡 방지):**

| 스탯 현재값 | 포인트 비용 |
|-----------|-----------|
| 1 ~ 100 | 1포인트 per 1스탯 |
| 101 ~ 200 | 2포인트 per 1스탯 |
| 201 ~ 300 | 4포인트 per 1스탯 |
| 301 이상 | 8포인트 per 1스탯 |

## 2. 클래스 시스템

> v1.1 업데이트 (2026-03-10): 클래스 명칭을 세계관 체계(기억술사/에테르 기사/그림자 직조사)로 통일

### 2.1 클래스 개요

3개의 기본 클래스로 시작하며, 레벨 30/60에 각각 전직 가능.

```
에테르 기사 (Ether Knight)
  ├─ Lv.30: 에테르 수호자 (Ether Guardian) / 에테르 폭주자 (Ether Berserker)
  │    ├─ Lv.60: 에테르 성기사 (Ether Paladin) / 황혼의 기사 (Twilight Knight)
  │    └─ Lv.60: 에테르 파괴자 (Ether Destroyer) / 망각 학살자 (Oblivion Slayer)

기억술사 (Mnemonist)
  ├─ Lv.30: 기억 원소술사 (Memory Elementalist) / 기억 소환사 (Memory Summoner)
  │    ├─ Lv.60: 기억 지배자 (Memory Overlord) / 빙결 기억술사 (Cryo-Mnemonist)
  │    └─ Lv.60: 망각 소환사 (Oblivion Summoner) / 기억 정령사 (Memory Spirit Master)

그림자 직조사 (Shadow Weaver)
  ├─ Lv.30: 그림자 추적자 (Shadow Tracker) / 심연의 칼날 (Abyss Blade)
  │    ├─ Lv.60: 안개 사냥꾼 (Mist Hunter) / 기억 저격수 (Memory Sniper)
  │    └─ Lv.60: 심연의 그림자 (Abyssal Shadow) / 망각의 독술사 (Lethe Poisoner)
```

### 2.5 클래스 전직 시스템

#### 1차 전직 (레벨 30)

**조건:**
- 레벨 30 달성
- 기본 클래스 스킬 3개 이상 최대 레벨
- 전직 퀘스트 완료 (클래스별 고유 스토리 퀘스트 5단계)
- 전직 비용: 골드 50,000

**효과:**
- 기존 스킬 강화 (모든 스킬 효과 +20%)
- 1차 전직 전용 스킬 트리 해금 (액티브 5개, 패시브 3개 추가)
- 스탯 성장률 +15% 증가
- 외형 변경 (의상, 무기 외형 업그레이드)

#### 2차 전직 (레벨 60)

**조건:**
- 레벨 60 달성
- 1차 전직 스킬 5개 이상 최대 레벨
- 전직 퀘스트 완료 (고난이도 던전 클리어 포함)
- 전직 재료 아이템 3종 수집
- 전직 비용: 골드 200,000

## 3. 전투 시스템

### 3.1 전투 방식 선택: 실시간 반자동 전투 (Semi-Auto Real-Time)

**선택된 방식:** 실시간 반자동 전투 (Real-Time with Active Pause)

#### 선택 이유

웹 브라우저 환경에서 완전한 실시간 액션은 다음 제약이 있음:
- 네트워크 레이턴시로 인한 반응성 저하 (50~200ms 지연)
- 브라우저 렌더링 한계 (복잡한 물리 연산 부적합)
- 마우스 전용 환경에서 복잡한 조작 어려움

**반자동 실시간 전투의 장점:**
- 자동 기본 공격으로 조작 부담 최소화
- 스킬은 플레이어가 수동으로 타이밍 결정 (전략적 깊이 유지)
- Active Pause: 스페이스바로 일시 정지 후 스킬 입력 가능
- 브라우저의 WebSocket을 통한 상태 동기화 최소화

### 3.2 전투 공식

#### 기본 물리 데미지 계산

```
기본 데미지 = (공격자 물리 공격력 - 방어자 물리 방어력 × 0.5) × 스킬 배율

물리 방어력 감소 공식:
  실제 방어 효과 = 방어력 / (방어력 + 방어자 레벨 × 10)

  예) 방어자 방어력 200, 레벨 50:
    실제 방어 효과 = 200 / (200 + 500) = 28.6% 피해 감소

최종 물리 데미지 = 기본 데미지 × (1 - 실제 방어 효과)
                  × 레벨 차이 보정
                  × 속성 상성 배율
                  × 크리티컬 배율 (해당 시)
```

#### 레벨 차이 보정

```
레벨 차이 보정 = 1 + (방어자 레벨 - 공격자 레벨) × 0.02

  - 공격자가 10레벨 낮으면 데미지 -20%
  - 공격자가 10레벨 높으면 데미지 +20%
  - 차이 최대 ±50% (25레벨 차이 이상은 고정)
```

#### 마법 데미지 계산

```
기본 마법 데미지 = 공격자 마법 공격력 × 스킬 배율

마법 방어력 감소 공식 (물리와 동일):
  마법 방어 효과 = 마법 방어력 / (마법 방어력 + 방어자 레벨 × 8)

최종 마법 데미지 = 기본 마법 데미지 × (1 - 마법 방어 효과)
                  × 속성 상성 배율
                  × 마법 크리티컬 배율 (해당 시)
```

#### 속성 상성 배율

| 공격 속성 | 강함 (×1.5) | 약함 (×0.7) | 무효 (×0) |
|---------|------------|------------|---------|
| 화염 | 빙결, 식물 | 화염, 물 | - |
| 빙결 | 바람, 화염 계열 적 | 땅, 빙결 | - |
| 번개 | 물, 금속 계열 | 땅, 번개 | - |
| 빛 | 어둠, 언데드 | 빛 계열 | - |
| 어둠 | 빛, 신성 계열 | 어둠, 언데드 | - |

#### 크리티컬 히트

```
크리티컬 판정:
  Random(0~100) < 크리티컬 확률(%) → 크리티컬 발생

크리티컬 데미지:
  최종 데미지 = 기본 데미지 × 크리티컬 배율

크리티컬 배율:
  기본 = 150%
  DEX 100 기준: 150% + (100 × 0.1%) = 160%
  LUK 50 기준: 160% + (50 × 0.2%) = 170%
  장비 및 스킬 보너스 추가 합산
```

#### 회피 판정

```
회피 판정:
  Random(0~100) < 회피율(%) → 회피 성공 (피해 0, "MISS" 표시)

회피율 상한: 50% (하드캡)
  → 적의 공격 명중률로 상쇄 가능

적 명중률:
  적 명중률 = 기본 95% + (적 레벨 - 플레이어 레벨) × 1%
  실제 회피율 = 플레이어 회피율 - (적 명중률 - 95%)
```

### 3.3 상태이상 종류

총 8가지 상태이상 설계:

| 상태이상 | 아이콘 색상 | 효과 | 지속시간 | 중첩 여부 | 저항 스탯 |
|---------|-----------|------|---------|---------|---------|
| 독 (Poison) | 초록 | 초당 최대 HP의 1.5% 독 데미지 | 5~15초 | 최대 3중첩 (데미지 합산) | VIT |
| 화상 (Burn) | 빨강 | 초당 최대 HP의 2% 화염 데미지, 물리 방어력 -15% | 3~8초 | 불가, 갱신 | WIS |
| 빙결 (Freeze) | 파랑 | 이동 불가, 행동 불가. 빙결 중 물리 피해 +30% | 1~3초 | 불가 | VIT |
| 기절 (Stun) | 노랑 | 모든 행동 불가. 회피 불가 | 1~2초 | 불가 | VIT |
| 출혈 (Bleed) | 진빨강 | 초당 최대 HP의 1% 물리 데미지. HP 재생 -80% | 5~10초 | 최대 5중첩 | VIT |
| 침묵 (Silence) | 보라 | 스킬 사용 불가 (기본 공격만 가능) | 3~6초 | 불가 | WIS |
| 공포 (Fear) | 검정 | 무작위 방향으로 도주. 스킬 사용 불가 | 2~4초 | 불가 | WIS |
| 약화 (Weaken) | 회색 | 물리/마법 공격력 -30%, 방어력 -20% | 5~15초 | 불가, 갱신 | VIT + WIS |

#### 상태이상 저항 공식

```
상태이상 저항률 = (저항 스탯 × 0.2) + (레벨 × 0.5) %

예) VIT 150, 레벨 50:
  독/빙결/기절/출혈 저항 = (150 × 0.2) + (50 × 0.5) = 30 + 25 = 55%
  → Random(0~100) < 55이면 상태이상 면역

저항 상한: 85% (하드캡)
```

- 즐겨찾기: 중요 아이템에 별표 표시 (실수로 버리기 방지)
- 잠금: 특정 아이템 판매/버리기 잠금

## 5. 경제 시스템

### 5.1 화폐 종류 및 획득 방법

#### 화폐 체계

| 화폐 | 아이콘 | 설명 | 주요 용도 |
|------|------|------|---------|
| 실버 (Silver) | 은화 | 게임 내 기본 화폐 | 일반 상점, 기본 강화 |
| 골드 (Gold) | 금화 | 100 실버 = 1 골드 | 고급 상점, 전직, 경매장 |
| 명성 포인트 (Reputation) | 별 | 퀘스트 체인 완료 시 획득 | 특수 NPC 상점, 전용 장비 |
| 레이드 토큰 | 토큰 | 레이드 클리어 시 획득 | 레이드 전용 장비 교환 |
| 시즌 코인 | 동전 | 시즌 콘텐츠 참여 보상 | 시즌 한정 아이템 교환 |

#### 골드 획득 방법

| 방법 | 획득량 | 주기 |
|------|-------|------|
| 몬스터 처치 | 레벨 × 0.5 ~ 3 골드 | 제한 없음 |
| 퀘스트 완료 | 50 ~ 5,000 골드 | 1회성 또는 일일 |
| 던전 클리어 | 던전 레벨 × 20 골드 | 주 3회 한정 |
| 아이템 판매 | 아이템 등급별 정가 | 제한 없음 |
| 경매장 거래 | 거래 가격 × 95% (5% 수수료) | 제한 없음 |
| 일일 미션 완료 | 500 ~ 2,000 골드 | 1일 1회 |
| 주간 미션 완료 | 5,000 ~ 20,000 골드 | 1주 1회 |

#### 골드 소모처

- 장비 강화/인챈트
- 전직 비용
- NPC 상점 구매
- 경매장 구매
- 스킬 리셋 (100 × 현재 레벨 골드)
- 펫 진화/강화

### 5.2 상점 시스템

#### NPC 상점 종류

| 상점 | 판매 품목 | 화폐 | 특징 |
|------|---------|------|------|
| 일반 상인 | 소비 아이템, 일반 장비 | 골드 | 상시 운영 |
| 무기 상인 | 고급 이하 무기 | 골드 | 도시마다 재고 상이 |
| 제작 재료 상인 | 제작 재료, 레시피 | 골드 | 일부 재료는 희귀 |
| 명성 상인 | 희귀~영웅 장비, 전용 외형 | 명성 포인트 | 명성 등급 조건 필요 |
| 레이드 보상 상인 | 레이드 전용 영웅~전설 장비 | 레이드 토큰 | 레이드 참여자 전용 |
| 시즌 상인 | 시즌 한정 외형, 소비품 | 시즌 코인 | 시즌 기간 한정 |

#### 동적 가격 시스템

NPC 상점의 일부 소비 아이템은 서버 수요에 따라 가격 변동:
```
기준가 × (1 + 수요 지수 × 0.1)
수요 지수: -3 ~ +5 범위 (24시간 단위 갱신)
최대 가격 변동: 기준가 ±30%
```

### 5.3 거래/경매 시스템

#### 플레이어 간 거래

**직접 거래:**
- 1:1 직접 거래 UI 지원
- 거래 확인 단계 2회 (실수 방지)
- 1일 거래 한도: 골드 1,000,000 (어뷰징 방지)

**경매장 (Auction House):**
- 서버 통합 경매장
- 아이템 등록 비용: 기본 등록비 100 골드 + 낙찰가의 5% 수수료
- 등록 기간: 1일/3일/7일 선택
- 즉시 구매가 설정 가능
- 미낙찰 시 아이템 반환 (등록비 비반환)

**경매장 기능:**
- 카테고리/등급/레벨 범위 필터
- 가격 히스토리 조회 (7일)
- 즐겨찾기 아이템 가격 알림 (브라우저 알림)
- 내 판매 내역 조회

#### P2W 방지 거래 제한

- **현금 아이템 경매장 등록 불가**: 유료 아이템은 캐릭터 귀속
- **성능 관련 아이템 거래 제한**: 강화석, 정련석은 일주일 1,000개 한도
- **신규 계정 제한**: 가입 7일 미만 계정은 경매장 이용 불가 (봇 방지)

## 6. 게임 밸런스 가이드라인

### 6.1 초반/중반/후반 콘텐츠 난이도 설계

#### 초반 콘텐츠 (레벨 1~30)

**목표:** 게임 시스템 학습, 캐릭터 육성 동기 부여

| 구간 | 레벨 | 주요 콘텐츠 | 난이도 | 예상 플레이 시간 |
|------|------|-----------|------|----------------|
| 튜토리얼 | 1~5 | 필드 몬스터, 기본 퀘스트 | 매우 쉬움 | 1~2시간 |
| 초보 구역 | 6~15 | 첫 번째 던전, 클래스 퀘스트 | 쉬움 | 5~10시간 |
| 전직 준비 | 16~25 | 파티 던전 입문, 스킬 숙련 | 보통 | 10~20시간 |
| 1차 전직 | 26~30 | 전직 퀘스트, 보스 레이드 입문 | 보통 | 5~10시간 |

**초반 밸런스 원칙:**
- 기본 장비로 솔로 플레이 가능
- 스킬 2~3개로 전투 가능 (과도한 시스템 요소 숨김)
- 실패 시 패널티 최소화 (장비 파괴 없음, 낮은 골드 손실)
- 자동 퀘스트 추적으로 방향 제시

#### 중반 콘텐츠 (레벨 31~60)

**목표:** 파티 플레이 유도, 클래스 정체성 확립, 다양한 콘텐츠 탐험

| 구간 | 레벨 | 주요 콘텐츠 | 난이도 | 예상 플레이 시간 |
|------|------|-----------|------|----------------|
| 전직 후 성장 | 31~40 | 중급 던전, 길드 시스템 개방 | 보통~약간 어려움 | 20~40시간 |
| 클래스 숙련 | 41~50 | 4인 레이드 입문, PvP 아레나 개방 | 어려움 | 40~80시간 |
| 고급 콘텐츠 | 51~59 | 8인 레이드, 하드 던전 | 어려움 | 80~150시간 |
| 2차 전직 | 60 | 2차 전직 퀘스트 (고난이도) | 매우 어려움 | 10~20시간 |

**중반 밸런스 원칙:**
- 솔로/파티 콘텐츠 비율 4:6 유지
- 주간 플레이 목표: 4~6시간으로 일주일 콘텐츠 완료 가능
- 과도한 그라인딩 없이 자연스러운 성장 곡선
- 다양한 클래스 역할(탱커/힐러/딜러) 가치 균등

#### 후반 콘텐츠 (레벨 61~100)

**목표:** 장기 목표 제공, 커뮤니티 활성화, 엔드게임 만족감

| 구간 | 레벨 | 주요 콘텐츠 | 난이도 | 예상 플레이 시간 |
|------|------|-----------|------|----------------|
| 엔드게임 입문 | 61~75 | 10인 레이드, 길드 전쟁 | 어려움 | 100~200시간 |
| 하드 엔드게임 | 76~90 | 신화 레이드, 시즌 콘텐츠 | 매우 어려움 | 200~500시간 |
| 최고난이도 | 91~100 | 헬 레이드, 길드 최강전 | 극악 | 500시간+ |
| 초월 (Transcendence) | T1~T20 | 초월 전용 콘텐츠 | 무한 성장 | 무제한 |

**후반 밸런스 원칙:**
- 캐릭터 성능 격차 완화: 최고급 장비와 일반 후반 장비 차이 최대 30% 이내
- 주 3회 레이드로 충분한 성장 가능
- 순위 시스템: 개인 실력 기반 (장비 스코어 + 클리어 타임)
- 신규/복귀 플레이어 캐치업 메카닉: 레벨 격차 보정 장비 지급

### 6.2 P2W 방지 설계 원칙

#### 핵심 원칙: "Pay for Cosmetics, Play for Power"

유료 재화는 외형 및 편의성에만 집중하고, 전투 성능과 관련된 모든 요소는 게임 내에서 획득 가능하도록 설계.

#### 유료 재화 허용 항목

| 항목 | 이유 | 상한 |
|------|------|------|
| 외형 스킨 (캐릭터/무기/방어구) | 전투 영향 없음 | 없음 |
| 코스튬 | 전투 영향 없음 | 없음 |
| 펫 외형 | 전투 영향 없음 | 없음 |
| 경험치 부스터 | 시간 단축만, 스탯 무관 | +100% EXP (최대) |
| 인벤토리 확장 (1회용) | 편의성, 동일 기능 게임 내 획득 가능 | 최대 +60칸 |
| 전용 이모트/감정표현 | 전투 영향 없음 | 없음 |
| 이름 변경권 | 편의성 | 없음 |

#### 유료 재화 금지 항목

| 항목 | 대안 |
|------|------|
| 강화 성공률 증가 아이템 | 천장 시스템으로 보완 |
| 스탯 포션 (영구) | 게임 내 한정 이벤트 보상으로만 지급 |
| 독점 성능 장비 | 모든 최고 성능 장비는 레이드/퀘스트로 획득 |
| 스킬 추가 해금 | 퀘스트/레벨로만 해금 |
| PvP 관련 성능 부스터 | PvP는 장비 스코어 평준화 시스템 도입 |

#### 게임 내 획득 방법 보장

모든 유료 아이템은 다음 방법 중 하나로 게임 내 획득 가능:
1. **게임 내 통화 구매:** 명성 포인트 또는 시즌 코인으로 구매
2. **이벤트 보상:** 월별 이벤트에서 보상으로 지급
3. **업적 보상:** 특정 업적 달성 시 지급
4. **시즌 패스 무료 트랙:** 유료 시즌 패스의 50% 아이템은 무료 트랙에서도 획득

#### 시즌 패스 설계 원칙

```
시즌 패스 구성:
  - 무료 트랙: 100단계 중 50개 아이템 (외형 30%, 소모품 70%)
  - 유료 트랙: 추가 50개 아이템 (외형 80%, 소모품 20%)

  유료 트랙 포함 아이템:
    - 성능 관련 소모품 한도: 최대 20개 (경험치 부스터, 이동속도 포션)
    - 절대 포함 불가: 강화석, 정련석, 스탯 강화 아이템
```

#### 밸런스 모니터링 지표

운영팀이 주기적으로 모니터링해야 할 지표:

| 지표 | 목표 범위 | 이상 기준 |
|------|---------|---------|
| 유료 결제 유저 vs 무료 유저 전투력 차이 | < 15% | 20% 초과 시 조정 |
| 클래스별 레이드 참여 비율 | 각 클래스 20~40% | 한 클래스 50% 초과 시 리밸런싱 |
| 신규 유저 30일 잔존율 | > 30% | 25% 미만 시 초반 콘텐츠 재검토 |
| 일일 접속 유저 중 결제 비율 | < 10% | 15% 초과 시 게임 내 획득처 보강 |
| 평균 레이드 클리어 성공률 | 40~70% | 30% 미만/80% 초과 시 난이도 조정 |

## 부록: 시스템 통합 공식 요약

### 데미지 최종 계산 전체 공식

```
최종 데미지 =
  ( 공격력 × 스킬 배율
    - 방어력 × (방어력 / (방어력 + 방어자 레벨 × 10)) )
  × 레벨 차이 보정
  × 속성 상성 배율
  × 콤보 보너스
  × 크리티컬 배율 (크리티컬 시)
  × 상태이상 보너스 (빙결 적에게 물리 +30% 등)
  × 버프/디버프 배율

최솟값: 1 (데미지는 항상 최소 1 보장)
```

### 캐릭터 성장 요약

```
레벨 1 → 100 성장 배율 (에테르 기사 기준):
  HP:  200 → ~22,000 (110배)
  MP:  100 → ~3,500  (35배)
  STR: 10  → ~220    (22배)
  물리 공격력: ~25 → ~600 (24배, 장비 제외)
  물리 방어력: ~15 → ~450 (30배, 장비 제외)
```
