# 에테르나 크로니클 — PvP 밸런스

> **작성일:** 2026-03-10 | **버전:** v1.0 | **분류:** 코어기획 — 전투/PvP

---

## 1. 개요

PvP 시스템은 공정한 경쟁 환경을 최우선으로 설계한다. 장비 격차가 아닌 플레이어 스킬과 전략이 승패를 결정하도록 스코어 정규화, 전용 조정값, 시즌 제도를 운용한다.

### 1.1 설계 원칙

| 원칙 | 설명 |
|------|------|
| **장비 평준화** | 장비 스코어 정규화로 신규/복귀 유저도 경쟁 가능 |
| **스킬 기반 승패** | 스탯 차이 최소화, 포지셔닝·스킬 사용·판단력이 핵심 |
| **메타 다양성** | 특정 클래스/빌드 독점 방지, 정기 밸런스 패치 |
| **보상 ≠ 전력** | PvP 보상은 코스메틱/칭호 중심, 전투력 격차 유발 최소화 |

## 2. 아레나 모드

### 2.1 모드 종류

| 모드 | 인원 | 입장 조건 | 시간 제한 | 설명 |
|------|------|-----------|-----------|------|
| **솔로 아레나** | 1v1 | 레벨 30+ | 5분 | 순수 개인 실력 경쟁 |
| **팀 아레나** | 3v3 | 레벨 30+, 파티 또는 자동 매칭 | 8분 | 팀 시너지 + 역할 분담 |
| **길드 전쟁** | 20v20 | 길드 레벨 10+ | 60분 | 별도 문서 참조 (guild_system_design.md) |

### 2.2 솔로 아레나 규칙

- **맵:** 대칭형 평지 아레나 (장애물 3~4개, 맵 3종 랜덤)
- **시작:** 양측 15m 거리에서 3초 카운트다운 후 시작
- **승리 조건:** 상대 HP 0 또는 시간 종료 시 잔여 HP% 높은 쪽
- **라운드:** Best of 3 (랭크), Best of 1 (비랭크)
- **Active Pause:** PvP에서 비활성 (실시간 전투만)

### 2.3 팀 아레나 규칙

- **맵:** 중앙 오브젝트형 맵 (점령 포인트 1개). 맵 5종 랜덤
- **승리 조건:**
  - 적 전원 처치 (즉시 승리)
  - 시간 종료 시: 점령 포인트 점유 시간 합산이 높은 팀
- **라운드:** Best of 5 (랭크), Best of 3 (비랭크)
- **클래스 제한:** 동일 클래스 최대 2인 (3인 동일 클래스 금지)
- **부활:** 없음 (라운드 내 1데스 아웃)

## 3. 장비 스코어 정규화

### 3.1 정규화 공식

PvP 아레나 진입 시 모든 장비 스탯에 정규화가 적용된다:

```
normalizedStat = baseStat + (gearStat × 0.3)
```

| 변수 | 설명 |
|------|------|
| `baseStat` | 레벨·클래스·전직에 의한 기본 스탯 (장비 미착용 기준) |
| `gearStat` | 장비가 부여하는 추가 스탯 (강화/소켓/세트 효과 포함) |
| `0.3` | PvP 감쇠 계수 (장비 스탯의 70% 감쇠) |

### 3.2 정규화 적용 범위

| 스탯 | 정규화 적용 | 비고 |
|------|------------|------|
| HP | ✅ | baseStat + gearStat × 0.3 |
| 물리 공격력 | ✅ | |
| 마법 공격력 | ✅ | |
| 물리 방어력 | ✅ | |
| 마법 방어력 | ✅ | |
| 치명타율 | ✅ | 최대 캡: 40% (PvP) |
| 치명타 데미지 | ✅ | 최대 캡: 180% (PvP) |
| 공격 속도 | ✅ | 최대 캡: +50% |
| 이동 속도 | ❌ | 기본값 고정 (클래스별 동일) |
| CC 저항 | ✅ | 최대 캡: 60% |

### 3.3 정규화 예시

```
Lv.80 에테르 기사 (전직 3차)
  baseStat 물리공격력: 500
  gearStat 물리공격력: 800 (전설 풀셋)

PvE 물리공격력: 500 + 800 = 1,300
PvP 물리공격력: 500 + (800 × 0.3) = 500 + 240 = 740

→ 장비 차이가 1,300 vs 예를 들어 고급 셋 700(500+200)일 때,
  PvE: 1,300 vs 700 (차이 600, 86% 격차)
  PvP: 740 vs 560 (차이 180, 32% 격차)
```

## 4. PvP 전용 조정값

PvE와 동일한 스킬을 사용하되, PvP 환경에서는 다음 전용 조정이 적용된다:

### 4.1 글로벌 조정

| 조정 항목 | PvP 적용값 | 이유 |
|-----------|-----------|------|
| **CC(군중제어) 지속 시간** | -50% | CC체인 무한 기절 방지 |
| **힐량** | -30% | 소모전 장기화 방지 |
| **즉사기** | 무효 (최대HP 30% 데미지로 변환) | 원콤 방지 |
| **소환수 HP** | -40% | 소환 빌드 과잉 방지 |
| **흡혈 효과** | -50% | 탱커/딜러 자힐 과잉 방지 |
| **보호막** | 최대 HP 15% 캡 | 무한 보호막 방지 |

### 4.2 클래스별 PvP 조정

| 클래스 | 조정 내용 | 목적 |
|--------|-----------|------|
| **에테르 기사** | 도발 스킬 PvP 비활성, 방어 스킬 쿨타임 +20% | 탱커 생존력 과잉 억제 |
| **기억술사** | 채널링 스킬 시전 시간 +0.5초, 범위 스킬 범위 -15% | 원거리 안전 딜링 견제 |
| **그림자 직조사** | 은신 지속 최대 5초(PvP), 은신 중 첫 타격 보너스 +30%→+15% | 암살 원콤 방지, 은신 장기화 방지 |

### 4.3 디미니싱 리턴 (DR)

동일 대상에게 같은 종류의 CC를 연속 적용 시 효과가 감소한다:

| 적용 횟수 | CC 지속 시간 |
|-----------|-------------|
| 1회 (첫 적중) | 100% (이미 -50% 적용된 값) |
| 2회 | 50% |
| 3회 | 25% |
| 4회+ | **면역** (15초간 해당 CC 타입 면역) |

CC 타입 분류: 기절(Stun) / 속박(Root) / 침묵(Silence) / 넉백(Knockback) / 슬로우(Slow)
- 다른 타입의 CC는 별도로 DR 추적

## 5. 금지 아이템

PvP 아레나에서 사용 불가한 아이템 목록:

| 아이템 | 금지 사유 |
|--------|-----------|

| **환경 변환 아이템** | 맵 지형 조작 방지 |

**허용 소비 아이템:**
- HP 포션 (쿨타임 30초, 최대HP 20% 회복)
- MP 포션 (쿨타임 30초, 최대MP 30% 회복)

## 6. 시즌 시스템

### 6.1 시즌 기본 정보

| 항목 | 값 |
|------|-----|
| 시즌 기간 | 3개월 (1/4/7/10월 시작) |
| 시즌 간 휴식기 | 1주일 (프리시즌: 랭크 미반영, 연습 모드) |

### 6.2 랭크 구간

| 구간 | MMR 범위 | 아이콘 색상 | 인구 목표 비율 |
|------|---------|------------|---------------|
| **브론즈** | 0~999 | 🟤 갈색 | 30% |
| **실버** | 1,000~1,499 | ⚪ 은색 | 25% |
| **골드** | 1,500~1,899 | 🟡 금색 | 20% |
| **플래티넘** | 1,900~2,199 | 🔵 청록 | 15% |
| **다이아몬드** | 2,200~2,499 | 💎 보라 | 8% |
| **챔피언** | 2,500+ | 🔴 빨강 | 2% |

### 6.3 시즌 보상

| 구간 | 코스메틱 | 칭호 | 추가 보상 |
|------|---------|------|-----------|
| 브론즈 | 시즌 프레임 (브론즈) | "도전자" | 골드 10,000 |
| 실버 | 시즌 프레임 + 무기 스킨 | "투사" | 골드 30,000 |
| 골드 | 위 + 방어구 스킨 | "전략가" | 골드 50,000 + 영웅 장비 상자 |
| 플래티넘 | 위 + 오라 이펙트 | "전장의 별" | 골드 100,000 + 전설 에테르 결정 |
| 다이아몬드 | 위 + 탈것 스킨 | "에테르 투사" | 골드 200,000 + 전설 장비 선택 상자 |
| 챔피언 | 위 + 전용 날개 + 조각상(로비) | "챔피언" | 골드 500,000 + 신화 에테르 결정 |

- 보상은 시즌 종료 시 **최고 도달 구간** 기준 지급
- 코스메틱은 해당 시즌 한정 (재획득 불가)
- 칭호는 영구 (시즌 번호 포함: "시즌 1 챔피언")

## 7. 매칭 시스템 — Glicko-2

### 7.1 Glicko-2 개요

Elo의 개선 버전으로, **레이팅(μ)**, **편차(φ)**, **변동성(σ)**을 함께 추적한다.

| 파라미터 | 초기값 | 설명 |
|----------|--------|------|
| μ (레이팅) | 1,500 | 실력 추정치 |
| φ (편차) | 350 | 레이팅 불확실성 (높을수록 변동 큼) |
| σ (변동성) | 0.06 | 실력 변동 가능성 |
| τ (시스템 상수) | 0.5 | 변동성 조절 (서버 설정) |

### 7.2 매칭 풀

```
매칭 요청 진입
    │
    ├── 0~30초: 좁은 범위 매칭 (MMR ± 50)
    ├── 30~60초: 중간 범위 (MMR ± 100)
    ├── 60~120초: 넓은 범위 (MMR ± 200)
    └── 120초+: 최대 범위 (MMR ± 400) + "매칭 품질 경고" 표시
```

### 7.3 매칭 품질 조건

| 조건 | 값 |
|------|-----|
| 최대 MMR 차이 (솔로) | 400 |
| 최대 MMR 차이 (팀) | 팀 평균 MMR 기준 300 |
| 파티 vs 솔로 보정 | 파티 팀 MMR에 +50 페널티 (소통 이점 상쇄) |
| 연승/연패 보정 | 3연승 시 상대 MMR +30, 3연패 시 상대 MMR -30 |
| 최소 게임 수 (랭크) | 10판 배치 매치 → 초기 MMR 결정 |

### 7.4 배치 매치

- 시즌 시작 시 10판 배치 매치 수행
- 초기 MMR: 소프트 리셋 값 기준
- 배치 중 φ(편차)를 높게 유지하여 빠른 수렴
- 배치 결과에 따라 브론즈~플래티넘 배치 (다이아/챔피언은 배치 불가)

## 8. 반칙 · 제재

### 8.1 탐지 시스템

| 유형 | 탐지 방법 | 제재 |
|------|-----------|------|
| **윈트레이딩** | 동일 IP/기기 매칭 반복 감지 | 1차 경고, 2차 시즌 밴, 3차 영구 밴 |
| **의도적 패배** | 매치 시작 후 무행동/즉시 항복 패턴 | 1차 경고 + 디저터 페널티 (30분) |
| **핵/치트** | 서버 사이드 판정 (데미지 검증, 이동속도 검증) | 즉시 영구 밴 |
| **AFK** | 30초 무입력 감지 | 자동 패배 + 디저터 페널티 |
| **욕설/비매너** | 채팅 필터 + 신고 기반 | 채팅 제한 → 임시 PvP 밴 |

### 8.2 디저터 페널티

| 이탈 횟수 (24시간 기준) | 페널티 |
|------------------------|--------|
| 1회 | 대기열 30분 금지 |
| 2회 | 대기열 2시간 금지 |
| 3회+ | 당일 PvP 금지 + MMR -50 |

## 9. 밸런스 패치 정책

| 항목 | 주기 | 내용 |
|------|------|------|
| **핫픽스** | 수시 | 게임 브레이킹 버그/익스플로잇 긴급 수정 |
| **밸런스 패치** | 격주 | 승률/픽률 데이터 기반 수치 조정 |
| **시즌 패치** | 분기 | 대규모 메타 변경, 신규 스킬/아이템 PvP 적용 |

**밸런스 기준 데이터:**

| 지표 | 정상 범위 | 조정 트리거 |
|------|-----------|------------|
| 클래스별 승률 (1v1) | 45%~55% | 범위 이탈 2주 연속 |
| 클래스별 픽률 (3v3) | 15%~50% | 5% 미만 또는 70% 이상 |
| 스킬 사용률 | 상위 80%가 최소 40% 사용 | 특정 스킬 사용률 5% 미만 |
| 평균 매치 시간 | 1v1: 2~4분, 3v3: 4~7분 | 범위 이탈 |

## 10. API 엔드포인트

```
GET    /api/pvp/profile/:userId      — PvP 프로필 (MMR, 시즌 랭크, 전적)
POST   /api/pvp/queue                — 매칭 대기열 등록
DELETE /api/pvp/queue                — 매칭 대기열 취소
GET    /api/pvp/match/:matchId       — 매치 상세 정보
POST   /api/pvp/match/:matchId/result — 매치 결과 제출 (서버 내부)

GET    /api/pvp/season/current       — 현재 시즌 정보
GET    /api/pvp/season/:seasonId/ranking — 시즌 랭킹 (페이지네이션)
GET    /api/pvp/season/:seasonId/rewards — 시즌 보상 목록

GET    /api/pvp/leaderboard          — 전체 리더보드
GET    /api/pvp/history/:userId      — 매치 히스토리
```
