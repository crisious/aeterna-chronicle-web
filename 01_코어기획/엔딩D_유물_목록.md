## 개요

엔딩D **"신들의 귀환"** (히든 엔딩)은 `story.artifact_count == 12` 달성 시 해금된다.  
12개 유물 = **신성 기억 파편 4개** + **히든 유물 8개**.

파편은 메인 스토리 진행으로 자연 획득되지만, 히든 유물 8개는 특정 조건(봉인 유적 방문, 사이드 퀘스트, 히든 보스 처치 등)을 만족해야 한다.

## 3. 엔딩D 판정 로직과의 관계

```typescript
// server/src/ending/endingEngine.ts — 엔딩 판정 우선순위
// 1순위: 엔딩D "신들의 귀환" → artifact_count == 12
// 2순위: 엔딩B "망각의 선택" → betrayal_score >= 70
// 3순위: 엔딩A "기억의 수호자" → fragment == 4 && party_alive == 63
// 4순위: 엔딩C "마지막 증인" → fragment >= 3

if (flags.artifact_count === 12) {
  return 'ENDING_D_GODS_RETURN';
}
```

| 히든 유물 | 8 | 봉인 유적, 사이드퀘, 히든 보스, 호감도, 무한 던전 |

### SSOT 정합

- `멀티엔딩_플래그_설계.md` → `story.artifact_count` 최대값 12 ✅
- `GDD_final.md` → 엔딩D "sacred_artifacts=12" ✅
- 파편 4개 = `story.fragment_count` (별도 플래그)와 중복 카운트 아님. `artifact_count`는 파편 + 히든 유물 통합 카운트.
