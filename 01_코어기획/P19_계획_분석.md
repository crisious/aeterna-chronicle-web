## 4. 생산 파이프라인

```
[프롬프트 작성] → [AI 생성 (AudioCraft)] → [후처리 (FFmpeg/SoX)]
     ↓                    ↓                        ↓
bgm_ai_music_design.md  raw WAV (32kHz)      OGG Vorbis (44.1kHz/16bit)
sound_design.md SFX ID                        assets/generated/audio/
```

### 디렉토리 구조
```
assets/generated/audio/
├── bgm/           # BGM 40곡 (.ogg)
├── sfx/
│   ├── combat/    # 전투 SFX 25개
│   ├── ui/        # UI SFX 15개
│   ├── system/    # 시스템 SFX 10개
│   ├── memory/    # 기억 메카닉 SFX 9개
│   └── ambient/   # 환경음 15개 + 승리스팅거 1개
├── voice/
│   ├── npc/       # NPC 보이스 10개
│   └── combat/    # 전투 보이스 10개
└── scripts/       # 생성/변환 스크립트
```
