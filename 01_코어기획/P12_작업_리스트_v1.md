## P12 개요 — 커뮤니티 확장 + 기술 부채 해소

P9 미사용 트랙 E(커뮤니티/소셜 확장)와 트랙 H(기술 부채 해소)를 병합.  
외부 플랫폼 연동(SNS/Discord/스트리머) + 사용자 생성 콘텐츠 + 성능 최적화로  
장기 운영 기반을 확보한다.

**핵심 원칙:**
- SNS 공유, Discord 봇, 스트리머 모드로 외부 노출 극대화
- 레퍼럴/UGC/커뮤니티 이벤트로 유저 유입·리텐션 강화
- TypeScript strict, 번들/DB/Redis 최적화로 기술 부채 청산
- 로드 테스트 + 모니터링 고도화로 프로덕션 안정성 확보
