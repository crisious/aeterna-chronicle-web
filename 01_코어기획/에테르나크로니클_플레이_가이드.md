## 1. 시작하기

### 1.1 게임 소개

**에테르나 크로니클(Aeterna Chronicle)**은 기억이 소멸되는 판타지 세계 **에테르나**를 배경으로 한 실시간 반자동 전투 RPG다.

- **장르**: Semi-Auto Real-Time RPG
- **플레이타임**: 메인 스토리 40~50시간 / 완전 클리어 80~100시간
- **플랫폼**: PC 웹(Phase 1) → Unity PC/모바일(Phase 2) → UE5 콘솔(Phase 3)

세계력 3,412년. 212년 전(세계력 3,200년) 발생한 **대망각(The Great Forgetting)** 이후, 인간과 이종족 모두 기억을 잃어가는 **기억 소멸 증후군(MDS)**이 창궐하고 있다. 플레이어는 기억 공명 능력을 가진 주인공 **에리언**을 조종해 세계에 흩어진 **신성 기억 파편 4개**를 수집하고, 망각의 신 **레테**의 음모를 저지한다.

모든 선택이 엔딩에 영향을 미친다. 4종 정규 엔딩 + 패배 루트가 존재.

### 1.2 클래스 선택 가이드

캐릭터 생성 시 5개 클래스 중 하나를 선택한다. 레벨 30에서 1차 전직, 레벨 60에서 2차 전직이 가능하며 최종 20개 세부 클래스로 분기된다.

| 클래스 | 포지션 | 난이도 | 핵심 스킬 | 추천 대상 |
|--------|--------|--------|-----------|-----------|
| **기억술사** | 원거리 마법 딜러 (주인공 에리언) | ★★☆ | 기억 파편 소환, 망각의 화살(적 버프 초기화), 기억 폭발(광역) | 스토리 몰입, 범용 딜러 |
| **수호자** (에테르 기사) | 근접 탱커/딜러 | ★☆☆ | 에테르 방벽, 충격파 베기, 에테르 각성 | 생존력 중시, 초보자 |
| **에테르학자** (그림자 직조사) | 암살/서포터 하이브리드 | ★★★ | 그림자 발걸음, 안개 분신, 심연의 포옹 | 고난이도 선호, 유틸리티 |
| **기억 파괴자** ★ | 근접 딜러/디버퍼 | ★★★ | 기억 침식, 파편 수집, 망각 폭류 | Ch.6 해금, 공격적 디버프 |
| **시간 수호자** ★ | 서포터/컨트롤러 | ★★☆ | 시간 정지, 시간 역류, 영원의 방벽 | Ch.7 해금, 파티 지원 |

#### 전직 트리 요약

```
기억술사 ─┬─ 기억 원소술사 ─┬─ 기억 지배자
          │                └─ 빙결 기억술사
          └─ 기억 소환사   ─┬─ 망각 소환사
                            └─ 기억 정령사

수호자   ─┬─ 에테르 수호자 ─┬─ 에테르 성기사
          │                └─ 황혼의 기사
          └─ 에테르 폭주자 ─┬─ 에테르 파괴자
                            └─ 망각 학살자

에테르학자 ─┬─ 그림자 추적자 ─┬─ 안개 사냥꾼
            │                └─ 기억 저격수
            └─ 심연의 칼날  ─┬─ 심연의 그림자
                              └─ 망각의 독술사

기억 파괴자 ─┬─ 기억 파쇄사   ─┬─ 기억 소멸자
             │                └─ 망각 해체사
             └─ 기억 침식자   ─┬─ 망각의 지배자
                               └─ 기억 폭주자

시간 수호자  ─┬─ 시간 수호사   ─┬─ 시간 지배자
             │                └─ 영원의 파수꾼
             └─ 시간 관측자   ─┬─ 시간 조율사
                               └─ 시간 역행자
```

> **💡 첫 플레이 추천:** 수호자(에테르 기사)로 시작하면 생존력이 높아 시스템 학습이 편하다. 스토리에 깊이 빠지고 싶다면 기억술사(에리언)를 추천.
> ★ 기억 파괴자는 챕터 6(안개해), 시간 수호자는 챕터 7(기억의 심연) 클리어 후 해금된다.

## 2. 전투 시스템

### 2.1 반자동 전투 개요

에테르나 크로니클의 전투는 **실시간 반자동(Semi-Auto Real-Time)** 방식이다.

- **기본 공격**: 자동 — 타겟 지정 후 자동 공격
- **스킬**: 수동 — 1~8번 키로 타이밍을 직접 결정
- **Active Pause**: 스페이스바로 정지 후 전략 수립 가능

#### 데미지 공식 (참고)
```
최종 데미지 = (공격력 × 스킬 배율 - 방어력 감소)
            × 레벨 차이 보정 × 속성 상성 배율
            × 콤보 보너스 × 크리티컬 배율 × 버프/디버프 배율
```

---

### 2.2 스킬 시스템 (클래스별 30개, 총 150개)

각 클래스는 기본 스킬 10개 + 1차 전직 스킬 10개 + 2차 전직 스킬 10개로 총 **30개 스킬**을 보유한다. 5개 클래스 합산 **총 150개 스킬**.

| 클래스 | 기본 (Lv.1~29) | 1차 전직 (Lv.30~59) | 2차 전직 (Lv.60+) |
|--------|----------------|---------------------|-------------------|
| 기억술사 | 기억 파편 소환, 망각의 화살, 기억 폭발 외 7종 | 원소 분기 / 소환 분기 각 10종 | 지배자/빙결/망각소환/정령 각 10종 |
| 수호자 | 에테르 방벽, 충격파 베기, 에테르 각성 외 7종 | 수호 분기 / 폭주 분기 각 10종 | 성기사/황혼/파괴/학살 각 10종 |
| 에테르학자 | 그림자 발걸음, 안개 분신, 심연의 포옹 외 7종 | 추적 분기 / 심연 분기 각 10종 | 사냥꾼/저격/그림자/독술 각 10종 |
| 기억 파괴자 ★ | 기억 침식, 파편 수집, 망각 폭류 외 7종 | 파쇄 분기 / 침식 분기 각 10종 | 소멸자/해체사/지배자/폭주자 각 10종 |
| 시간 수호자 ★ | 시간 정지, 시간 역류, 영원의 방벽 외 7종 | 수호 분기 / 관측 분기 각 10종 | 지배자/파수꾼/조율사/역행자 각 10종 |

**퀵슬롯 구성**: 스킬 8슬롯 + 소비아이템 4슬롯 = 총 12슬롯(단축키 1~=)

### 2.3 상태이상 10종

| 상태이상 | 효과 | 지속 | 해제 |
|----------|------|------|------|
| **독** | 3초마다 최대 HP 2% 지속 피해 | 15초 | 해독제 |
| **화상** | 2초마다 고정 피해 + 방어력 -10% | 10초 | 빙결/물 속성 |
| **빙결** | 이동 불가, 다음 피격 피해 +30% | 5초 | 화염 속성/시간 경과 |
| **기절** | 모든 행동 불가 | 3초 | 시간 경과 |
| **출혈** | 이동 시 HP 감소 | 12초 | 치유 스킬/포션 |
| **침묵** | 스킬 사용 불가 | 8초 | 해제 스킬 |
| **공포** | 랜덤 방향 이동 (조작 불가) | 4초 | 시간 경과 |
| **약화** | 공격력/방어력 -20% | 10초 | 버프 스킬 |

### 2.4 콤보 시스템 (최대 15체인)

3초 이내 연속 공격 시 콤보 카운터가 적립된다. **최대 15체인**까지 누적 가능.

| 콤보 수 | 데미지 보너스 | 추가 효과 |
|---------|-------------|-----------|
| 1~5 | +5%/체인 | — |
| 6~10 | +10%/체인 | 크리티컬 확률 +10% |
| 11~15 | +15%/체인 | MP 소비 -30%, 크리티컬 +20% |

**스킬 체인 보너스**: 특정 스킬을 연속 사용하면 체인 보너스가 발동한다.

| 체인 조합 | 체인명 | 효과 |
|-----------|--------|------|
| 아이스 랜스 → 파이어볼 | 원소 폭발 | 데미지 ×2.5 |
| 그림자 발걸음 → 심연의 포옹 | 암흑 포옹 | 적 시야 차단 5초 |
| 에테르 방벽 → 충격파 베기 | 반격 일격 | 방벽 흡수량 비례 데미지 |
| 기억 소환 → 망각의 화살 | 기억 궤멸 | 적 버프 전부 해제 + 10초 재적용 불가 |

## 3. 챕터 공략 개요

### 3.0 전체 흐름

```
Ch1 에레보스 (Lv.1~20)
  ↓
Ch2 실반헤임 (Lv.20~35) → 파편 ①
  ↓
Ch3 솔라리스 (Lv.35~50) → 파편 ②
  ↓
Ch4 아르겐티움 + 북방 + 브리탈리아 (Lv.50~80) → 파편 ③④
  ↓
Ch5 망각의 고원 (Lv.80~100) → 최종 결전
  ↓ (시즌 2)
Ch6 무한 안개해 (Lv.80~100) → 기억 파괴자 해금, 레이드 보스 3종
  ↓ (시즌 3)
Ch7 기억의 심연 (Lv.95~∞) → 시간 수호자 해금, 무한 던전, 월드 보스
```

### 3.1 챕터 1: 잊혀진 도시의 목격자 — 에레보스

| 항목 | 내용 |
|------|------|
| **지역** | 에레보스 — 대망각의 진원지, 유령 도시 |
| **레벨 범위** | 1~20 |
| **보스** | **기억의 골렘** (에레보스 하수도) |
| **동료 합류** | **세라핀** (엘파리스 정찰병, 300세) |
| **파편 획득** | 없음 (단서 수집 단계) |
| **핵심 퀘스트** | 칸텔라 마을 사건 조사, 왕립 도서관 탐사, 레테 교단 서약문 수집 |

**공략 포인트:**
- 칸텔라 마을에서 기억 소멸 폭풍을 목격하며 게임이 시작된다
- 왕립 도서관에서 에테르나 신화와 대망각의 진실을 밝히는 문서를 확보하라
- 세라핀과의 **신뢰도**를 관리하라 — 이 시점부터 엔딩 A 조건에 영향
- 하수도 보스 **기억의 골렘**은 기억 공명으로만 약화 가능. Active Pause를 적극 활용

> ⚠️ 세라핀 신뢰도가 0이 되면 이탈하며 **엔딩 A 영구 불가**.

### 3.2 챕터 2: 기억의 숲 — 실반헤임

| 항목 | 내용 |
|------|------|
| **지역** | 실반헤임 — 기억 소멸이 일어나지 않는 고대 숲 |
| **레벨 범위** | 20~35 |
| **보스** | **말라투스** (수천 년 기억을 흡수한 거대 고목, 3페이즈) |
| **동료 합류** | 없음 (세라핀 동행 유지) |
| **파편 획득** | **신성 기억 파편 ① — 말라투스 고목 뿌리 아래** |
| **핵심 퀘스트** | 엘파리스 외교(전투/평화 분기), 말라투스 수수께끼, 카일의 유산 |

**공략 포인트:**
- 숲 내부는 **시간 지연** 지형 — 숲 안 1일 = 바깥 3시간. 장기 체류에 주의
- 엘파리스 경비병과의 첫 만남에서 **전투 루트 / 평화 루트** 분기. 평화 루트를 추천 (이후 세력 친밀도 보너스)
- 말라투스 보스전은 3페이즈. 기억 공명으로 과거 기억을 열람하는 특수 메커닉이 있다
- 카일(에리언의 전생)이 봉인해둔 **첫 번째 신성 기억 파편**을 획득

> 💡 세라핀의 고향이 실반헤임이다. 죽은 연인 **카엘** 관련 기억 단편이 발견되며, 세라핀 개인 서사가 진행된다.

## 5. 제작 & 경제

### 5.1 에테르 결정 3종

에테르나 세계의 핵심 자원인 에테르 결정은 3가지 형태로 존재한다.

| 형태 | 특성 | 주요 산지 | 용도 |
|------|------|-----------|------|
| **붉은 에테르 결정** | 고에너지, 불안정 | 솔라리스 사막 | 무기 제작, 고성능 기관차 동력 |
| **황금 에테르 결정** | 안정적, 희귀 | 아르겐티움 | 최고급 장비, 황금 에테르 탑 |
| **청백 에테르 결정** | 약에너지, 풍부 | 전 지역 | 일상용 조명, 난방, 기본 제작 |

#### 에테르 결정 가공 체계

```
에테르 파편 ×5  →  정제 에테르 결정  (숙련도 Lv.40)
정제 ×3 + 기억 결정  →  순수 에테르 결정  (숙련도 Lv.60)
순수 ×3 + 신성 기억 파편  →  신성 에테르 결정  (숙련도 Lv.80)
```

시즌패스는 **무료 트랙 + 프리미엄 트랙** 2줄 구조다.

### 패배 루트: 망각의 세계

**달성 조건:**
- 최종 보스 레테에게 **패배**
- 또는 의도적으로 **항복 선택지** 선택

**결말:**
세계 전체가 기억을 잃고, 문명이 완전히 리셋된다. 모든 고통과 함께 모든 기억도 사라진다. "망각도 또 다른 형태의 구원인가?" — 게임의 핵심 질문을 던지며 끝난다.

</details>

### 엔딩 판정 우선순위

```
D (신들의 귀환)  →  C (망각의 선택)  →  A (기억의 수호자)  →  B (마지막 증인, 기본값)
```

- 엔딩 D 조건을 만족하면 다른 조건과 무관하게 D
- D 미충족 + C 조건 충족이면 C
- D·C 미충족 + A 조건 충족이면 A
- 위 모두 미충족이면 B (기본 엔딩)

## 9. 팁 & FAQ

### 💡 초보자 필수 팁

1. **Active Pause를 습관화하라.** 보스전에서 패닉에 빠지면 스페이스바부터 눌러라. 멈춘 상태에서 적 패턴, 스킬 쿨타임, 아이템을 확인할 수 있다.

2. **세라핀 신뢰도를 절대 0으로 만들지 마라.** Ch1부터 관리 시작. 엔딩 A를 노리지 않더라도 세라핀 이탈은 후반 시나리오에 큰 영향.

3. **제작 숙련도는 연금술부터 올려라.** HP/MP 포션 자급자족이 골드 절약의 핵심. 연금술 Lv.15면 상급 HP 포션 제작 가능.

4. **에테르 파편은 버리지 마라.** 재료 창고에 자동 분류되니 신경 쓸 필요 없지만, 분해용으로 쓰지 말 것. 에테르 결정 가공의 핵심 재료.

5. **속성 상성을 활용하라.** 빙결 몬스터에 화염 스킬 = 데미지 ×1.5. 상성만 잘 맞춰도 난이도가 크게 낮아진다.

6. **장비 강화는 +15까지만 무리 없이.** +16부터 하락 리스크. 천장 시스템이 있으니 극한 강화는 자원이 충분할 때.

7. **던전은 영웅 난이도부터 레이드 토큰이 나온다.** 전설/신화 장비 교환에 필수인 토큰이니, 장비가 갖춰지면 영웅 난이도에 도전.

8. **Ch4에서 북방 영원빙원을 반드시 방문하라.** 우르그롬 합류 조건이자 엔딩 A 필수 조건.

### ❓ FAQ

**Q: P2W 요소가 있나요?**
> A: 없다. 전투력에 영향을 주는 유료 아이템은 일절 없다. 코스메틱과 편의 기능만 유료. 가챠도 없다.

**Q: 솔로 플레이로 엔딩까지 갈 수 있나요?**
> A: 메인 스토리는 전부 솔로 가능. 일부 던전은 파티 추천이지만 스토리 난이도에서는 자동전투로 솔로 클리어 가능.

**Q: 엔딩 A를 놓쳤는데 다시 할 수 있나요?**
> A: 2회차 플레이에서 가능. 1회차에서 놓친 선택지를 다르게 선택하면 된다.

**Q: 마에스트로 크리오는 어떻게 212년을 살았나요?**
> A: 대망각 당시 갓난아기여서 기억이 없었고, 기억이 없으니 망각의 폭풍을 피한 유일한 생존자다. 에테르 장수 효과로 외형은 60대.

**Q: 클래스 변경이 가능한가요?**
> A: 불가. 캐릭터 생성 시 선택한 클래스가 고정된다. 다만 전직 분기에서 2갈래 중 선택 가능.

**Q: 색맹인데 아이템 등급 구분이 가능한가요?**
> A: 가능. 3종 색맹 모드(적녹/녹적/청황) 제공. 색상 외에 아이콘 모양(테두리/빛남/별/왕관)으로도 등급을 구분할 수 있다.

**Q: 한 손으로 플레이 가능한가요?**
> A: 좌측/우측 원핸드 모드를 제공한다. 접근성 설정에서 활성화.

**Q: 봉인 의식의 12인이 뭔가요?**
> A: 212년 전(세계력 3,200년) 대망각 시 레테를 봉인하기 위해 기억을 바친 12인의 기억 수호자. 카일(에리언의 전생)이 핵심 수행자였다.
