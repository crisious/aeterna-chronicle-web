## 1. 수익화 철학

### 1.1 핵심 원칙

| 원칙 | 설명 |
|------|------|
| P2W 제로 | 전투 성능에 직접 영향을 주는 아이템은 유료 판매하지 않는다 |
| 무료 플레이어 완전성 | 모든 게임 콘텐츠(스토리, 던전, 레이드, 엔딩)는 무과금으로 100% 경험 가능 |
| 유료 재화 대체 획득 보장 | 모든 유료 아이템은 게임 내 활동(명성 포인트, 시즌 코인, 업적)으로도 획득 가능 |
| 과금 압박 최소화 | "지금 안 사면 손해"식 설계 금지. 한정 판매는 시즌 단위 복각 보장 |
| 유료/무료 전투력 차이 ≤ 15% | 경험치 부스트 등 편의 아이템 포함 시에도 실질 전투력 격차 15% 이내 유지 |

### 1.2 수익 구조 비율 목표

| 수익원 | 매출 비율 목표 |
|--------|-------------|
| 코스메틱 (스킨/이펙트/마운트) | 45% |
| 시즌 패스 (프리미엄 트랙) | 25% |
| 편의 아이템 | 15% |
| 프리미엄 화폐 직접 판매 | 10% |
| 기타 (이름 변경, 서버 이전 등) | 5% |

## 5. ARPU/ARPPU 시뮬레이션 프레임워크

### 5.1 기본 지표 정의

| 지표 | 정의 | 목표값 |
|------|------|-------|
| DAU | 일일 활성 사용자 | Phase 1: 5,000 / Phase 2: 30,000 / Phase 3: 100,000 |
| MAU | 월간 활성 사용자 | DAU × 3.5 |
| 과금률 (Conversion) | DAU 중 월 1회 이상 결제 비율 | 5~10% |
| ARPU | 전체 유저 1인당 월 평균 매출 | ₩500~₩1,500 |
| ARPPU | 과금 유저 1인당 월 평균 매출 | ₩10,000~₩25,000 |

### 5.2 시뮬레이션 모델

```
월 매출 = MAU × 과금률 × ARPPU

보수적 시나리오 (Phase 2 기준):
  MAU = 100,000
  과금률 = 5%
  ARPPU = ₩12,000
  → 월 매출 = 100,000 × 0.05 × 12,000 = ₩60,000,000 (6천만 원)

기본 시나리오:
  MAU = 100,000
  과금률 = 8%
  ARPPU = ₩18,000
  → 월 매출 = 100,000 × 0.08 × 18,000 = ₩144,000,000 (1억 4,400만 원)

낙관적 시나리오:
  MAU = 100,000
  과금률 = 10%
  ARPPU = ₩25,000
  → 월 매출 = 100,000 × 0.10 × 25,000 = ₩250,000,000 (2억 5천만 원)
```

### 5.3 시즌 패스 매출 기여도 모델

```
시즌 패스 매출 = MAU × 시즌 패스 구매율 × 시즌 패스 가격

기본 시나리오:
  시즌 패스 구매율 = 12% (과금 유저의 ~80%)
  시즌 패스 가격 = ₩14,000 (1,500 EC)
  → 시즌당 시즌 패스 매출 = 100,000 × 0.12 × 14,000 = ₩168,000,000
  → 월 환산 (÷3) = ₩56,000,000 → 전체 매출의 ~39%
```

### 5.4 LTV (Lifetime Value) 추정

```
LTV = ARPPU × 평균 과금 기간 (월)

보수적: ₩12,000 × 6개월 = ₩72,000
기본:   ₩18,000 × 9개월 = ₩162,000
낙관적: ₩25,000 × 12개월 = ₩300,000

CPI (설치당 비용) 목표: LTV의 30% 이하
  → 기본 시나리오 기준 CPI ≤ ₩48,600
```

| 금지 항목 | 이유 | 대체 방안 |
|---------|------|---------|
| 스탯 직접 판매 (영구 STR/INT/DEX 등) | P2W 직결 | 게임 내 스탯 포인트 시스템으로만 성장 |
| 장비 가챠 (랜덤 뽑기로 성능 장비 획득) | 사행성 + P2W | 레이드/퀘스트로 확정 드롭, 제작 시스템 |
| 강화 성공률 증가 아이템 | 과금 압박 유발 | 천장(Pity) 시스템으로 보완 (50회 시도 시 보장) |
| 독점 성능 장비 (과금 전용 스탯 장비) | 무과금 유저 차별 | 최고 성능 장비는 레이드/제작으로만 획득 |
| 스킬 추가 해금/강화 (유료) | 클래스 밸런스 붕괴 | 레벨/퀘스트로만 해금 |
| PvP 성능 부스터 | 공정성 파괴 | PvP는 장비 스코어 평준화 시스템 적용 |
| VIP 등급 (과금액 기반 특권) | 유저 계층화 | 플레이 활동 기반 칭호/보상 체계 |
| 에너지/체력 시스템 (플레이 시간 유료 제한) | 플레이 자유 침해 | 무제한 플레이, 보너스만 일일 제한 |

### 6.2 조건부 제한

| 항목 | 조건 | 상한 |
|------|------|------|
| 경험치 부스트 | 시간 단축만, 스탯 무관 | +100% EXP 상한, 중첩 불가 |
| 자동 수집 | 편의성만, 드롭률 변경 없음 | 필드 아이템만, 보스/레이드 드롭 미적용 |
| 인벤토리 확장 | 동일 기능 게임 내 획득 가능 | 최대 +60칸 (120칸 상한) |
| 부활석 (던전) | 편의성, 던전 보상 변경 없음 | 1회/던전 제한, 레이드 미적용 |

## 7. 운영 모니터링 지표

### 7.1 P2W 방지 KPI

| 지표 | 목표 범위 | 이상 기준 | 대응 |
|------|---------|---------|------|
| 유료/무료 유저 전투력 차이 | < 15% | 20% 초과 | 편의 아이템 효과 하향 |
| 클래스별 레이드 참여 비율 | 20~40% | 한 클래스 50% 초과 | 리밸런싱 패치 |
| 일일 접속 유저 중 결제 비율 | < 10% | 15% 초과 | 게임 내 획득처 보강 |
| 시즌 패스 100단계 완주율 | > 60% | 40% 미만 | 미션 난이도/보상량 조정 |

### 7.2 매출 건전성 KPI

| 지표 | 목표 | 이상 기준 |
|------|------|---------|
| 상위 1% 유저 매출 비중 | < 40% | 60% 초과 시 고래 의존도 과다 |
| 신규 유저 7일 내 첫 결제 비율 | 3~8% | 15% 초과 시 초반 과금 압박 의심 |
| 환불 비율 | < 2% | 5% 초과 시 아이템 만족도 점검 |
| 시즌 패스 갱신율 | > 50% | 30% 미만 시 보상 구조 재검토 |
