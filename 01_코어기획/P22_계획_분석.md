## 2. 목표

챕터 1(에레보스)을 **처음부터 끝까지 플레이 가능한 데모**로 빌드한다. 서버-클라이언트-DB-에셋을 최소한으로 연결하여 핵심 게임플레이 루프를 검증한다.

### 2.1 데모 범위

| 요소 | 포함 범위 |
|------|-----------|
| 지역 | 에레보스 (시작 마을 + 첫 번째 필드 + 첫 번째 던전) |
| 클래스 | 6클래스 중 최소 1클래스 선택 가능 |
| 퀘스트 | 챕터 1 메인 퀘스트 라인 |
| 전투 | 필드 몬스터 + 첫 보스 (기억의 골렘) |
| NPC | 에리언, 초기 상인, 퀘스트 NPC |
| 기억 메카닉 | 첫 번째 기억 파편 수집 |
| UI | 인벤토리, 스탯, 퀘스트 로그 |

### 2.2 기술 스택

| 레이어 | 구성 |
|--------|------|
| DB | PostgreSQL 로컬 + Prisma migrate |
| 서버 | Express API (최소 엔드포인트: auth, character, quest, combat, npc) |
| 클라이언트 | Vite dev 또는 빌드 번들 |
| 에셋 | public/ 또는 static/ 에서 직접 서빙 |
