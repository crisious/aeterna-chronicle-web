## P14 개요 — 데이터/ML + 시즌 4

P12 계획 분석에서 예고한 트랙 F(데이터 분석+ML)와 트랙 G(콘텐츠 확장 시즌 4+)를 병합.  
데이터 인프라가 콘텐츠 밸런스를 지능화하고, 신규 콘텐츠가 데이터 수집의 의미 있는 소스를 제공하는 상호보완 구조.

**핵심 원칙:**
- ClickHouse 웨어하우스 + 이벤트 SDK로 전수 행동 데이터 수집
- ML 이탈 예측/추천/밸런스 자동 튜닝으로 운영 자동화
- 챕터 8 "시간의 균열" + 6번째 클래스 + 신규 지역으로 시즌 4 콘텐츠
- 8인 레이드/펫 진화/PvP 시즌 3으로 엔드게임 확대
- A/B 테스트 + 실시간 대시보드로 데이터 기반 의사결정
