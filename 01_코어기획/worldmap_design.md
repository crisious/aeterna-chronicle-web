## 1. 세계 개요

### 세계명: **에테르나 (Aeterna)**

'영원'을 뜻하는 고대어에서 유래했으나, 아이러니하게도 세계의 기억이 서서히 소멸되고 있는 세계다. 물리적 시간과 기억이 뒤엉킨 이 세계에서는 오래된 기억이 충분히 쌓이면 현실의 일부가 되며, 기억을 잃은 존재는 세계로부터 지워진다.

| 항목 | 내용 |
|------|------|
| 시대 배경 | 세계력 3,412년 — 대망각 발생 후 212년 |
| 기술 수준 | 에테르-기계 문명 (증기기관 + 에테르 결정 동력) |
| 주요 갈등 | 기억 소멸 vs. 기억의 수호 / 칼리마르 제국 vs. 이종족 연합 |
| 대륙명 | 아르카넘 (Arcanum) |
| 주변 바다 | 무한 안개해 (Sea of Endless Mist) |
| 주요 고원 | 망각의 고원 (Plateau of Oblivion) — 대륙 중앙부 |

### 에테르 에너지와 세계의 원리

에테르나 세계의 모든 생명과 문명은 **에테르 에너지**를 중심으로 작동한다.

- **에테르 결정(Ether Crystal)**: 에테르 에너지가 물질화된 광물. 동력원이자 화폐이며 마법의 매개체.
- **기억 연결망**: 살아있는 모든 존재는 에테르 연결망을 통해 세계와 연결되어 있다. 이 연결이 끊기면 기억 소멸 증후군이 발생한다.
- **신성 기억 파편**: 약 212년 전 (세계력 3,200년) 봉인된 열두 신의 기억이 물질화된 파편. 에테르 결정보다 수십 배 강한 에너지를 방출한다.
- **대망각의 안개**: 망각의 고원에서 뿜어져 나오는 기억 소멸 에너지. 현재 진행형으로 확산 중.

## 2. 대륙 구조 — 아르카넘

### 대륙 구역 분류

| 방위 | 주요 지역 | 지형 특성 |
|------|----------|----------|
| 북부 | 북방 영원빙원 | 극지 설원, 에테르 빙정 지대 |
| 중북부 | 아르겐티움 | 평원, 에테르 탑 도시 |
| 중앙 | 망각의 고원 / 에레보스 | 고원, 유령 도시 |
| 동부 | 실반헤임 | 고대 밀림, 기억의 숲 |
| 남부 | 솔라리스 사막 | 광대한 사막, 에테르 결정 매장지 |
| 서부 해안 | 브리탈리아 자유항 | 항구 도시, 안개 해안 |

### 대륙 규모

- **동서 길이**: 약 4,000km (에테르 기관차 기준 7~10일 이동)
- **남북 길이**: 약 3,000km
- **주요 산맥**: 척추산맥(아르겐티움 동쪽) / 빙원산맥(북부 경계)
- **주요 강**: 에테르강 (망각의 고원 → 아르겐티움 관통 → 서부 해안)

### 4-1. 에레보스 (Erebos) — 망각의 폐허

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 위치 | 대륙 중앙, 망각의 고원 기슭 |
| 규모 | 과거 도시 인구 50만 명의 대도시 (현재 무인) |
| 기후 | 안개 낀 흐린 날씨 지속. 빛이 희미하고 낮아도 어둡다 |
| 챕터 연동 | 챕터 1 메인 무대 |
| 등장 NPC | 세라핀 (합류), 마에스트로 크리오 (이전 거주지 흔적) |

#### 지리적 특성

약 212년 전 대망각의 **진원지**. 하룻밤 사이에 시민 50만 명이 존재 자체를 잃은 유령 도시. 건물들은 원형 그대로 서 있지만 주민이 없다. 망각의 고원의 에너지가 이 도시 전체에 스며들어 있어, 간헐적으로 과거의 기억 파편이 홀로그램처럼 재생된다.

**특수 지형 현상**:
- **기억 재현(Memory Echo)**: 특정 구역에서 212년 전 장면이 반투명 홀로그램으로 투영됨
- **기억 소멸 안개**: 도시 전체를 감싸는 회색 안개. 오래 머물수록 단기 기억이 흐릿해짐
- **에테르 결정 고갈 구역**: 망각의 에너지로 에테르 결정이 모두 소진. 조명, 기계 장비 작동 불가

### 4-2. 실반헤임 (Sylvanheim) — 기억의 숲

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 위치 | 대륙 동부, 끝없는 고대 밀림 지대 |
| 규모 | 대륙 동부 전체를 덮는 광대한 숲 |
| 기후 | 숲 내부는 별도의 시간 흐름 (1일 = 외부 3시간) |
| 챕터 연동 | 챕터 2 메인 무대 |
| 주요 종족 | 엘파리스 (나무 인간족) |
| 등장 NPC | 세라핀 (고향), 수호 고목 말라투스 (보스) |

#### 지리적 특성

수천 년 수령의 고대 나무들이 가득한 신비로운 밀림. 나무 자체가 수천 년의 기억을 저장하여 **기억 소멸이 일어나지 않는** 에테르나 유일의 안전 구역. 엘파리스 종족의 성지이며, 숲 내부 시간 흐름이 바깥과 다르게 흘러 오래된 과거의 흔적이 보존되어 있다.

**특수 지형 현상**:
- **시간 지연(Time Delay)**: 숲 내부 1일 = 바깥 세계 3시간. 숲 내에서 장기 체류 시 세계의 변화로부터 격리됨
- **기억 수호(Memory Guard)**: 이 숲에서는 기억 소멸이 일어나지 않음. MDS 환자도 일시적으로 증상이 정지됨
- **나무와의 교감**: 에테르 공명 능력 보유자는 숲 나무들과 직접 교감 가능. 과거 기억 열람 가능
- **생물 발광**: 야간에 균류와 이끼류가 발광. 환상적 경관

#### 세부 구역 및 랜드마크

- **동면 동굴**: 우르살리스 종족의 동면 공간. 내부는 이상하게 따뜻하고 에테르 결정이 벽에 박혀 희미하게 빛남

#### 지리적 특성

어느 국가에도 속하지 않는 **독립 자치 도시**이자 에테르나 최대 무역항. 모든 종족과 세력의 상인·용병·정보상·밀수업자들이 모여드는 혼돈의 허브. 바다 안개 속에서 모습을 드러내는 유령선들과 **잊혀진 섬** 전설로 유명.

**특수 지형 현상**:
- **무한 안개(Endless Mist)**: 항구 바깥을 둘러싼 안개. 안개 속에서 방향 감각 상실 가능. 특수 항법 도구 필요
- **유령선 출몰**: 안개 속에서 불빛 없는 선박이 간헐적으로 목격됨. 접근 시 과거 선원들의 기억 공명 발동
- **정보 밀도**: 에테르나에서 정보가 가장 빠르게 순환하는 곳. NPC들이 모든 지역의 소문 보유

#### 선술집 벽 낙서의 세계관적 의미

브리탈리아 '닻과 별' 선술집 벽에는 수십 년간 수많은 사람들이 남긴 낙서가 가득하다. 이 낙서들은 단순한 장식이 아니라 에테르나의 역사와 소문, 경고를 담은 **살아있는 아카이브**다.

- 피에트로-레나의 연인 낙서 → 세계의 평범한 일상성
- K의 잊혀진 섬 메모 → 히든 서브 퀘스트 단서
- "레테 교단을 믿지 마라" → 전직 교단원 R의 경고
- "실반헤임의 말라투스를 보았다. 가지 마라." → 챕터 2 복선

- **날씨**: 안개가 짙게 끼는 날이 많음. 갑자기 바람이 강해지는 날씨 변화
- **사운드**: 뱃고동 소리, 파도 소리, 선술집 재즈풍 음악, 선원들의 떠들썩한 대화
- **특이 효과**: 항구에서 바다를 바라볼 때 안개 속에 유령선 실루엣 깜빡임
