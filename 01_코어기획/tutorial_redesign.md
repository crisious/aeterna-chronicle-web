# 에테르나 크로니클 — 튜토리얼 재설계

> **작성일:** 2026-03-13 | **버전:** v1.0 | **분류:** 코어기획 — 접근성  
> **티켓:** P17-16 (#324) | **선행:** 없음

---

## 1. 개요

기존 선형 튜토리얼을 **모듈식/스킵 가능** 구조로 전면 재설계한다. 신규 유저의 학습 부담을 최소화하면서, 숙련자는 즉시 게임에 진입할 수 있도록 한다.

### 1.1 설계 원칙

| 원칙 | 설명 |
|------|------|
| **모듈식** | 독립 완결형 튜토리얼 모듈, 순서 무관 수강 가능 |
| **스킵 가능** | 모든 모듈을 개별/일괄 스킵 가능 |
| **재열람** | 설정 메뉴에서 언제든 다시 볼 수 있음 |
| **비침습적** | 게임 흐름을 끊지 않는 인라인 팁 우선 |
| **상황 기반** | 첫 조우 이벤트에서 자연스럽게 학습 유도 |

---

## 2. 모듈 구조도

```
튜토리얼 시스템
├── 📌 필수 모듈 (스킵 불가 → "숙련자 모드" 선택 시 일괄 스킵)
│   └── M0: 기본 이동 + 전투 (3분)
│
├── 📖 선택 모듈 (첫 조우 시 팝업 제안, 거부 가능)
│   ├── M1: 스킬 시스템 (2분)
│   ├── M2: 인벤토리 + 장비 (2분)
│   ├── M3: 던전 입장 (2분)
│   ├── M4: PvP 아레나 (2분)
│   ├── M5: 제작 시스템 (2분)
│   └── M6: 기억 파편 + 클래스 (2분)
│
└── 💡 상황 팁 (일회성 인라인 팝업)
    ├── T1: 첫 보스 조우
    ├── T2: 첫 사망
    ├── T3: 첫 아이템 드롭
    ├── T4: 첫 레벨업
    ├── T5: 첫 파티 합류
    └── T6: 첫 Active Pause 사용 유도
```

## 3. 모듈 상세

### M0: 기본 이동 + 전투 (필수, 3분)

| 단계 | 시간 | 내용 | 트리거 |
|------|------|------|--------|
| 1 | 30초 | WASD 이동 연습 — 마커 3곳 순서대로 이동 | 캐릭터 생성 직후 |
| 2 | 30초 | 카메라 조작 — 마우스/키보드 회전 | 이동 완료 |
| 3 | 60초 | 기본 공격 — 더미 대상 3체 격파 | 카메라 완료 |
| 4 | 60초 | 첫 전투 — 약한 적 2체 조우, Active Pause 안내 | 공격 완료 |

- **스킵 조건:** "숙련자 모드" 선택 시에만 스킵 (개별 스킵 불가)
- **완료 보상:** 기본 포션 5개

### M1: 스킬 시스템 (선택, 2분)

| 단계 | 시간 | 내용 | 트리거 |
|------|------|------|--------|
| 1 | 40초 | 스킬 슬롯 설명 — 1~8번 키 안내 | 첫 스킬 습득 시 |
| 2 | 40초 | 스킬 장착 — 인벤토리에서 슬롯 배치 | 설명 완료 |
| 3 | 40초 | 스킬 사용 — 더미 대상에 시전 연습 | 장착 완료 |

- **트리거:** 레벨 2 달성 또는 첫 스킬 습득
- **거부 시:** "나중에 설정 > 튜토리얼에서 볼 수 있어요" 안내

### M2: 인벤토리 + 장비 (선택, 2분)

| 단계 | 시간 | 내용 | 트리거 |
|------|------|------|--------|
| 1 | 30초 | 인벤토리 열기 (I키) + 정렬/필터 | 첫 장비 아이템 획득 |
| 2 | 30초 | 장비 비교 — 스탯 비교 UI 안내 | 인벤토리 열기 |
| 3 | 30초 | 장비 장착 — 드래그 또는 더블클릭 | 비교 완료 |
| 4 | 30초 | 아이템 등급 색상 + 아이콘 식별자 안내 | 장착 완료 |

### M3: 던전 입장 (선택, 2분)

| 단계 | 시간 | 내용 | 트리거 |
|------|------|------|--------|

| 2 | 40초 | 던전 UI — 보상/소요시간/추천레벨 안내 | 입구 상호작용 |
| 3 | 40초 | 부활 시스템 — 체크포인트/부활 횟수 안내 | UI 확인 |

|------|------|------|--------|

| 3 | 40초 | 클래스 특성 — 선택한 클래스 기본 스킬 체험 | 프리뷰 확인 |

**클래스 ID 참조:** `ether_knight` / `memory_weaver` / `shadow_weaver` / `memory_breaker` / `time_guardian` / `void_wanderer`

## 4. 상황 팁 시스템

게임 진행 중 특정 이벤트 **첫 발생** 시 화면 하단에 비침습적 팝업(3초 자동 소멸, 클릭 시 상세).

| ID | 트리거 | 팝업 텍스트 | 상세 내용 |
|----|--------|------------|-----------|
| T1 | 첫 보스 조우 | "💡 Active Pause(Space)로 전투를 멈추고 전략을 세워보세요" | 보스 패턴/텔레그래프 설명 |
| T2 | 첫 사망 | "💀 체크포인트에서 부활합니다. 난이도를 조절할 수 있어요" | 난이도 설정 바로가기 |
| T3 | 첫 아이템 드롭 | "✨ 아이템을 주웠습니다! I키로 인벤토리를 열어보세요" | 인벤토리 튜토리얼 연결 |
| T4 | 첫 레벨업 | "⬆️ 레벨업! 새로운 스킬을 배울 수 있습니다" | 스킬 트리 안내 |
| T5 | 첫 파티 합류 | "🤝 파티에 참여했습니다. 파티원의 역할을 확인하세요" | 파티 UI 안내 |
| T6 | 첫 전투 10초 이상 | "⏸ Space를 눌러 Active Pause! 전투 중 잠시 멈출 수 있어요" | Active Pause 상세 |

### 4.1 팝업 설정

| 옵션 | 기본값 | 범위 |
|------|--------|------|
| 상황 팁 ON/OFF | ON | — |
| 팝업 지속 시간 | 3초 | 1~10초 |
| 팝업 위치 | 하단 중앙 | 상단/하단/좌하/우하 |
| 이미 본 팁 재표시 | OFF | — |

## 5. 복습 시스템

### 5.1 설정 메뉴 재열람

```
⚙️ 설정 > 🎓 튜토리얼
├── M0: 기본 이동 + 전투    [✅ 완료] [다시보기]
├── M1: 스킬 시스템         [✅ 완료] [다시보기]
├── M2: 인벤토리 + 장비     [⬜ 미완료] [시작]
├── M3: 던전 입장           [⬜ 미완료] [시작]
├── M4: PvP 아레나          [🔒 레벨 10+] [—]
├── M5: 제작 시스템         [🔒 레벨 5+] [—]
├── M6: 기억 파편 + 클래스  [⬜ 미완료] [시작]
│
├── 💡 상황 팁 초기화       [모든 팁 다시 보기]
└── 🎯 연습 모드            [입장]
```

### 5.2 연습 모드

| 항목 | 내용 |
|------|------|
| 위치 | 전용 연습장 (안전 구역, 세계관: 훈련장) |
| 더미 대상 | HP/방어력 조절 가능한 더미 5체 상시 배치 |
| 스킬 제한 | 없음 — 모든 보유 스킬 자유 사용 |
| 쿨타임 | 선택적 초기화 (쿨타임 OFF 토글) |
| 데미지 표시 | DPS 미터 + 총 데미지 + 스킬별 비중 |
| 나가기 | 언제든 즉시 퇴장 (진행 상태 미영향) |
