# 에테르나 크로니클 — 접근성 설계

> **작성일:** 2026-03-10 | **버전:** v1.0 | **분류:** 코어기획 — UX/접근성

---

## 1. 개요

본 문서는 에테르나 크로니클의 접근성(Accessibility) 설계를 정의한다. 시각, 청각, 조작, 인지 영역의 접근성 기능과 WCAG 2.1 AA 기준 충족 방안을 포함한다.

### 1.1 설계 원칙

| 원칙 | 설명 |
|------|------|
| **보편적 설계** | 접근성 기능이 모든 유저의 경험을 향상시키도록 설계 |
| **기본 활성화** | 핵심 접근성 기능은 기본 ON (옵트아웃 방식) |
| **성능 무영향** | 접근성 기능 활성화가 프레임레이트에 영향 없음 |
| **점진적 구현** | 출시 시 필수 기능 → 업데이트로 확장 |

## 2. 시각 접근성

### 2.1 색맹 모드

3종 색맹 타입에 대응하는 색상 팔레트를 제공한다:

| 모드 | 대상 | 조정 내용 |
|------|------|-----------|
| **적녹색맹 (Protanopia)** | 적색-녹색 구분 불가 | 적→주황, 녹→청록으로 대체 |
| **녹적색맹 (Deuteranopia)** | 녹색-적색 구분 불가 | 녹→파랑, 적→주황으로 대체 |
| **청황색맹 (Tritanopia)** | 청색-황색 구분 불가 | 청→보라, 황→분홍으로 대체 |

**등급 색상 매핑 (색맹 모드 적용 시):**

| 등급 | 기본 색상 | 적녹색맹 | 녹적색맹 | 청황색맹 | 추가 식별자 |
|------|-----------|----------|----------|----------|------------|
| 일반 | 회색 | 회색 | 회색 | 회색 | 테두리 없음 |
| 고급 | 초록 | 청록 | 파랑 | 초록 | 단선 테두리 |
| 희귀 | 파랑 | 파랑 | 남색 | 보라 | 이중 테두리 |
| 영웅 | 보라 | 보라 | 보라 | 분홍 | 빛남 이펙트 |
| 전설 | 주황 | 주황 | 주황 | 주황 | 빛남 + 별 아이콘 |
| 신화 | 빨강 | 주황-빨강 | 주황 | 빨강 | 빛남 + 왕관 아이콘 |

**핵심 규칙:** 색상만으로 정보를 전달하지 않는다. 모든 색상 구분에는 **형태(아이콘/테두리/패턴)** 보조 식별자가 함께 제공된다.

### 2.2 고대비 모드

| 항목 | 기본 | 고대비 |
|------|------|--------|
| 배경-텍스트 대비 | 4.5:1 | 7:1 이상 |
| UI 테두리 | 1px, 반투명 | 2px, 100% 불투명 백색/흑색 |
| 아이콘 | 컬러 아이콘 | 고대비 단색 아이콘 (윤곽선 강조) |
| 미니맵 | 반투명 오버레이 | 불투명 배경 + 진한 마커 |
| 바닥 텔레그래프 | 반투명 색상 영역 | 진한 색상 + 격자 패턴 + 깜빡임 |
| 적/아군 구분 | 빨강/파랑 | 빨강/파랑 + 아군 흰색 윤곽선, 적 검정 윤곽선 |

### 2.3 UI 스케일

| 항목 | 값 |
|------|-----|
| 조정 범위 | 50% ~ 200% (10% 단위) |
| 기본값 | 100% |
| 적용 대상 | 모든 UI 요소 (HUD, 메뉴, 대화창, 인벤토리, 미니맵) |
| 독립 스케일 | HUD / 대화창 / 미니맵 개별 스케일 조정 가능 |
| 폰트 크기 | 별도 조정 (기본 14px, 범위 10~28px) |

### 2.4 추가 시각 기능

| 기능 | 설명 | 기본값 |
|------|------|--------|
| **화면 흔들림 감소** | 타격/폭발 시 카메라 쉐이크 강도 0~100% | 100% |
| **번쩍임 감소** | 스킬 이펙트 번쩍임 강도 조절 (광과민성 보호) | ON (70%) |
| **모션 감소** | 배경 파티클, 환경 애니메이션 간소화 | OFF |
| **포커스 하이라이트** | 현재 선택/타겟 대상에 강조 윤곽선 | ON |
| **데미지 숫자 크기** | 플로팅 데미지 텍스트 크기 조절 (50~200%) | 100% |
| **미니맵 확대** | 미니맵 줌 레벨 독립 조정 | 100% |

## 3. 청각 접근성

### 3.1 자막 시스템

| 항목 | 옵션 |
|------|------|
| 자막 ON/OFF | 기본 ON |
| 자막 크기 | 소/중/대/특대 (기본: 중) |
| 자막 배경 | 없음/반투명 검정/불투명 검정 (기본: 반투명) |
| 자막 색상 | 흰색/노랑/청록/초록 (기본: 흰색) |
| 화자 이름 표시 | ON/OFF (기본: ON). 화자별 색상 구분 |
| 효과음 자막 | ON/OFF. [검의 충돌], [폭발], [발걸음] 등 환경음 표기 |
| 자막 위치 | 하단/상단/좌측 (기본: 하단 중앙) |
| 자막 표시 시간 | 1~10초 (기본: 3초) |

### 3.2 시각적 사운드 인디케이터

청각 정보를 시각적으로 대체하는 시스템:

| 사운드 유형 | 시각 표현 | 위치 |
|------------|-----------|------|
| **적 발견 경고** | 화면 가장자리 붉은 펄스 (방향 표시) | 화면 테두리 |
| **피격** | 피격 방향 빨간 화살표 | 캐릭터 주변 |
| **스킬 시전 완료** | 스킬 아이콘 밝은 테두리 펄스 | HUD 스킬바 |
| **아이템 드롭** | 바닥 빛기둥 + 미니맵 마커 | 월드 + 미니맵 |
| **NPC 대화 가능** | 머리 위 말풍선 아이콘 점멸 | NPC 위치 |
| **보스 페이즈 전환** | 화면 전체 플래시 + 페이즈 번호 텍스트 | 화면 중앙 |
| **버프/디버프 만료 임박** | 아이콘 깜빡임 (3초 전부터) | HUD 버프바 |
| **위험 영역** | 바닥 패턴 애니메이션 (깜빡이는 격자) | 월드 |

### 3.3 오디오 설정

## 4. 조작 접근성

### 4.1 키보드 전용 플레이

게임의 모든 기능을 키보드만으로 수행 가능하도록 보장한다:

| 기능 | 기본 키 | 비고 |
|------|---------|------|
| 이동 | WASD | - |
| 스킬 1~8 | 1~8 | 스킬 슬롯 |
| 소비아이템 1~4 | F1~F4 | 아이템 슬롯 |
| Active Pause | Space | 일시정지 토글 |
| 타겟 전환 | Tab | 가장 가까운 적 순환 |
| NPC 상호작용 | E | 대화/상점/퀘스트 |
| 인벤토리 | I | 토글 |
| 캐릭터 정보 | C | 토글 |
| 지도 | M | 토글 |
| 메뉴 탐색 | 방향키 / Tab | UI 내 포커스 이동 |
| 메뉴 확인 | Enter / Space | 선택 실행 |
| 메뉴 취소 | Esc | 뒤로/닫기 |

### 4.2 키 리바인딩

| 항목 | 설명 |
|------|------|
| 범위 | 모든 게임 조작 키 재설정 가능 |
| 제한 | Esc(시스템 메뉴)만 고정, 나머지 전부 변경 가능 |
| 프리셋 | 기본 / WASD / 방향키 / 커스텀 3개 저장 |
| 충돌 경고 | 중복 키 설정 시 경고 표시 (강제 허용은 가능) |
| 매크로 | 단일 키에 복합 입력 불가 (치트 방지) |

### 4.3 원핸드 모드

한 손으로 플레이 가능한 레이아웃:

**좌측 원핸드:**
- 이동: WASD
- 스킬: Q/E/R/F/Z/X/C/V (8슬롯)
- 소비아이템: 1/2/3/4
- Active Pause: Tab
- 타겟: Caps Lock

**우측 원핸드:**
- 이동: 방향키
- 스킬: Numpad 1~8
- 소비아이템: Numpad +/-/*/Enter
- Active Pause: Numpad 0
- 타겟: Numpad .

### 4.4 마우스 조작 대체

| 기능 | 마우스 | 키보드 대체 |
|------|--------|------------|
| 타겟 지정 | 좌클릭 | Tab (순환) + Enter (확정) |
| 이동 (클릭이동) | 우클릭 | WASD |
| 카메라 회전 | 마우스 드래그 | Q/E 또는 [/] |
| UI 항목 선택 | 클릭 | 방향키 + Enter |
| 툴팁 | 호버 | 포커스 시 자동 표시 |

### 4.5 입력 보조

| 기능 | 설명 | 기본값 |
|------|------|--------|
| **키 입력 딜레이** | 키 반복 입력 인식 간격 조절 (50~500ms) | 100ms |
| **동시 입력 허용 시간** | 두 키를 "동시"로 인식하는 시간 창 (0~200ms) | 50ms |
| **토글 모드** | 길게 누르기 → 한 번 누르면 활성/한 번 더 누르면 해제 | OFF |
| **자동 달리기** | 이동 키 더블탭 시 자동 달리기 토글 | ON |
| **스킬 퀵캐스트** | 키를 누르면 즉시 시전 (범위 확인 생략) | OFF |
| **스킬 노멀캐스트** | 키를 누르면 범위 표시 → 다시 누르면 시전 | ON |

## 5. 난이도 접근성

### 5.1 난이도 설정

| 난이도 | 대상 | 주요 조정 |
|--------|------|-----------|
| **스토리** | 스토리만 즐기고 싶은 유저 | 자동전투 ON, 적 HP -50%, 적 데미지 -70%, 부활 무제한 |
| **일반** | 대부분의 유저 | 기본 밸런스 |
| **하드코어** | 고난이도 선호 유저 | 적 HP +30%, 적 데미지 +50%, 부활 제한(던전당 3회), 포션 효과 -30% |

### 5.2 난이도별 상세 조정

| 파라미터 | 스토리 | 일반 | 하드코어 |
|----------|--------|------|----------|
| 적 HP 배율 | ×0.5 | ×1.0 | ×1.3 |
| 적 데미지 배율 | ×0.3 | ×1.0 | ×1.5 |
| 적 AI 수준 | 기본 FSM (회피 안 함) | 정상 | 강화 (회피+반격 추가) |
| 부활 | 제자리 즉시 부활, 무제한 | 체크포인트 부활, 던전 5회 | 체크포인트 부활, 던전 3회 |
| 포션 효과 | ×1.5 | ×1.0 | ×0.7 |
| 보상 | 경험치 ×0.7, 드롭률 ×0.8 | ×1.0 | 경험치 ×1.3, 드롭률 ×1.2 |
| 자동전투 | 기본 ON | 선택적 ON | 비활성 |
| 보스 텔레그래프 | 확장 표시 (2초 추가) | 기본 | 축소 표시 (-0.5초) |

### 5.3 자동전투 시스템 (스토리 난이도)

| 기능 | 설명 |
|------|------|
| 자동 이동 | 퀘스트 목표 지점으로 자동 네비게이션 |
| 자동 공격 | AI가 최적 스킬 로테이션 수행 |
| 자동 포션 | HP 50% 이하 시 자동 사용 |
| 자동 타겟 | 가장 가까운 적 자동 타겟 |
| 수동 전환 | 아무 전투 키 입력 시 즉시 수동으로 전환, 5초 무입력 시 자동 복귀 |

## 6. 인지 접근성

### 6.1 정보 표시 조절

| 기능 | 설명 | 기본값 |
|------|------|--------|
| **퀘스트 가이드** | 화면 내 퀘스트 목표 방향 화살표 | ON |
| **미니맵 마커** | 퀘스트/NPC/적 위치 표시 수준 (없음/최소/전체) | 전체 |
| **데미지 숫자** | 플로팅 데미지 텍스트 (없음/본인만/전체) | 전체 |
| **버프 타이머** | 버프/디버프 남은 시간 숫자 표시 | ON |
| **스킬 쿨타임** | 쿨타임 남은 초 숫자 표시 | ON |
| **적 HP 표시** | 적 HP바 위 퍼센트 숫자 | ON |
| **튜토리얼 리플레이** | 이전 튜토리얼 다시 보기 | 설정 메뉴에서 |
| **단순 UI 모드** | 불필요한 UI 요소 숨김 (데미지 숫자, 경험치 팝업 등) | OFF |

### 6.2 일시정지 활용

Active Pause(스페이스바)가 인지 접근성의 핵심 도구:
- 전투 중 언제든 일시정지하여 상황 파악 가능
- 정지 중 스킬 설명, 적 정보, 동료 상태 확인 가능
- 정지 중 동료 NPC에게 명령 가능
- 스토리 난이도에서는 자동 일시정지 옵션 (보스 페이즈 전환 시)

## 8. 접근성 테스트 체크리스트

릴리즈 전 **필수** 통과 항목:

### 8.1 시각 테스트

- [ ] 색맹 3종 모드에서 등급 구분 가능 여부
- [ ] 고대비 모드에서 모든 텍스트 가독성 (7:1 대비)
- [ ] UI 스케일 50%, 100%, 200%에서 레이아웃 깨짐 없음
- [ ] 폰트 최소(10px)/최대(28px)에서 잘림/겹침 없음
- [ ] 번쩍임 감소 모드에서 3회/초 이상 번쩍임 없음
- [ ] 화면 흔들림 0%에서 카메라 완전 고정 확인
- [ ] 바닥 텔레그래프가 색맹+고대비 모드에서 식별 가능
- [ ] 데미지 숫자 OFF 시 완전히 숨겨지는지 확인

### 8.2 청각 테스트

- [ ] 자막 ON 시 모든 대사/환경음 자막 표시
- [ ] 효과음 자막에 방향 정보 포함 ([왼쪽에서 검 소리])
- [ ] 음소거 상태에서 시각 인디케이터만으로 모든 전투 정보 파악 가능
- [ ] 모노 오디오 모드에서 좌우 정보 손실 없음 (시각 보조)
- [ ] 자막 크기/배경/색상 변경이 실시간 반영

### 8.3 조작 테스트

- [ ] 키보드 전용으로 메인 퀘스트 챕터 1 완전 클리어 가능
- [ ] Tab/Enter로 모든 메뉴 항목 접근 가능
- [ ] 포커스 트랩 없음 (모든 모달에서 Esc로 탈출)
- [ ] 원핸드 모드(좌/우)로 전투 3회 완료 가능
- [ ] 키 리바인딩 후 충돌 경고 정상 동작
- [ ] 퀵캐스트/노멀캐스트 전환 즉시 반영

### 8.4 인지 테스트

- [ ] 스토리 난이도에서 자동전투로 챕터 1 클리어 가능
- [ ] 퀘스트 가이드 OFF 시에도 NPC 대화로 목표 파악 가능
- [ ] Active Pause 중 모든 정보 패널 정상 표시
- [ ] 튜토리얼 리플레이가 설정에서 접근 가능

### 8.5 성능 테스트

- [ ] 모든 접근성 기능 동시 활성화 시 60fps 유지 (중간 사양 기준)
- [ ] 접근성 설정 변경 시 로딩/끊김 없이 실시간 적용
