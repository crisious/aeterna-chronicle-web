## 1. 펫 시스템 개요

### 1.1 설계 철학

| 원칙 | 설명 |
|------|------|
| 세계관 연동 | 펫은 에테르나 세계의 생물/정령/기억체로, 세계관과 유기적으로 연결 |
| 전투 보조 (비주력) | 펫은 보조적 패시브/액티브 역할. 주력 전투는 플레이어 캐릭터 중심 |
| P2W 제로 유지 | 펫 외형은 유료 판매 가능, 펫 전투 성능은 게임 내 획득/성장만 |
| 수집 재미 | 지역별/시즌별 고유 펫으로 수집 동기 부여 |

### 1.2 기본 사양

| 항목 | 수치 |
|------|------|
| 펫 인벤토리 | 20칸 (game_systems.md 기준) |
| 동시 소환 | 1마리 |
| 펫 최대 레벨 | 50 |
| 펫 등급 | 6단계 (아이템 등급 체계와 동일) |
| 펫 스킬 슬롯 | 최대 3개 (등급에 따라 해금) |
| Phase 도입 | Phase 1: 기본 5종 / Phase 2: 20종 / Phase 3: 40종+ |

## 2. 펫 획득 방법

### 2.1 획득 경로

| 경로 | 설명 | 획득 가능 등급 |
|------|------|-------------|
| **퀘스트 보상** | 메인/서브 퀘스트 완료 시 지급 | 고급~영웅 |
| **필드 드롭** | 특정 몬스터 처치 시 '펫 알' 드롭 | 일반~희귀 |
| **던전/레이드 보상** | 보스 처치 후 확률 드롭 | 희귀~전설 |
| **제작 (크래프팅)** | 펫 재료 수집 → 제작대에서 부화 | 고급~영웅 |
| **이벤트 보상** | 시즌/이벤트 참여 보상 | 영웅~신화 |
| **NPC 친밀도** | 특정 NPC 친밀도 최대 달성 시 | 영웅 (NPC 연관 고유 펫) |
| **탐험 발견** | 히든 장소에서 발견 | 희귀~전설 |

### 2.2 지역별 고유 펫

| 지역 | 고유 펫 (대표) | 등급 | 획득 방법 |
|------|-------------|------|---------|
| 에레보스 | 기억 위습 (Memory Wisp) | 희귀 | Ch.1 메인 퀘스트 보상 |
| 실반헤임 | 빛나무 요정 (Glow Sprite) | 영웅 | 숲 심층부 탐험 발견 |
| 솔라리스 사막 | 모래 도마뱀 (Sand Drake) | 고급 | 사막 몬스터 드롭 |
| 북방 영원빙원 | 빙정 여우 (Frost Fox) | 영웅 | 에테르 빙정 제작 |
| 브리탈리아 | 안개 고양이 (Mist Cat) | 희귀 | 크리오 친밀도 보상 |
| 아르겐티움 | 에테르 기어 (Ether Gear) | 전설 | Ch.4 레이드 드롭 |

### 2.3 펫 알 부화 시스템

필드 드롭/제작으로 획득한 '펫 알'은 부화 과정 필요:

| 단계 | 내용 | 소요 시간 |
|------|------|---------|
| 알 획득 | 인벤토리에 '펫 알' 보관 | — |
| 부화 시작 | 펫 관리소 NPC에게 맡기기 | 실시간 4~24시간 (등급별 상이) |
| 부화 촉진 | 에테르 결정 투입으로 시간 단축 (게임 내 화폐) | 결정 100개당 1시간 단축 |
| 부화 완료 | 펫 인벤토리에 등록 | — |

> **부화 결과:** 일반~희귀 알은 등급 확정. 영웅+ 알은 부화 시 기본 스킬 랜덤 결정.

## 3. 펫 등급 체계

### 3.1 등급별 스펙

| 등급 | 색상 | 기본 스탯 배율 | 스킬 슬롯 | 최대 진화 단계 |
|------|------|-------------|---------|-------------|
| 일반 (Common) | 회색 | ×1.0 | 1개 | 1단계 |
| 고급 (Uncommon) | 초록 | ×1.3 | 1개 | 2단계 |
| 희귀 (Rare) | 파랑 | ×1.6 | 2개 | 2단계 |
| 영웅 (Epic) | 보라 | ×2.0 | 2개 | 3단계 |
| 전설 (Legendary) | 주황 | ×2.5 | 3개 | 3단계 |
| 신화 (Mythic) | 빨강 | ×3.5 | 3개 + 고유 스킬 | 3단계 |

### 3.2 펫 스탯

| 스탯 | 설명 | 기본값 (Lv.1 일반) |
|------|------|-----------------|
| 펫 HP | 펫의 체력 (0 시 귀환, 60초 후 재소환 가능) | 100 |
| 펫 공격력 | 액티브 스킬 데미지 기준 | 10 |
| 펫 방어력 | 피해 감소 | 5 |
| 수집 범위 | 자동 수집 반경 (칸) | 2 |
| 이동 속도 | 주인 추종 속도 | 1.0 |

## 4. 펫 성장 시스템

### 4.1 레벨 성장

| 항목 | 내용 |
|------|------|
| 경험치 획득 | 주인 전투 시 자동 획득 (주인 EXP의 10%) |
| 레벨당 스탯 증가 | 등급 배율 × (HP +5, 공격력 +1, 방어력 +0.5) |
| 최대 레벨 | 50 |
| 레벨 50 필요 누적 EXP | ~500,000 (주인 Lv.50~60 구간에서 자연 달성) |

### 4.2 진화 시스템

일정 조건 충족 시 펫의 외형과 능력이 강화:

| 진화 단계 | 조건 | 효과 |
|---------|------|------|
| 1단계 → 2단계 | 펫 Lv.20 + 진화 재료 3종 | 외형 변화, 스탯 +30%, 스킬 슬롯 +1 (해당 시) |
| 2단계 → 3단계 | 펫 Lv.40 + 진화 재료 5종 + 특수 퀘스트 | 외형 변화, 스탯 +50%, 고유 패시브 강화 |

**진화 재료:**
- **에테르 정수:** 필드/던전 몬스터 드롭 (공용)
- **종족별 결정:** 지역 특산 (기억 결정/화염 결정/빙정 결정 등)
- **진화의 돌:** 레이드 보상 또는 명성 상점 구매

### 4.3 스킬 시스템

| 스킬 유형 | 설명 | 슬롯 |
|---------|------|------|
| 패시브 스킬 | 상시 발동 (주인 스탯 보정, 상태이상 저항 등) | 자동 장착 |
| 액티브 스킬 | 쿨다운 기반 자동 발동 | 슬롯 장착 |
| 고유 스킬 | 신화 등급 전용, 교체 불가 | 고정 |

**패시브 스킬 풀:**

| 스킬명 | 효과 | 획득 조건 |
|--------|------|---------|
| 에테르 감응 | 주인 MP 회복 +10% | 에테르 계열 펫 기본 |
| 강인한 동반자 | 주인 최대 HP +5% | 우르살리스 계열 펫 |
| 날카로운 눈 | 주인 크리티컬 확률 +3% | 사냥 계열 펫 |
| 행운의 발걸음 | 아이템 드롭률 +5% | 희귀+ 펫 Lv.20 달성 |
| 기억 보존 | 기억 파편 획득량 +10% | 기억 위습 계열 전용 |
| 불꽃 저항 | 화염 속성 피해 -10% | 화염 계열 펫 |

**액티브 스킬 풀:**

| 스킬명 | 효과 | 쿨다운 | 등급 제한 |
|--------|------|--------|---------|
| 돌진 | 적에게 펫 공격력 ×150% 데미지 | 15초 | 고급+ |
| 회복의 숨결 | 주인 HP 5% 즉시 회복 | 30초 | 희귀+ |
| 도발 | 3초간 적 어그로 끌기 | 20초 | 영웅+ |
| 에테르 폭발 | 주변 적 전체에 펫 공격력 ×200% 데미지 | 45초 | 전설+ |
| 기억 방벽 | 주인에게 3초간 기억 실드 (다음 상태이상 1회 면역) | 60초 | 전설+ |

### 4.4 스킬 학습

| 방법 | 설명 |
|------|------|
| 레벨업 자동 학습 | 등급에 따라 특정 레벨에서 자동 습득 |
| 스킬 서적 | 드롭/제작으로 획득한 서적 사용 → 새 스킬 학습 |
| 스킬 교체 | 학습한 스킬 중 슬롯에 장착할 스킬 선택 (자유 교체) |
| 스킬 강화 | 동일 스킬 서적 중복 사용 시 스킬 레벨 +1 (최대 Lv.5) |

## 5. 전투 기여

### 5.1 전투 행동 패턴

| 상태 | 펫 행동 |
|------|--------|
| 비전투 | 주인 뒤를 따라다님, 자동 수집 활성 |
| 전투 진입 | 주인 근처에서 액티브 스킬 쿨다운마다 자동 발동 |
| 주인 HP < 30% | 회복 스킬 우선 발동 (보유 시) |
| 펫 HP = 0 | 자동 귀환, 60초 쿨다운 후 재소환 가능 |

### 5.2 자동 수집

| 항목 | 설명 |
|------|------|
| 수집 대상 | 필드 드롭 아이템 (몬스터 처치 후 바닥 아이템) |
| 수집 범위 | 펫 주변 2~4칸 (펫 레벨/등급에 따라 증가) |
| 수집 속도 | 1초당 1개 |
| 제외 대상 | 보스/레이드 드롭, 퀘스트 아이템 (수동 획득 필요) |
| 유료 자동 수집과의 관계 | 유료 아이템은 '즉시 수집' (범위 무관), 펫은 '범위 내 자동' → 중첩 가능하나 효과 차등 |

### 5.3 전투 기여 밸런스

| 항목 | 수치 |
|------|------|
| 펫 DPS 기여율 (주인 대비) | 5~15% (등급별) |
| 펫 패시브 버프 합계 | 최대 +15% 스탯 보정 |
| 펫 사망 페널티 | 60초 재소환 쿨다운 (패시브 유지, 액티브/자동수집 중단) |

> **P2W 방지:** 펫 전투 성능은 게임 내 성장으로만 결정. 유료 아이템으로 펫 스탯/스킬 강화 불가.

## 6. 펫 외형 커스터마이징

### 6.1 커스터마이징 항목

| 항목 | 설명 | 획득 |
|------|------|------|
| 펫 코스튬 | 펫 의상/액세서리 (모자, 망토, 안경 등) | 유료(EC) / 시즌 패스 / 이벤트 |
| 펫 이펙트 | 이동/대기 시 시각 이펙트 (후광, 발자국, 오라) | 유료(EC) / 업적 보상 |
| 펫 탑승 외형 | 펫 크기가 충분한 경우 탑승 외형 (마운트 기능 아님, 시각 연출만) | 유료(EC) |
| 이름 변경 | 펫 이름 자유 설정 | 무료 (최초), 이후 이름 변경권 |

### 6.2 수익화 연동

| 카탈로그 (monetization_design.md 연동) | 가격 (EC) |
|--------------------------------------|----------|
| 펫 코스튬 | 200 EC |
| 펫 이펙트 | 300 EC |
| 펫 탑승 외형 | 500 EC |

> **외형 = 순수 코스메틱.** 스탯/스킬 영향 없음. P2W 원칙 준수.

## 7. UI 설계

### 7.1 펫 관리 화면

```
┌─────────────────────────────────────────┐
│  [펫 관리]                    [정렬 ▼]  │
├────────┬────────────────────────────────┤
│ 펫 목록 │  선택된 펫 상세 정보           │
│ (20칸)  │  ┌──────────────────────┐     │
│         │  │ [펫 3D/2D 프리뷰]    │     │
│ [일반]  │  │                      │     │
│ ■□□□□  │  └──────────────────────┘     │
│ [고급]  │                                │
│ ■■□□□  │  이름: 기억 위습 (Lv.15)       │
│ [희귀]  │  등급: ★★★ 희귀               │
│ ■□□□□  │  진화: 1/2                     │
│ ...     │  HP: 280 / 공격력: 22          │
│         │                                │
│         │  [패시브] 기억 보존 (+10% 파편) │
│         │  [액티브] 회복의 숨결 (Lv.3)   │
│         │                                │
│         │  [소환] [진화] [스킬변경]       │
├─────────┴────────────────────────────────┤
│  [코스튬 적용]  [이름 변경]  [놓아주기]  │
└──────────────────────────────────────────┘
```

### 7.2 HUD 펫 표시

- 주인 캐릭터 옆에 소형 펫 아이콘 + HP 바
- 액티브 스킬 쿨다운 표시 (퀵슬롯 하단 소형 아이콘)
- 자동 수집 시 아이템 획득 팝업 (좌측 하단)
- 펫 사망 시 귀환 이펙트 + 재소환 쿨다운 타이머

### 7.3 펫 도감

수집한/미수집 펫 목록, 지역별/등급별 필터:

| 표시 항목 | 수집 전 | 수집 후 |
|---------|--------|--------|
| 실루엣 | ✅ (어두운 실루엣) | 풀 이미지 |
| 이름 | "???" | 정식 이름 |
| 서식지 | 지역명만 | 상세 획득 경로 |
| 도감 설명 | 비공개 | 세계관 플레이버 텍스트 |
| 수집률 | 전체 수집 진행률 표시 (%) | — |
