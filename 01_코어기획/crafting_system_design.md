## 1. 제작 시스템 개요

### 1.1 설계 철학

| 원칙 | 설명 |
|------|------|
| 세계관 연동 | 에테르 결정 가공, 기억 마법 도구 제작 등 에테르-기계 문명 반영 |
| 비전투 성장 경로 | 제작/채집만으로도 유의미한 경험치와 보상 획득 가능 (game_systems.md: 아이템 레벨 × 5 EXP) |
| 자급자족 가능 | 핵심 소비 아이템과 중급 장비는 자체 제작으로 충당 가능 |
| 경제 순환 기여 | 재료 거래와 제작 결과물이 경매장 경제의 핵심 축 |
| P2W 제로 | 제작 성공률/결과물에 영향을 주는 유료 아이템 없음 |

### 1.2 기본 사양

| 항목 | 수치 |
|------|------|
| 제작 숙련도 레벨 | 카테고리별 1~100 |
| 재료 창고 | 무제한 (game_systems.md 기준, 재료 자동 분류) |
| 제작대 위치 | 각 도시 NPC 제작대 + 휴대용 제작도구 (필드 제작, 일부 제한) |
| Phase 도입 | Phase 1: 기본 제작 / Phase 2: 고급 제작+채집 / Phase 3: 특수 제작 |

## 2. 제작 카테고리

### 2.1 카테고리 목록

| 카테고리 | 설명 | 주요 산출물 | 숙련도 해금 |
|---------|------|-----------|-----------|
| **대장간** (무기/방어구) | 금속 기반 장비 제작 및 수리 | 검, 방패, 갑옷, 장갑, 신발 | Lv.1~ |
| **마법공방** (마법장비) | 천/가죽 기반 마법 장비, 지팡이 | 로브, 지팡이, 마법봉, 책 | Lv.1~ |
| **연금술** (소비아이템) | 포션, 버프 아이템, 상태이상 치료제 | HP/MP 포션, 버프 물약, 해독제 | Lv.1~ |
| **에테르 공학** (에테르장치) | 에테르 결정 기반 기계 장치 | 순간이동 스크롤, 에테르 조명, 펫 부화기 촉진제 | Lv.20~ |
| **가구 제작** (하우징) | 개인 공간/길드 홀 가구 | 테이블, 의자, 조명, 장식 | Lv.10~ |
| **인챈트** (마법 부여) | 기존 장비에 마법 효과 부여 | 루니/소울/오라 인챈트 | Lv.30~ |

### 2.2 카테고리별 숙련도 보상

| 숙련도 레벨 | 혜택 |
|-----------|------|
| Lv.10 | 고급 등급 아이템 제작 해금 |
| Lv.25 | 희귀 등급 아이템 제작 해금, 제작 시간 -10% |
| Lv.40 | 영웅 등급 아이템 제작 해금, 보너스 옵션 확률 +10% |
| Lv.60 | 전설 등급 아이템 제작 해금, 대량 제작 (5개 일괄) 해금 |
| Lv.80 | 신화 등급 재료 가공 해금, 제작 시간 -25% |
| Lv.100 | 마스터 칭호, 마스터 전용 레시피 해금, 보너스 옵션 확률 +25% |

## 3. 재료 수급

### 3.1 채집 시스템

필드에 분포된 채집 포인트에서 재료 획득:

| 채집 유형 | 도구 | 지역 분포 | 주요 재료 |
|---------|------|---------|---------|
| 광석 채굴 | 곡괭이 | 전 지역 (사막 집중) | 철광석, 에테르 원석, 미스릴, 오리할콘 |
| 약초 채집 | 낫 | 전 지역 (숲 집중) | 치유초, 마나풀, 기억화, 망각의 이끼 |
| 벌목 | 도끼 | 실반헤임, 브리탈리아 | 고대목, 에테르목, 기억목 |
| 가죽 가공 | (몬스터 드롭) | 전 지역 | 일반 가죽, 드레이크 가죽, 에테르 수피 |
| 에테르 채취 | 에테르 추출기 | 에테르 결정 매장지 | 에테르 파편, 정제 에테르, 순수 에테르 |

**채집 경험치:** 아이템 레벨 × 5 EXP (game_systems.md 기준)
**첫 채집 보너스:** +100% EXP

### 3.2 몬스터 드롭 재료

| 몬스터 유형 | 드롭 재료 | 용도 |
|-----------|---------|------|
| 일반 (필드) | 기본 재료 (뼈, 가죽, 클로) | 일반~고급 제작 |
| 엘리트 | 정련 재료 (강화된 뼈, 마법 가죽) | 희귀~영웅 제작 |
| 던전 보스 | 핵심 재료 (보스 고유 소재) | 영웅~전설 제작 |
| 레이드 보스 | 신화 재료 (보스 에센스) | 전설~신화 제작 |

### 3.3 분해 시스템

불필요한 장비를 분해하여 재료 회수:

| 장비 등급 | 분해 결과 | 회수율 |
|---------|---------|--------|
| 일반 | 기본 재료 1~2개 | 30% |
| 고급 | 기본 재료 2~3개 + 정련 재료 0~1개 | 40% |
| 희귀 | 정련 재료 2~3개 + 희귀 재료 0~1개 | 50% |
| 영웅 | 희귀 재료 2~3개 + 핵심 재료 0~1개 | 50% |
| 전설 | 핵심 재료 2~3개 + 에테르 정수 1개 | 60% |
| 신화 | 분해 불가 (영구 귀속) | — |

### 3.4 거래소 연동

- 모든 재료는 경매장에서 거래 가능
- 제작 결과물 중 귀속 아이템 제외하고 거래 가능
- 재료 시세 참조 기능 (최근 7일 평균가 표시)

## 4. 레시피 시스템

### 4.1 레시피 획득 방법

| 획득 유형 | 설명 | 예시 |
|---------|------|------|
| **자동 해금** | 숙련도 레벨 달성 시 자동 습득 | 기본 HP 포션 (연금술 Lv.1) |
| **NPC 구매** | 제작 관련 NPC에게서 골드로 구매 | 에테르 단검 레시피 (무기 상인, 5,000G) |
| **퀘스트 보상** | 메인/서브 퀘스트 완료 보상 (story_design.md: 레시피 및 설계도 보상 유형) | "기억 결정 갑옷" 레시피 (Ch.2 서브 퀘스트) |
| **발견형** | 특정 재료 조합을 제작대에서 직접 시도 → 성공 시 레시피 영구 등록 | 재료 3종을 실험적으로 투입 → "에테르 조명" 발견 |
| **드롭** | 던전/레이드 보스 처치 시 레시피 드롭 | "오리할콘 대검" 레시피 (Ch.4 던전) |
| **명성 상점** | 세력 친밀도 상점에서 명성 포인트로 교환 | 엘파리스 명성 Lv.4 → "기억목 지팡이" 레시피 |

### 4.2 레시피 등급

| 등급 | 색상 | 제작 결과물 등급 | 숙련도 요구 |
|------|------|-------------|-----------|
| 입문 | 회색 | 일반 | 1~10 |
| 숙련 | 초록 | 고급 | 10~25 |
| 전문 | 파랑 | 희귀 | 25~40 |
| 장인 | 보라 | 영웅 | 40~60 |
| 대장인 | 주황 | 전설 | 60~80 |
| 마스터 | 빨강 | 신화 | 80~100 |

### 4.3 레시피 예시

#### 대장간 — 에테르 블레이드 (희귀 한손검)

```
레시피: 에테르 블레이드
등급: 전문 (희귀)
요구 숙련도: 대장간 Lv.30
획득: Ch.1 메인 퀘스트 "잊혀진 도시의 목격자" 보상

재료:
  - 정련 철광석 ×5
  - 에테르 원석 ×3
  - 기억 결정 ×1
  - 골드 2,000

제작 시간: 30초
결과물:
  에테르 블레이드 (Lv.30 한손검)
  물리 공격력 +85
  마법 공격력 +30
  특수: 에테르 관통 (방어력 무시 5%)
  보너스 옵션: 1~2개 랜덤 (성공률 기반)
```

#### 연금술 — 상급 HP 포션

```
레시피: 상급 HP 포션
등급: 숙련 (고급)
요구 숙련도: 연금술 Lv.15
획득: 자동 해금

재료:
  - 치유초 ×3
  - 맑은 물 ×1
  - 에테르 파편 ×1

제작 시간: 5초 (대량 제작: 5개 일괄 25초)
결과물:
  상급 HP 포션
  효과: HP 1,500 즉시 회복
  스택: 최대 50개
```

#### 에테르 공학 — 순간이동 스크롤

```
레시피: 순간이동 스크롤
등급: 전문 (희귀)
요구 숙련도: 에테르 공학 Lv.25
획득: NPC 구매 (에테르 공학자, 3,000G)

재료:
  - 정제 에테르 ×2
  - 기억화 ×1 (위치 기억 매개체)
  - 에테르 양피지 ×1

제작 시간: 15초
결과물:
  순간이동 스크롤 ×1
  효과: 저장된 위치로 즉시 이동
  스택: 최대 20개
```

## 5. 제작 프로세스

### 5.1 기본 제작 흐름

```
1. 제작대 접근 (도시 NPC 또는 휴대용 제작도구)
2. 카테고리 선택 (대장간 / 마법공방 / 연금술 / ...)
3. 레시피 선택 (보유 레시피 목록에서)
4. 재료 확인 (자동으로 재료 창고에서 검색, 부족분 강조 표시)
5. 제작 수량 선택 (1개 / 대량 제작)
6. [제작 시작] → 제작 진행 바 (제작 시간 경과)
7. 제작 완료 → 결과 표시 (기본 스탯 + 보너스 옵션)
8. 인벤토리에 추가 + 제작 숙련도 EXP 획득
```

### 5.2 성공률 시스템

| 등급 차이 | 성공률 | 대성공 확률 |
|---------|--------|-----------|
| 숙련도 ≥ 요구 + 20 | 100% | 20% |
| 숙련도 ≥ 요구 + 10 | 100% | 10% |
| 숙련도 ≥ 요구 | 95% | 5% |
| 숙련도 = 요구 - 5 | 80% | 2% |
| 숙련도 = 요구 - 10 | 60% | 0% |
| 숙련도 < 요구 - 10 | 제작 불가 | — |

**대성공 효과:**
- 보너스 옵션 +1개 추가
- 기본 스탯 +10% 증가
- 제작 EXP +50% 추가

**실패 시:**
- 재료 50% 반환 (나머지 소멸)
- 실패 EXP: 성공 시의 30% 지급 (숙련도 성장은 계속)

### 5.3 보너스 옵션 시스템

제작 결과물의 보너스 옵션은 제작 시 랜덤 결정:

| 결과물 등급 | 보너스 옵션 수 | 옵션 풀 |
|-----------|-------------|--------|
| 일반 | 0개 | — |
| 고급 | 0~1개 | 기본 옵션 풀 |
| 희귀 | 1~2개 | 기본 + 중급 옵션 풀 |
| 영웅 | 2~3개 | 중급 + 고급 옵션 풀 |
| 전설 | 3~4개 + 고유 효과 | 고급 옵션 풀 + 제작 전용 고유 효과 |
| 신화 | 4~5개 + 세트 효과 | 전체 옵션 풀 + 마스터 전용 효과 |

> **제작 전용 고유 효과:** 드롭 장비에는 없는 제작 전용 특수 옵션 (예: "장인의 손길: 강화 성공률 +3%")

## 6. 특수 제작 — 에테르 결정 가공

### 6.1 에테르 결정 가공 개요

에테르나 세계의 핵심 자원인 에테르 결정을 가공하는 고급 제작 시스템. 세계관의 에테르-기계 문명을 직접 체험하는 콘텐츠.

| 항목 | 내용 |
|------|------|
| 해금 조건 | 에테르 공학 숙련도 Lv.40 + Ch.3 메인 퀘스트 완료 |
| 제작 장소 | 에테르 가공소 (아르겐티움, 솔라리스 전용 NPC) |
| 핵심 재료 | 에테르 결정 (파편/일반/정제/순수/신성 — 5등급) |

### 6.2 에테르 결정 등급

| 등급 | 획득처 | 용도 |
|------|-------|------|
| 에테르 파편 | 필드 채취, 몬스터 드롭 | 기본 연금술/제작 재료 |
| 일반 에테르 결정 | 채취, 분해, 상점 | 에테르 장치 기본 재료 |
| 정제 에테르 결정 | 파편 5개 가공 | 중급 장치, 순간이동 스크롤 |
| 순수 에테르 결정 | 정제 3개 가공 + 특수 촉매 | 고급 에테르 장비, 인챈트 재료 |
| 신성 에테르 결정 | 순수 3개 + 신성 기억 파편 (퀘스트 아이템) | 신화급 제작, 엔딩 D 관련 아이템 |

### 6.3 에테르 가공 레시피

| 레시피 | 재료 | 결과물 | 숙련도 |
|--------|------|--------|--------|
| 에테르 정제 | 에테르 파편 ×5 | 정제 에테르 결정 ×1 | Lv.40 |
| 에테르 순화 | 정제 에테르 ×3 + 기억 결정 ×1 | 순수 에테르 결정 ×1 | Lv.60 |
| 에테르 신성화 | 순수 에테르 ×3 + 신성 기억 파편 ×1 | 신성 에테르 결정 ×1 | Lv.80 |
| 에테르 코어 | 순수 에테르 ×5 + 오리할콘 ×3 | 에테르 동력 코어 (장비 소켓 삽입) | Lv.70 |
| 기억 증폭기 | 정제 에테르 ×2 + 기억화 ×3 + 기억 결정 ×1 | 기억 증폭기 (기억 공명 스킬 강화 소비아이템) | Lv.50 |

### 6.4 에테르 소켓 시스템

영웅+ 등급 장비에는 '에테르 소켓'이 존재. 가공한 에테르 코어를 삽입하여 추가 효과:

| 코어 유형 | 소켓 효과 | 제작 재료 |
|---------|---------|---------|
| 화염 에테르 코어 | 화염 속성 데미지 +8% | 순수 에테르 + 이프리타 화염석 |
| 빙결 에테르 코어 | 빙결 속성 데미지 +8% | 순수 에테르 + 에테르 빙정 |
| 번개 에테르 코어 | 번개 속성 데미지 +8% | 순수 에테르 + 번개 결정 |
| 생명 에테르 코어 | HP +500, HP 재생 +3/초 | 순수 에테르 + 고대목 수액 |
| 기억 에테르 코어 | 기억 파편 획득량 +15%, 기억 스킬 데미지 +5% | 순수 에테르 + 기억 결정 ×3 |

> 소켓 삽입은 비파괴적 (기존 코어 제거 후 새 코어 삽입 가능, 제거 시 코어 파괴).

### 7.2 UI 기능 상세

| 기능 | 설명 |
|------|------|
| 레시피 검색 | 이름/재료/결과물 등급으로 검색 |
