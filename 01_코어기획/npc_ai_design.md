# 에테르나 크로니클 — NPC AI 행동 패턴

> **작성일:** 2026-03-10 | **버전:** v1.0 | **분류:** 코어기획 — AI 시스템

---

## 1. 개요

본 문서는 에테르나 크로니클의 NPC AI 설계를 정의한다. 적 몬스터(일반/엘리트/보스), NPC 동료, 감정·대화 AI의 행동 모델과 구현 구조를 포함한다.

### 1.1 설계 원칙

| 원칙 | 설명 |
|------|------|
| **예측 가능한 학습 곡선** | 플레이어가 패턴을 읽고 대응할 수 있어야 한다 |
| **Active Pause 연계** | 스페이스바 일시정지 시 AI 상태가 투명하게 노출되어야 한다 |
| **성능 우선** | 웹 클라이언트(PC 브라우저) 환경에서도 60fps 유지 |
| **기억-망각 테마 반영** | 보스/엘리트의 고유 메카닉에 세계관 이원성을 녹여낸다 |

## 2. 일반 몬스터 AI — 유한 상태 기계(FSM)

### 2.1 상태 정의

```
[Idle] ──(감지)──▶ [Chase] ──(사거리 진입)──▶ [Attack]
  ▲                   │                          │
  │                   │                     (HP < 15%)
  │              (감지 해제)                     │
  │                   │                          ▼
  └───(리셋)─────◀───┘                       [Flee]
  │
  └──(순찰 경로 존재)──▶ [Patrol] ──(감지)──▶ [Chase]
```

| 상태 | 진입 조건 | 행동 | 전이 조건 |
|------|-----------|------|-----------|
| **Idle** | 스폰 직후, 리셋 후 | 제자리 대기, 랜덤 모션 재생(3~8초 주기) | 감지 범위 내 적 발견 → Chase |
| **Patrol** | 순찰 웨이포인트가 설정된 몬스터 | 웨이포인트 간 이동 (속도: 이동속도 × 0.6) | 감지 범위 내 적 발견 → Chase |
| **Chase** | 적 감지 | 대상 추적, 최대 추적 거리(leashRange) 유지 | 사거리 진입 → Attack / 추적 거리 초과 → Idle |
| **Attack** | 사거리 진입 | 기본 공격 패턴 반복 (공격속도 기반 쿨타임) | 대상 사망 → Idle / HP < 15% → Flee |
| **Flee** | HP 15% 미만 | 대상 반대 방향 도주 (속도: 이동속도 × 1.3) | 5초 경과 또는 거리 20m 이상 → Idle |

### 2.2 어그로 테이블

| 요소 | 어그로 가중치 | 비고 |
|------|--------------|------|
| 물리 데미지 1pt | +1.0 어그로 | 기본 기준 |
| 마법 데미지 1pt | +1.2 어그로 | 기억술사 견제 |
| 힐량 1pt | +0.5 어그로 | 간접 기여 |
| 버프 시전 1회 | +20 어그로 (고정) | - |
| 도발 스킬 | +500 어그로 (고정) + 3초 강제 타겟 | 에테르 기사 전용 |
| 시간 감쇠 | -5% / 초 | 전투 이탈 시 빠른 감소 |

**어그로 전환 임계값:** 현재 타겟 대비 어그로가 **130%** 이상인 새 대상이 발생하면 타겟 전환 (hysteresis 방지).

### 2.3 감지 시스템

- **기본 감지 반경:** 8m (원형)
- **시야각:** 120° 전방 (측면/후방 접근 시 반경 4m로 축소)
- **고저차 보정:** 수직 3m 이내만 감지
- **은신 감지:** 그림자 직조사 은신 시 감지 반경 60% 감소

## 3. 엘리트 몬스터 AI — 행동 트리(Behavior Tree)

### 3.1 구조 개요

엘리트 몬스터는 FSM 대신 행동 트리(BT)를 사용해 조건부 의사결정을 수행한다.

```
Root [Selector]
├── [Sequence] 위기 대응
│   ├── [Condition] HP < 30%?
│   ├── [Action] 방어 버프 시전 (쿨타임 20초)
│   └── [Action] 후퇴 → 원거리 공격 패턴 전환
│
├── [Sequence] 디버프 대응
│   ├── [Condition] 디버프 2개 이상 활성?
│   └── [Action] 정화 스킬 시전 (쿨타임 15초)
│
├── [Sequence] 어그로 관리
│   ├── [Condition] 최고 어그로 대상이 근접 탱커?
│   ├── [Selector]
│   │   ├── [Sequence] 탱커 무시 시도
│   │   │   ├── [Condition] 후방에 원거리 딜러 존재?
│   │   │   └── [Action] 돌진 → 후방 딜러 공격 (30% 확률)
│   │   └── [Action] 탱커 대상 강공격
│
├── [Sequence] 일반 전투
│   ├── [Action] 스킬 로테이션 (1→2→3→기본공격×3 반복)
│   └── [Action] 사거리 유지 이동
│
└── [Action] Idle 복귀
```

### 3.2 엘리트 등급별 AI 차이

| 등급 | BT 깊이 | 특수 행동 | 스킬 수 |
|------|---------|-----------|---------|
| **엘리트 I** (고급) | 3레벨 | 기본 회피, 단일 버프 | 2개 |
| **엘리트 II** (희귀) | 4레벨 | 어그로 전환, 디버프 정화 | 3개 |
| **엘리트 III** (영웅) | 5레벨 | 아군 소환, 지형 활용, 페이즈 전환(2페이즈) | 4~5개 |

### 3.3 조건부 스킬 사용 규칙

```
스킬 선택 우선순위:
1. 긴급 대응 (HP < 30% → 방어/회복)
2. 디버프 대응 (활성 디버프 ≥ 2 → 정화)
3. 전략 스킬 (어그로 조작, 소환, 범위기)
4. 기본 로테이션 (데미지 효율 순서)
```

## 4. 보스 AI — 페이즈 전환 시스템

### 4.1 공통 구조

모든 보스는 **3페이즈** 구조를 따른다:

| 페이즈 | HP 구간 | 특성 |
|--------|---------|------|
| **Phase 1** | 100% ~ 60% | 기본 패턴 학습 구간. 텔레그래프 명확 |
| **Phase 2** | 60% ~ 30% | 신규 스킬 추가. 패턴 변형. 전투 템포 상승 |
| **Phase 3** | 30% ~ 0% | 고유 메카닉 전개. 광폭화(Enrage) 타이머 시작 |

**전이 연출:** 페이즈 전환 시 1.5초 무적 + 전환 연출 재생 + Active Pause 자동 트리거 (플레이어 대응 시간 제공).

**광폭화(Enrage):** Phase 3 진입 후 **180초** 내 처치 실패 시 공격력 +200%, 방어력 +100%, 스킬 쿨타임 50% 감소.

### 4.2 Ch1 보스: 망각의 수호자 (에레보스)

> *에레보스 망각 폐허의 최심부에서 기억의 잔해를 흡수하며 존재를 유지하는 거대 골렘.*

**기본 정보:**
- 권장 레벨: 15~18 | 권장 인원: 4인
- 체력: 850,000 | 공격력: 320 | 방어력: 180
- 속성: 무(Void) | 약점: 에테르(기억) 속성 +25%

#### Phase 1 (100%~60%) — 기억의 포식자

| 스킬 | 쿨타임 | 설명 |
|------|--------|------|
| **망각의 일격** | 6초 | 전방 부채꼴(90°, 6m). 피격 시 3초간 "기억 혼란" (스킬 아이콘 무작위 셔플) |
| **기억 흡수** | 12초 | 랜덤 1인 대상 채널링(3초). 완료 시 대상 최대HP 10% 흡수, 보스 HP 5% 회복 |
| **돌진** | 15초 | 최원거리 대상에게 돌진. 경로 상 피격 시 넉백 5m |

**핵심 메카닉 — 기억 흡수 차단:**
- "기억 흡수" 채널링 중 보스에게 **총 15,000 데미지**를 가하면 차단 가능
- 차단 성공 시 보스 3초 스턴 + "기억 과부하" 디버프 (받는 피해 +20%, 10초)
- Active Pause 활용: 채널링 시작 시 시각적 텔레그래프(보라색 쇄도선) 표시 → 일시정지 후 집중 공격 지시

#### Phase 2 (60%~30%) — 망각의 확산

| 추가 스킬 | 쿨타임 | 설명 |
|-----------|--------|------|
| **망각의 안개** | 20초 | 필드 중앙에 반경 8m 안개 생성(15초 지속). 안개 내 아군: 시야 50% 감소, 미니맵 비활성 |
| **기억 분열체** | 30초 | 소형 분열체 3기 소환 (HP 각 50,000). 10초 내 처치 실패 시 보스 HP 5% 회복/기 |

- Phase 1 스킬의 쿨타임 20% 감소
- "기억 흡수" 대상이 2인으로 증가 (동시 채널링)

#### Phase 3 (30%~0%) — 망각의 구현

| 추가 스킬 | 쿨타임 | 설명 |
|-----------|--------|------|
| **완전한 망각** | 45초 | 필드 전체 공격. 3초 시전 후 전원에게 최대HP 30% 데미지. 시전 중 안전지대 1곳 생성(빛기둥) |
| **기억 역류** | - (HP 10% 도달 시 1회) | 보스 자폭 패턴. 8초 카운트다운 후 필드 전체 즉사급 데미지. 필드 가장자리 보호막 3개 중 1개에 숨어야 생존 |

- 광폭화 타이머: 180초
- "기억 흡수" 채널링 시간 3초 → 2초로 단축 (차단 난이도 상승)

### 4.3 Ch3 보스: 사막의 불꽃왕 (솔라리스)

> *솔라리스 사막 깊은 곳, 용암 위에 군림하는 고대 화염 정령. 지형 자체를 무기로 사용한다.*

**기본 정보:**
- 권장 레벨: 40~45 | 권장 인원: 4인
- 체력: 2,400,000 | 공격력: 680 | 방어력: 350
- 속성: 화염 | 약점: 빙결 속성 +30%

#### Phase 1 (100%~60%) — 사막의 군주

| 스킬 | 쿨타임 | 설명 |
|------|--------|------|
| **화염 작렬** | 5초 | 전방 직선(12m). 1초 텔레그래프(지면 적열) 후 화염 방사 |
| **용암 분출** | 10초 | 랜덤 3지점에 용암 웅덩이 생성(반경 2m, 15초 지속). 접촉 시 초당 HP 3% 화상 |
| **열파** | 18초 | 자신 중심 원형(8m). 피격 시 넉백 + "과열" 디버프 (받는 화염 피해 +15%, 8초) |

**핵심 메카닉 — 지형 변환:**
- 전투장은 3×3 격자(총 9구역)로 나뉨
- 용암 분출 누적으로 구역이 "용암 지대"로 변환 (구역 내 지속 데미지)
- 플레이어는 각 구역 모서리의 **에테르 결정 장치**(HP 20,000)를 파괴하여 용암 구역을 "냉각 구역"으로 전환 가능
- 냉각 구역에서 보스 피격 시 이동속도 -30%, 화염 스킬 데미지 -20%

#### Phase 2 (60%~30%) — 지형의 지배자

| 추가 스킬 | 쿨타임 | 설명 |
|-----------|--------|------|
| **지각 변동** | 25초 | 전투장 격자 무작위 재배치 (용암/냉각/일반 구역 셔플). 2초 사전 경고 |
| **화염 기둥** | 15초 | 각 용암 구역에서 화염 기둥 분출 (구역 전체 데미지). 냉각 구역은 안전 |

- 용암 분출 생성 빈도 2배
- 에테르 결정 장치 HP 30,000으로 증가

#### Phase 3 (30%~0%) — 솔라리스의 심판

| 추가 스킬 | 쿨타임 | 설명 |
|-----------|--------|------|
| **태양의 심판** | 40초 | 4초 시전. 모든 용암 구역 동시 폭발 (즉사급). 냉각 구역으로 피난 필수 |
| **용암 갑옷** | - (Phase 3 진입 시 영구) | 보스 주변 2m 지속 화염 오라. 근접 공격 시 반사 데미지 (공격력의 15%) |

- 광폭화 타이머: 180초
- 냉각 구역이 30초마다 자동으로 용암 구역으로 재변환 → 지속적 결정 파괴 필요

### 4.4 Ch5 보스: 레테 / 레나르도 최종전

> *기억의 신 레테가 마지막 시험을 건다. 기억과 망각의 경계에서 세계의 운명이 결정된다.*

**기본 정보:**
- 권장 레벨: 90~100 | 권장 인원: 4~8인 (난이도 선택)
- 체력: Phase별 상이 | 총 유효 체력: ~12,000,000
- 속성: 기억(Phase 1,3) / 망각(Phase 2) | 약점: 페이즈별 전환

#### Phase 1 (레나르도의 야망) — HP 4,000,000

레나르도(인간 형태)가 신성 기억 파편의 힘으로 공격.

| 스킬 | 쿨타임 | 설명 |
|------|--------|------|
| **기억의 칼날** | 4초 | 3연속 검격. 각 타격이 다른 방향 (전방→좌측→우측) |
| **파편 폭발** | 12초 | 수집한 기억 파편 1개를 투척. 착탄 지점 반경 5m 폭발 + "기억 침식" (스킬 1개 봉인, 6초) |
| **시간 왜곡** | 20초 | 대상 1인의 쿨타임 전체 +5초 증가 (8초 지속) |
| **기억 소환** | 25초 | 이전 챕터 보스의 축소판 소환 (HP 200,000, 스킬 1개만 사용). 동시 최대 1기 |

**핵심 메카닉 — 기억-망각 게이지:**
- 화면 상단에 "기억-망각 게이지" 표시 (중앙 시작, 좌=기억 / 우=망각)
- 레나르도의 기억 속성 공격을 받으면 게이지가 "기억" 쪽으로 이동
- 게이지가 한쪽 끝에 도달하면 전원 "기억 과부하" 또는 "망각 과부하" (받는 피해 +50%, 10초)
- **균형 유지가 핵심**: 필드의 "기억 제단"(기억 게이지 감소)과 "망각 제단"(망각 게이지 감소)을 교대로 활성화

#### Phase 2 (레테의 시험) — HP 4,000,000

레나르도 쓰러진 후, 레테가 직접 강림. 속성이 망각으로 전환.

| 스킬 | 쿨타임 | 설명 |
|------|--------|------|
| **망각의 물결** | 8초 | 전방 부채꼴(180°, 10m). 피격 시 "기억 상실" (UI 일부 비활성: HP바/미니맵/스킬 쿨타임 표시 숨김, 5초) |
| **존재 소거** | 15초 | 대상 1인을 3초간 "투명화" (아군에게 보이지 않음, 힐/버프 불가). 본인은 조작 가능 |
| **시간 되감기** | 30초 | 모든 플레이어를 5초 전 위치로 강제 이동 (데미지 없음, 포지셔닝 교란) |
| **망각의 심판** | 40초 | 전원에게 질문 프롬프트 표시: 이전 페이즈에서 레나르도가 사용한 스킬 중 하나 선택. 정답 시 데미지 면제, 오답 시 최대HP 25% 데미지 |

**핵심 메카닉 — 기억 유지:**
- Phase 2에서는 화면 효과가 점진적으로 흐려지며 UI 정보가 감소
- "망각의 심판"은 플레이어의 실제 기억력을 테스트 (세계관 테마 직결)
- 기억-망각 게이지가 "망각" 쪽으로 자동 드리프트 → 기억 제단 활성화 빈도 증가 필요

#### Phase 3 (경계의 붕괴) — HP 4,000,000

레테와 레나르도가 융합. 기억과 망각 속성이 **매 30초마다 전환**.

| 스킬 | 쿨타임 | 설명 |
|------|--------|------|
| **이원의 일격** | 5초 | 현재 속성에 따라 검격(기억) 또는 파동(망각) 교대 사용 |
| **경계 붕괴** | 20초 | 필드에 기억 영역(파랑)과 망각 영역(보라)을 반반 생성. 현재 보스 속성과 같은 영역에 서면 받는 피해 +100% |
| **최종 심판** | 60초 | 전원 선택지: "기억을 수호한다" / "망각을 받아들인다". 다수결에 따라 보스의 남은 체력 동안 속성 고정. 엔딩 분기에 영향 |
| **존재의 소멸** | - (HP 5% 시 1회) | 전멸기. 10초 카운트다운. 기억 제단 + 망각 제단 동시 활성화하여 차단 |

- 광폭화 타이머: 300초 (8인 레이드 기준)
- 엔딩 분기 영향: "최종 심판" 선택이 엔딩 A/C 가중치에 반영

## 5. NPC 동료 AI

### 5.1 포지셔닝 시스템

NPC 동료는 클래스별 기본 포지션을 유지한다:

| 동료 클래스 | 기본 포지션 | 거리 기준 |
|------------|------------|-----------|
| 에테르 기사 (레이나) | 전방 탱킹 라인 | 보스 기준 3m 이내 |
| 기억술사 (에리언/NPC) | 후방 원거리 | 보스 기준 10~12m |
| 그림자 직조사 (세라핀) | 측면/후방 플랭킹 | 보스 기준 5~7m (측면 90°) |

**동적 포지셔닝 규칙:**
1. 범위 공격 텔레그래프 감지 시 → 안전 영역으로 이동 (반응 시간: 0.8초)
2. 아군 HP < 30% → 보호 대상에게 접근 (힐러/서포터)
3. 보스 페이즈 전환 시 → 기본 포지션으로 리셋

### 5.2 스킬 사용 우선순위

```
[최우선] 생존 스킬 (본인 HP < 20%)
    ↓
[높음] 탱킹 스킬 (어그로 유지, 도발 쿨타임 체크)
    ↓
[높음] 힐/버프 스킬 (아군 HP < 50% 또는 버프 만료)
    ↓
[중간] CC 스킬 (보스 캐스팅 중 방해 가능 시)
    ↓
[기본] 데미지 스킬 (로테이션 순서)
    ↓
[최하] 기본 공격
```

### 5.3 위험 회피

| 위험 유형 | 감지 방법 | 회피 행동 |
|-----------|-----------|-----------|
| 범위 공격 텔레그래프 | 바닥 표시 감지 | 가장 가까운 안전 영역으로 이동 |
| 상태이상 영역 | 디버프 영역 진입 전 경로 체크 | 우회 경로 탐색 |
| 집중 타겟 (어그로 과다) | 어그로 수치 모니터링 | 공격 중단 + 후퇴 (어그로 감소 대기) |
| 즉사 메카닉 | 보스 스킬 ID 기반 하드코딩 | 최우선 회피 (다른 모든 행동 중단) |

### 5.4 Active Pause 시 직접 명령

스페이스바로 Active Pause 진입 시, NPC 동료에게 직접 명령 가능:

| 명령 | 효과 | UI 표시 |
|------|------|---------|
| **집중 공격** | 지정 대상에게 모든 공격 스킬 집중 | 빨간색 타겟 아이콘 |
| **방어 태세** | 공격 중단, 회피/방어만 수행 | 파란색 방패 아이콘 |
| **위치 이동** | 지정 좌표로 이동 후 해당 위치 유지 | 초록색 깃발 |
| **스킬 사용** | 특정 스킬 즉시 사용 지시 (쿨타임 확인) | 스킬 아이콘 하이라이트 |
| **자유 행동** | AI 자율 판단 복귀 (기본값) | 회색 자동 아이콘 |

## 6. 감정 · 대화 AI

### 6.1 친밀도 시스템

NPC 동료/주요 NPC는 **5단계 친밀도**를 가진다:

| 단계 | 친밀도 수치 | 호칭 변화 | 대화 톤 | 해금 콘텐츠 |
|------|-----------|-----------|---------|-------------|
| **1. 경계** | 0~99 | 공식 호칭("당신") | 형식적, 짧은 문장 | 기본 퀘스트 |
| **2. 호감** | 100~299 | 이름 호칭 | 친절, 정보 공유 | 서브 퀘스트 1세트 |
| **3. 신뢰** | 300~599 | 별명/약칭 | 농담 섞인 대화, 비밀 공유 | 개인 스토리 1장 |
| **4. 우정** | 600~899 | 친밀한 별명 | 감정 표현, 의지, 고민 상담 | 개인 스토리 2장 + 전용 장비 |
| **5. 맹세** | 900~1000 | 전용 호칭(캐릭별 고유) | 깊은 유대, 과거 회상, 맹세 | 최종 개인 스토리 + 합체기 해금 |

**친밀도 획득 경로:**

| 행동 | 획득량 | 일일 상한 |
|------|--------|-----------|
| 대화 (일반) | +5 | 3회/일 |
| 대화 (정답 선택지) | +15 | - |
| 선물 (좋아하는 아이템) | +20 | 2회/일 |
| 선물 (싫어하는 아이템) | -10 | - |
| 함께 전투 (던전 클리어) | +10 | 5회/일 |
| 함께 전투 (보스 처치) | +25 | - |
| 개인 퀘스트 완료 | +50 | - |

### 6.2 대화 분기 구조

```
[대화 시작]
    │
    ├── [친밀도 체크] → 단계별 대화 풀(Pool) 선택
    │
    ├── [컨텍스트 체크]
    │   ├── 현재 메인 퀘스트 진행도
    │   ├── 최근 전투 결과 (승/패)
    │   ├── 시간대 (아침/낮/저녁/밤)
    │   ├── 현재 위치 (지역별 반응)
    │   └── 파티 구성원 (특정 NPC 조합 시 특수 대화)
    │
    ├── [선택지 생성] → 2~4개 선택지 제시
    │   ├── 호감 선택지 (친밀도 +)
    │   ├── 중립 선택지 (정보 획득)
    │   ├── 반감 선택지 (친밀도 -)
    │   └── [조건부] 특수 선택지 (퀘스트/아이템 조건 충족 시)
    │
    └── [결과 처리]
        ├── 친밀도 변동 적용
        ├── 후속 대화/퀘스트 트리거
        └── 감정 상태 갱신
```

### 6.3 감정 상태 시스템

NPC는 내부적으로 감정 상태를 유지한다:

| 감정 | 트리거 | 대화 영향 | 시각적 표현 |
|------|--------|-----------|------------|
| **기쁨** | 선물, 전투 승리, 정답 선택지 | 밝은 톤, 추가 정보 제공 | 미소 표정, 밝은 이펙트 |
| **슬픔** | 아군 쓰러짐, 기억 관련 이벤트 | 조용한 톤, 과거 회상 | 고개 숙임, 어두운 톤 |
| **분노** | 반감 선택지 연속, 적 등장 | 거친 톤, 대화 거부 가능 | 붉은 이펙트, 강한 표정 |
| **불안** | 보스 전 대기, 위험 지역 진입 | 조심스러운 톤, 경고 대사 | 주변 두리번거림 |
| **평온** | 안전 지역, 휴식 | 느긋한 톤, 일상 대화 | 편안한 모션 |

감정은 시간 경과에 따라 "평온"으로 자연 회귀 (반감기: 300초).
