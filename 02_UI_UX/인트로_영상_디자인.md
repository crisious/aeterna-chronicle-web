## 개요

### 인트로 영상 목적

게임 시작 시 타이틀 화면 이전 또는 직후에 재생되는 **무성 / 내레이션 영상**으로, 다음 세 가지를 30~60초 안에 전달한다:

1. **세계관**: 에테르나는 기억이 소멸하는 세계다
2. **갈등**: 200년 전 대망각이 시작되었고 세계는 무너지고 있다
3. **주인공**: 단 한 사람, 기억 공명 능력을 가진 에리언이 존재한다

### 영상 기술 스펙

| 항목 | 내용 |
|------|------|
| 생성 툴 | WAN 2.2 (Text-to-Video) |
| 해상도 | 720p (1280×720) |
| 프레임 수 | 81 frames per clip |
| Steps | 30 |
| CFG Scale | 7.0 |
| 총 씬 수 | 메인 7씬 + NPC 카메오 6컷 |
| 총 영상 길이 | 약 58~66초 (편집 템포에 따라 가변) |
| 편집 방식 | 씬 간 크로스페이드 또는 블랙아웃 컷 |
| 음악 | 별도 BGM 오버레이 (에레보스 탐색 BGM 또는 전용 인트로 음악) |

## WAN 2.2 생성 실행 순서 (큐 + 재시도 규칙)

> 목표: 메인 서사 컷을 먼저 확보하고, 카메오는 리스크 분산으로 병렬 처리.
> 권장 전략: **핵심 씬 우선 → 카메오 병렬 → 마무리 씬 확정**

### 1) 생성 큐 우선순위

| 우선순위 | 컷 | 이유 |
|---|---|---|
| P0 | **씬06 에리언(FINAL 12)**, **씬07 타이틀(FINAL 13)** | 감정/브랜딩 핵심 컷. 퀄리티 기준점으로 사용 |
| P1 | FINAL 01~05 (세계관 메인 씬) | 세계관 전달의 뼈대 |
| P2 | FINAL 06~11 (카메오 6컷) | 서사 보강. 메인 대비 리테이크 부담 낮음 |

> 번호 체계 참고: 카메오를 중간에 넣어서 에리언/타이틀은 FINAL 12/13으로 배치됨.

### 2) 실제 실행 순서 (권장)

1. **씬06 에리언** 생성 (기준 룩 확정)
2. **씬07 타이틀** 생성 (엔드 톤 확정)
3. **씬01~05** 순차 생성 (세계관 연결)
4. **카메오 6컷** 병렬/순차 생성
   - 누아리엘 → 다리오 → 로웬 → 시엘라 → 오르덴 → 카르마드
5. 편집 타임라인에 임시 삽입 후 리듬 확인
6. 이탈 컷만 재생성

### 3) 컷당 생성 횟수 규칙

- 기본: **컷당 2테이크**
- 중요 컷(씬06, 씬07): **최대 4테이크**
- 카메오 컷: **최대 3테이크**
- 4회 이상 반복 금지 (시간 대비 효율 하락)

### 4) 실패 유형별 재시도 규칙

| 실패 유형 | 증상 | 수정 방법 |
|---|---|---|
| 플리커/떨림 | 프레임 밝기 점프, 손떨림 | 프롬프트에 `no flicker`, `minimal camera shake` 재강조, 카메라 동작 단순화 |
| 얼굴/손 붕괴 | 손가락 이상, 눈 비대칭 | 액션 수 축소(한 컷 한 액션), 클로즈업 시간 단축 |
| 배경 과밀 | 노이즈, 오브젝트 난잡 | 배경 오브젝트 키워드 20~30% 축소 |
| 캐릭터 비일관 | 의상/헤어 변형 | `consistent costume details` + 캐릭터 고유 소품 재명시 |
| 톤 이탈 | 너무 밝거나 장르 이탈 | `dark fantasy`, `muted palette` 재추가, Negative 강화 |

### 5) 퀄리티 합격 기준 (PASS/FAIL)

- **PASS 조건 (모두 충족):**
  - 인물 식별 가능(얼굴/의상/소품)
  - 카메라 의도 유지(과한 흔들림 없음)
  - 조명 방향 일관
  - 2초 이상 시청 시 붕괴 프레임 체감 없음
- **FAIL 조건 (하나라도 해당):**
  - 손/얼굴 붕괴가 컷 핵심 구간에 노출
  - 액션이 프롬프트 의도와 다름
  - 장르 톤(다크 판타지) 이탈
