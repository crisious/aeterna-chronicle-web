## 일러스트 AI 프롬프트 공통 스타일 가이드

> **v1.2 업데이트**: 전체 AI 영상 프롬프트 WAN 2.2 기반으로 전면 재작성
> 각 캐릭터 프롬프트는 Checkpoint AI / ComfyUI / A1111 환경에 최적화되어 있습니다.
> 공통 Prefix와 Negative Prompt를 각 캐릭터 프롬프트에 추가하면 일관된 화풍을 유지할 수 있습니다.

### 권장 모델
- **Checkpoint**: `Deliberate v6` / `RealisticVision V6` / `DreamShaper XL` (다크 판타지 일러스트 적합)
- **LoRA**: `DetailTweaker` (디테일 강화) + `add-detail-xl` 조합 권장
- **VAE**: `vae-ft-mse-840000-ema-pruned`
- **Sampler**: DPM++ 2M Karras, Steps 28~35, CFG 7~8

### 공통 Prefix (모든 캐릭터 프롬프트 앞에 추가)
```
masterpiece, best quality, ultra-detailed, 8k uhd,
```

### 공통 Style Suffix (모든 캐릭터 프롬프트 뒤에 추가 가능)
```
dark fantasy RPG concept art, watercolor ink hybrid style,
cinematic lighting, volumetric atmosphere, muted color palette with luminous accents,
professional game art illustration, highly detailed character design
```

### 공통 Negative Prompt (전 캐릭터 공통)
```
nsfw, (worst quality:2), (low quality:2), (normal quality:2), lowres, blurry,
extra limbs, (deformed hands:1.3), bad anatomy, missing fingers, extra fingers,
childlike, cartoonish, anime style, oversaturated, flat lighting, plain background,
watermark, text, signature, logo, username, artist name, duplicate
```

### 추천 해상도
- 캐릭터 일러스트: `768×1152` (세로형 포트레이트)
- 전신샷: `832×1216`
- 환경/보스: `1216×832` (가로형)
