## 0. 디자인 방향성 (Why)

어머~! 이번 전투 시스템은 **"FF6의 긴장감을 웹에서 재현"** 이 목표예요! 🎯

| 핵심 감각 | FF6 원전 | 에테르나 적용 |
|-----------|---------|---------------|
| **박자감** | ATB 게이지가 채워지는 순간의 설렘 | 게이지 풀 시 청색 발광 펄스 + 미세 카메라 쉐이크 |
| **정보 밀도** | 한 화면에 파티 4명 + 적 + 행동창 | 좌측 파티 패널 / 우측 적 배열 / 하단 커맨드 |
| **서사적 연출** | 마법명 텍스트 카드 등장 | 스킬명 BitmapText 카드 + 등급 글로우 |
| **대미지 쾌감** | 숫자 팝업의 큰 스케일 차이 | 크리티컬 32px 금색, 일반 16px 흰색 |
| **리듬감** | ATB 차기 기다림 → 폭발 | 콤보 게이지와 동기화 (C→B→A→S) |

**디자인 원칙 3가지**
1. **가독성 우선** — ATB 게이지는 화면 어디서든 0.3초 안에 파악 가능해야 함
2. **FF6 향수 + 픽셀 정합** — 2px 그리드, 짝수 단위, 안티앨리어싱 금지
3. **레이어 위계** — 배경 < 엔티티 < VFX < HUD < 팝업 순서 고정

## 3. ATB 게이지 상세 스펙 (FF6 리스펙트) ⚔️

### 3.1 시각 상태

```
[EMPTY     ]  0%      ── #2A2A3A 바탕만
[▓▓▓▓▓     ]  50%     ── #5A7FA8 채움, 흐름 애니 없음
[▓▓▓▓▓▓▓▓▓▓] 100%    ── #89CFF0 + 펄스 글로우 (box-shadow 8px)
[▓░▓░▓░▓░▓░] FROZEN  ── #4A4A5A 스트라이프 패턴 (빙결/스턴)
[▓▓▓▓▓▓░░░░] HASTE   ── #FFF176 테두리 1px, 충전 속도 1.5x
```

### 3.2 애니메이션 (CSS + Phaser 트윈)

| 상태 변화 | 지속 | 이징 | 시각 |
|-----------|------|------|------|
| 0→100% 충전 | 게임 로직 속도 | linear | 폭 증가 |
| 100% 도달 순간 | 400ms | easeOutBack | 펄스 1회 + 밝기 +20% |
| 100% 유지 펄스 | 1200ms loop | sine | 글로우 투명도 0.5~1.0 왕복 |
| 행동 실행 → 0% | 200ms | easeInQuad | 폭 감소 + 페이드 |
| Stun 진입 | 300ms | easeOut | 색상 전환 + 스트라이프 페이드인 |

### 3.3 Phaser 의사코드

```ts
// ATB 게이지 렌더 (60fps)
atbBar.fillStyle(BATTLE_COLORS.atb.empty).fillRect(0, 0, 160, 8);
atbBar.fillStyle(getAtbColor(actor)).fillRect(0, 0, 160 * pct, 8);
if (pct >= 1.0) {
  scene.tweens.add({
    targets: atbBar, alpha: { from: 0.7, to: 1.0 },
    duration: 600, yoyo: true, repeat: -1, ease: 'Sine.easeInOut',
  });
}
```

## 10. VFX 카탈로그 (스킬별 연출 규격)

| 스킬 유형 | VFX 타입 | 지속 | 파티클 수 | 색상 |
|----------|---------|------|----------|------|
| 물리 공격 | 슬래시 (3프레임) | 200ms | — | `#FFFFFF` |
| 화염 스킬 | 불기둥 + 스파크 | 500ms | 12 | `#FF6B35` |
| 빙결 스킬 | 결정 파편 | 600ms | 8 | `#00E5FF` |
| 회복 스킬 | 상승 빛 파티클 | 800ms | 15 | `#2ECC71` |
| 에테르 스킬 | 마법진 + 발광 링 | 1000ms | 20 | `#89CFF0` |
| 궁극기 | 화면 암전 + 카드 + 폭발 | 2500ms | 30+ | 스킬별 |

**궁극기 프레임**
1. 0~300ms: 화면 암전 (검정 60%)
2. 300~800ms: 스킬명 카드 등장 (슬라이드인 + 금 글로우)
3. 800~1800ms: VFX 본체 재생
4. 1800~2200ms: 대미지 팝업 연쇄
5. 2200~2500ms: 암전 페이드아웃

## 12. 접근성 (WCAG AAA 준수) ♿

- **색상 외 정보**: ATB 게이지에 수치 텍스트 병행 표시 옵션 (설정에서 토글)
- **색약 모드**: 적=패턴(스트라이프), 아군=솔리드 구분 / HP 임계치에 형태 아이콘 추가
- **모션 감도**: `prefers-reduced-motion` 준수 — 펄스/쉐이크 제거, 페이드만 유지
- **대비**: 대미지 팝업 외곽선 2px 필수로 배경 대비 7:1 이상 보장
- **키보드 전용**: 커맨드 창 모든 액션 키보드 접근 가능, 포커스 링 2px `--accent-ether`
