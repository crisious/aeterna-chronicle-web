## 1) 모션 카탈로그 — 캐릭터

### 1.1 대기 (idle)

| 항목 | 규격 |
|------|------|
| 프레임 수 | 4 |
| FPS | 5 |
| 루프 | ✅ |
| 이징 | ease-in-out (호흡 리듬) |
| 설명 | 미세한 호흡 움직임. 무기 자연 흔들림 |

**프레임 분해:**

| 프레임 | 포즈 | 변화량 |
|--------|------|--------|
| 0 | 기본 자세. 무게중심 중앙 | 기준 |
| 1 | 미세 상체 상승 (1px). 숨 들이쉼 | +1px Y |
| 2 | 기본 자세로 복귀 | 0 |
| 3 | 미세 상체 하강 (1px). 숨 내쉼 | -1px Y |

**클래스별 차이:**

| 클래스 | idle 특이사항 |
|--------|-------------|
| 에테르 기사 | 방패 미세 들썩. 에테르 발광 펄스 (2/4프레임) |
| 기억술사 | 부유 토메 회전. 오브 발광 펄스 |
| 그림자 직조사 | 망토 자락 흔들림. 그림자 미세 변동 |
| 기억 파괴자 | 무기 무게에 의한 약간 기울임. 숨 크게 |
| 시간 수호자 | 부유 기어 회전. 시계 초침 움직임 |

| 2 | 전방 스윙 시작 | swing start |

|--------|----------|----------|
| 에테르 기사 | 검 수직 참격 → 수평 베기 (교대) | 에테르 청색 잔상 궤적 |
| 기억 파괴자 | 해머 오버헤드 스매시 | 충격파 이펙트. 지면 균열 |

## 4) 타이밍 & 이징 총괄

### 4.1 이징 함수 정의

| 이징 이름 | 수식 개념 | 용도 |
|-----------|----------|------|
| linear | t | 이동, 루프 애니메이션 |
| ease-in | t² | 공격 준비 (느리게 시작) |
| ease-out | 1-(1-t)² | 복귀 (느리게 종료) |
| ease-in-out | smoothstep | 호흡, 대기 |
| snap | step (즉시) | 타격 순간, 피격 |
| bounce | 바운스 감쇠 | 대미지 숫자, 아이템 드롭 |

### 4.2 히트 스톱 규격

| 공격 유형 | 히트 스톱 (프레임) | 적용 |
|-----------|------------------|------|
| 일반 공격 | 2 | 공격자+피격자 동시 정지 |
| 스킬 공격 | 3 | 공격자+피격자 동시 정지 |
| 크리티컬 | 4 | + 카메라 흔들림 |
| 보스 공격 | 3 | 피격자만 정지 (보스 강인함) |

### 4.3 대미지 팝업 애니메이션

| 단계 | 지속 | 이징 | 설명 |
|------|------|------|------|
| 출현 | 0.1s | snap | 즉시 출현. 크기 120% |
| 바운스 | 0.15s | bounce | 100%로 축소 |
| 상승 | 0.5s | ease-out | 위로 20px 이동. 불투명도 유지 |
| 소멸 | 0.3s | ease-in | 불투명도 0. 위로 추가 10px |

**색상:**

| 유형 | 색상 | 크기 |
|------|------|------|
| 일반 피해 | 흰 `#FFFFFF` | 12px |
| 크리티컬 | 금 `#FFD700` | 16px + "!" |
| 치유 | 녹 `#2ECC71` | 12px + "+" |
| 미스 | 회 `#808080` | 10px |
| 면역 | 파랑 `#3498DB` | 10px |
