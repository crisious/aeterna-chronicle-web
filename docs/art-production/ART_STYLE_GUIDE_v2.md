**단일 스타일: 16-bit 픽셀아트 (SNES/GBA 스타일)**

| 규칙 | 내용 |
|------|------|
| 스타일 | **16-bit 픽셀아트** — SNES RPG (FF6, 크로노트리거) 기반 |

## 2) 에셋별 규격

### 캐릭터 (플레이어/NPC)
| 항목 | 규격 |
|------|------|
| 원본 크기 | **32×48 px** (1타일=32px 기준, 1.5타일 높이) |
| 게임 표시 | 32×48 또는 2x 스케일 → 64×96 |
| 스타일 | FF6 스프라이트 — 2등신~3등신, 뚜렷한 윤곽선, 제한 색상 |
| 방향 | 4방향 (상/하/좌/우) × 3프레임 워크 = 12프레임 |
| 프롬프트 키워드 | `pixel art, 16-bit SNES RPG sprite, 32x48, chibi proportions, limited palette, dark fantasy` |

### 몬스터
| 항목 | 규격 |
|------|------|
| 원본 크기 | 일반: **32×32**, 엘리트: **48×48**, 보스: **64×64** |
| 스타일 | FF6 몬스터 스프라이트 — 정면, 단일 포즈, 디테일한 픽셀 |
| 프롬프트 | `pixel art monster sprite, 16-bit SNES style, front-facing, dark fantasy RPG, limited palette, black outline` |

### 환경 오브젝트 (나무/바위/크리스탈 등)
| 항목 | 규격 |
|------|------|
| 원본 크기 | 소형: **16×16**, 중형: **32×32**, 대형: **32×64** |
| 스타일 | RPG Maker VX/MV 스타일 픽셀 타일 오브젝트 |
| 프롬프트 | `pixel art RPG object, 16-bit style, top-down perspective, limited palette, clean edges, game asset` |

### 지면 타일
| 항목 | 규격 |
|------|------|
| 원본 크기 | **32×32 px** (심리스 반복 가능) |
| 스타일 | 탑다운 RPG 타일 — 잔디/돌/모래/눈/금속 |
| 프롬프트 | `pixel art ground tile, 32x32, seamless tileable, top-down RPG, 16-bit SNES style` |

### 배경 (패럴랙스)
| 항목 | 규격 |
|------|------|
| 원본 크기 | **320×180 px** (게임 해상도의 1/4, pixelArt 4x 스케일) |
| 게임 표시 | Phaser pixelArt: true → 4x nearest-neighbor → 1280×720 |
| 스타일 | 16-bit 픽셀아트 배경 — 색 수 32색 이하 |
| 프롬프트 | `pixel art background, 16-bit SNES RPG style, limited palette, panoramic, dark fantasy` |

## 4) ComfyUI 프롬프트 표준

### 공통 접두사 (모든 에셋)
```
pixel art, 16-bit SNES RPG style, limited color palette, clean pixel edges, no anti-aliasing, game asset,
```

### 공통 네거티브
```
realistic, photorealistic, 3D render, soft painting, watercolor, anime cel shading, blurry, gradient, high detail illustration, anti-aliased, smooth edges, photograph
```

### 해상도별 SD 1.5 설정
| 에셋 | 생성 크기 | Steps | CFG |
|------|-----------|-------|-----|
| 캐릭터 32×48 | 256×384 (8x) | 25 | 7.0 |
| 몬스터 32×32 | 256×256 (8x) | 25 | 7.0 |
| 오브젝트 32×32 | 256×256 (8x) | 22 | 7.0 |
| 타일 32×32 | 256×256 (8x) | 22 | 7.0 |
| 배경 320×180 | 640×360 (2x) | 28 | 7.0 |

> SD 1.5는 512 이하 해상도에서 최적. 생성 후 8x→원본 크기로 다운스케일(nearest-neighbor).

## 6) 실행 계획

### Phase 1: 핵심 (에테르 평원 + 전투)
1. 플레이어 캐릭터 6클래스 × 4방향 = 24 스프라이트
2. 오프라인 폴백 몬스터 3마리
3. NPC 2명 (마테우스, 크리오)
4. 에테르 평원 오브젝트 3종 (나무, 바위, 크리스탈)
5. 에테르 평원 타일 1종
6. 에테르 평원 배경 2장 (FAR, SKY)

### Phase 2: 전 존 배경 + 오브젝트
- 10존 × 배경 2장 = 20장
- 10존 × 오브젝트 3종 = 30장
- 10존 × 타일 1종 = 10장

### Phase 3: 몬스터 + NPC 전수
- 몬스터 120종
- NPC 40명
