## 1) 코스메틱 시스템 개요

### 1.1 핵심 원칙

| 원칙 | 설명 |
|------|------|
| P2W 금지 | 코스메틱은 외형 변경만. 스탯 영향 0 |
| 등급 시각화 유지 | 코스메틱 적용 시에도 장비 등급 테두리/오라는 보존 |
| 시즌 테마 차별화 | 시즌별 고유 비주얼 테마로 수집 동기 부여 |
| 시간 제한 + 재판매 | 시즌 종료 후 일부 복각. FOMO + 장기 수익 |

### 1.2 총괄 수량 (150종)

| 시즌 | 수량 | 테마 |
|------|------|------|
| 시즌 1: 기억의 잔광 | 50종 | 에테르+기억 결정 모티프 |
| 시즌 2: 깨어나는 봉인 | 50종 | 봉인 해체+원소 폭발 모티프 |
| 시즌 3: 기억의 심연 | 50종 | 심연+시간 왜곡 모티프 |
| **합계** | **150종** | |

### 3.2 시즌 2: 깨어나는 봉인 (Awakening)

| 요소 | 규칙 |
|------|------|
| 주요 색상 | 봉인 적 `#FF6347` + 불꽃 주황 `#FF8C00` + 금색 `#FFD700` |
| 모티프 | 12봉인 문양, 균열, 불꽃, 폭발, 사슬 해체 |
| 질감 | 갈라짐 + 내부 발광 + 용암/마그마 흐름 |
| 금지 요소 | 기억 결정/별빛 (시즌1), 심연/왜곡 (시즌3) |

> 시즌 2 카탈로그: 시즌 1과 동일 50종 구조 (스킨 10 + 무기 8 + 오라 5 + 펫 7 + 마운트 5 + 이모트 8 + 칭호 7)  
> 비주얼 테마만 "봉인 해체+화염+금색" 으로 전환.

#### 시즌 2 대표 아이템 (SSR급)

| 카테고리 | 이름 | 비주얼 키워드 |
|---------|------|-------------|
| 캐릭터 스킨 | 봉인 해방 기사 | 갈라진 갑옷 + 내부 용암 빛 + 봉인 문양 |
| 무기 스킨 | 봉인 파괴 대검 | 거대 검 + 12봉인 문양 각인 + 붉은 균열 |
| 오라 | 봉인 해체 오라 | 캐릭 주위 12봉인 문양 파편 회전 + 불꽃 파티클 |
| 펫 | 봉인 불사조 | 화염 깃털 불사조 미니 |
| 마운트 | 봉인 전차 | 불꽃 바퀴 전차 + 봉인 문양 깃발 |
| 이모트 | 봉인 해제 의식 | 봉인 문양 그리기 → 폭발 연출 |
| 칭호 프레임 | 봉인 해방자 프레임 | 불꽃 + 갈라진 금속 테두리 |

## 4) 코스메틱 스프라이트 규격
