## 5) 의존성 그래프

```
[스타일 가이드 확정] ─────────────────────────────────┐
         │                                             │
         ▼                                             ▼
[마스터 프롬프트 확정] ──► [LoRA 학습 (선택)]    [폴더 구조 세팅]
         │                       │                     │
         ▼                       ▼                     ▼
[AI 이미지 생성] ◄──── [LoRA/ControlNet 적용]   [Git LFS 설정]
         │
         ▼
[배경 제거] ──► [색보정] ──► [QA 검증] ──► [스프라이트 시트 조립]
                                │                     │
                                ▼                     ▼
                         [수동 QA 리뷰]         [메타데이터 JSON]
                                │                     │
                                ▼                     ▼
                         [Git 커밋 (art/* 브랜치)]
                                │
                                ▼
                         [develop 머지] ──► [인게임 테스트]
```

### 5.1 크리티컬 패스

```
스타일 가이드 → 마스터 프롬프트 → 캐릭터 AI 생성 → 후처리 → QA → 시트 조립
                                                                    │
                    ┌──────────── 병렬 ─────────────┐               ▼
                    ▼                                ▼          인게임 테스트
              몬스터 생성                      타일/아이콘 생성
```

### 5.2 병렬화 가능 구간

| 구간 | 병렬 가능 | 비고 |
|------|----------|------|
| AI 생성 | ✅ 높음 | 카테고리별 독립 생성 |
| 배경 제거 | ✅ 높음 | 완전 자동, 배치 |
| 색보정 | ✅ 높음 | 완전 자동, 배치 |
| QA (자동) | ✅ 높음 | 배치 실행 |
| QA (수동) | ⚠️ 중간 | 리뷰어 병목 |
| 시트 조립 | ✅ 높음 | 모션 완결 단위 |
| 인게임 테스트 | ❌ 낮음 | 순차 통합 |
