## 1) NPC 디자인 총괄

### 1.1 규격

| 항목 | 규격 |
|------|------|
| 대상 | 주요 NPC 30명 |
| 초상화 | 256×256 px (대화 UI 표시용) |
| 필드 스프라이트 | 64×64 px (기본, idle+walk) |
| 뷰 | 정면 초상화 + 정면 스프라이트 (최소) |
| 표정 | 기본/기쁨/분노/슬픔 — 4종 (초상화) |

### 1.2 성격→외형 매핑 가이드

| 성격 유형 | 외형 표현 규칙 |
|-----------|--------------|
| 지혜/학자 | 안경/서적/긴 로브, 차분한 색상, 침착한 표정 |
| 전사/호탕 | 넓은 어깨, 갑옷/무기 소지, 강렬한 색상 |
| 신비/마법 | 발광 눈/문양, 부유 오브젝트, 보라/청 계열 |
| 음침/위험 | 후드/그림자, 어두운 색상, 붉은 포인트 |
| 친근/상인 | 밝은 색상, 넉넉한 체형, 주머니/가방/상품 |
| 귀족/권위 | 화려한 의상, 금/은 장식, 높은 자세 |
| 야성/자연 | 가죽/모피, 동물 모티프, 녹/갈 계열 |
| 기계/공학 | 고글/도구벨트, 기어 장식, 금속/갈 계열 |

## 3) AI 생성 프롬프트 템플릿

### 3.1 NPC 초상화 프롬프트

```
Template: NPC_PORTRAIT

2D pixel art RPG NPC portrait, bust shot, front-facing,
{npc_name}, {role_description},
{appearance_description},
{clothing_description},
{personality_vibe} expression,
{region_name} region, {region_color_palette} color scheme,
dark fantasy medieval setting, memory/ether theme,
top-left 45-degree lighting, 3-step shading,
2px black outline, no anti-aliasing,
256x256 resolution, dark gradient background,
game dialogue portrait style

Variables:
  npc_name: NPC 이름
  role_description: "mentor mage" / "blacksmith" / "healer" 등
  appearance_description: 위 디자인 시트의 외형 항목
  clothing_description: 위 디자인 시트의 의상 항목
  personality_vibe: "calm and wise" / "fierce and proud" / "warm and gentle" 등
  region_name: 소속 지역명
  region_color_palette: 지역 팔레트 (style-guide.md 참조)
```

### 3.2 NPC 필드 스프라이트 프롬프트

```
Template: NPC_FIELD_SPRITE

2D pixel art RPG NPC sprite, full body, front-facing,
{npc_name}, {role_description},
simplified version of portrait design,
{clothing_simplified},
standing idle pose,
64x64 pixel art, clean pixel edges,
2px black outline, no anti-aliasing,
transparent background, game sprite ready

Variables:
  clothing_simplified: 초상화 대비 단순화된 의상 설명
```

### 3.3 NPC 표정 변형 프롬프트

```
Template: NPC_EXPRESSION

2D pixel art RPG NPC portrait, bust shot,
{npc_name}, {expression_type} expression,
same design as base portrait,
{expression_specific_change},
256x256 resolution, dark gradient background

Variables:
  expression_type: "happy" | "angry" | "sad"
  expression_specific_change: "smiling, eyes crinkled" / "furrowed brows, gritted teeth" 등
```

### 3.4 네거티브 프롬프트 (NPC 전용)

```
Negative: 3D render, realistic photo, blurry, watermark,
low quality, deformed, extra limbs, bad anatomy,
wrong eye color, inconsistent clothing from reference,
modern clothing, sci-fi elements (unless specified),
noisy, grain, JPEG artifacts, cropped
```
