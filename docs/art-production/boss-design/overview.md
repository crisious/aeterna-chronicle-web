## 1) 레이드 보스 6종 상세 디자인

### 1.1 네뷸로스 (Nebulos) — 에레보스 레이드

| 항목 | 설명 |
|------|------|
| 크기 | XXL (256×256 px) |
| 형태 | 성운 형태의 거대 존재. 중심 코어 + 가스 외피 + 촉수 |
| 코어 | 중앙에 박동하는 기억 수정 (심장). 크랙에서 에테르 빛 |
| 외피 | 반투명 성운 가스. 안개 회색 + 보라 + 청 혼합 |
| 촉수 | 6~8개 가스 촉수. 끝에 기억 파편 박힌 집게 |
| 눈 | 가스 내부에 떠다니는 다수의 발광 눈 (5~7개) |

#### 페이즈별 외형 변화

| 페이즈 | HP 범위 | 외형 변화 | 색상 변화 |
|--------|---------|----------|----------|
| 1 | 100~75% | 기본 형태. 안개 회색 주도. 느린 촉수 움직임 | 안개 회색 `#2D2D3F` + 보라 `#5C4A72` |
| 2 | 75~50% | 외피 수축, 촉수 추가 2개. 눈 발광 강화 | 보라 강화 `#6A0DAD` + 적색 안개 추가 |
| 3 | 50~25% | 외피 폭발적 확장, 코어 노출. 전장 안개 | 적보라 `#8B008B` + 에테르 펄스 `#89CFF0` |
| 4 (광폭) | 25~0% | 코어 균열, 기억 폭풍. 모든 눈 적색 전환 | 적색 `#FF0000` + 흑색 `#0A0A0A` |

#### 공격 모션

| 모션 | 프레임 | FPS | 설명 |
|------|--------|-----|------|
| idle | 6 | 4 | 성운 유동, 촉수 느린 흔들림 |
| tentacle_slam | 8 | 12 | 촉수 들어올리기 → 바닥 강타 |
| gaze_beam | 6 | 10 | 눈 집중 → 기억 흡수 빔 발사 |
| nebula_burst | 8 | 10 | 외피 수축 → 전방위 폭발 |
| phase_transition | 10 | 8 | 코어 빛 증폭 → 형태 변화 → 충격파 |
| memory_storm | 6 | 12 | P3+ 전용. 기억 파편 회오리 |

## 3) AI 생성 프롬프트 — 보스 전용

### 3.1 레이드 보스 프롬프트

```
Template: RAID_BOSS

2D pixel art RPG raid boss, {boss_name},
MASSIVE {form_description},
{detail_description},
{phase_number} phase appearance,
{region_name} region, {boss_color_palette},
XXL size (256x256 pixels),
menacing, epic scale, intricate details,
3px black outline + 3px {aura_color} outer glow,
particle effects around body,
dark fantasy RPG, boss encounter art,
top-left 45-degree lighting, 3-step shading,
transparent background

Variables:
  boss_name: 보스 이름
  form_description: "nebula entity" / "glass-flame dragon" 등
  detail_description: 위 디자인 시트 상세
  phase_number: "phase 1 (full health)" ~ "phase 4 (enraged)"
  boss_color_palette: 보스 전용 팔레트
  aura_color: 오라 색상
```

### 3.2 던전 보스 프롬프트

```
Template: DUNGEON_BOSS

2D pixel art RPG dungeon boss, {boss_name},
{form_description},
{detail_description},
{region_name} region, {region_color_palette},
XL size (128x128 pixels),
intimidating pose, boss encounter,
3px black outline + 2px {aura_color} glow,
dark fantasy RPG boss sprite,
transparent background, game asset ready
```
