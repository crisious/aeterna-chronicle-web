## 1) 컷씬 시스템 개요

### 1.1 기술 규격

| 항목 | 규격 |
|------|------|
| 해상도 | 1280×720 px (16:9) |
| 스타일 | 2D 일러스트 + 제한 애니메이션 (Ken Burns 효과 + 오버레이 파티클) |
| 프레임 | 캐릭터 스탠딩 일러 + 배경 일러 + 대사 텍스트 박스 |
| 전환 | 페이드 인/아웃 (0.5초), 디졸브 (0.8초), 와이프 (특수 장면) |
| 음악 | 컷씬별 전용 BGM 또는 지역 BGM 변형 |
| 보이스 | 주요 대사 음성 (한/영/일 3언어) — Phase 14+ |

### 1.2 컷씬 유형

| 유형 | 길이 | 용도 | 연출 수준 |
|------|------|------|----------|
| 오프닝 | 2~3분 | 게임 시작 | 풀 애니메이션 (외주) |
| 챕터 오프닝 | 30~60초 | 챕터 시작 | 일러스트 + Ken Burns + 나레이션 |
| 핵심 스토리 | 30~90초 | 주요 사건 | 일러스트 + 대사 + 이펙트 |
| 보스 인트로 | 15~30초 | 보스전 직전 | 보스 클로즈업 + 대사 + 셰이크 |
| 엔딩 | 2~5분 | 4가지 엔딩 | 풀 일러스트 시퀀스 + 크레딧 |
