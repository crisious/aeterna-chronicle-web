## 2) 등급별 테두리/이펙트 규격

```
┌─ 외부 글로우 (등급별) ─────────────────┐
│ ┌─ 테두리 (등급 색상) ──────────────┐ │
│ │ ┌─ 내부 배경 (그라데이션) ──────┐ │ │
│ │ │                             │ │ │
│ │ │    아이콘 콘텐츠 (60×60)     │ │ │
│ │ │                             │ │ │
│ │ └─────────────────────────────┘ │ │
│ └────────────────────────────────┘ │
└──────────────────────────────────────┘
```

### 2.2 등급별 상세 규격

| 등급 | 테두리 두께 | 테두리 색상 | 내부 배경 | 외부 글로우 | 코너 장식 | 파티클 |
|------|-----------|-----------|----------|-----------|----------|--------|
| 일반 | 1px | `#808080` | `#1A1A2E` → `#0D0D15` | 없음 | 없음 | 없음 |

## 3) 아이템 아이콘 디자인 시트 (100종)

> P11 icon-spec.md의 100종 목록을 유지. 여기서는 카테고리별 디자인 지침 추가.

### 3.1 무기 (20종)

**디자인 지침**:
- 검/창/활/지팡이/단검 — 실루엣만으로 무기 타입 구분 필수
- 클래스 시그니처 색상 반영: 기사(강철 청), 기억술사(보라), 그림자(녹), 시간(금)
- 등급 올라갈수록: 장식 증가 + 발광 요소 추가 + 문양 복잡도 증가
- 16×16 축소 시 형태 인식 가능해야 함

| ID 범위 | 수량 | 주요 클래스 | 설명 |
|--------|------|-----------|------|
| ICON-W-01~05 | 5 | 에테르 기사 | 철검→에테르검→기억수호검→봉인성검→양손대검 |
| ICON-W-06~10 | 5 | 기억술사 | 나무지팡이→결정지팡이→원소지팡이→망각의홀→마법서 |
| ICON-W-11~15 | 5 | 그림자 직조사 | 단궁→그림자장궁→안개석궁→기억저격총→독단검 |
| ICON-W-16~20 | 5 | 시간 수호자 | 시간단검→시간사슬검→영원의창→시간파괴대검→신화블레이드 |

### 3.2 방어구 (20종)

**디자인 지침**:
- 부위(머리/몸통/손/발/방패/망토) 실루엣 구분 필수
- 재질감 표현: 천(부드러운 선) → 가죽(중간) → 판금(직선+하이라이트)
- 등급별: 단색→문양→발광→결정 부착→완전 에테르화

| ID 범위 | 수량 | 부위 | 설명 |
|--------|------|------|------|
| ICON-A-01~06 | 6 | 머리 | 투구 4등급 + 관 + 왕관 |
| ICON-A-07~12 | 6 | 몸통 | 천로브→가죽→판금→황금갑주→빙결→신화 |
| ICON-A-13~15 | 3 | 손/발 | 장갑, 장화 등급별 |
| ICON-A-16~18 | 3 | 방패 | 나무→기사→대방패 |
| ICON-A-19~20 | 2 | 망토 | 그림자→시간 |

### 3.3 장신구 (15종)

**디자인 지침**:
- 작은 오브젝트이므로 중앙 보석/펜던트 강조
- 등급은 보석 색상+발광으로 구분 (테두리 + 보석 이중 구분)

| ID 범위 | 수량 | 종류 |
|--------|------|------|
| ICON-J-01~04 | 4 | 목걸이 (파편→에테르→봉인→에바) |
| ICON-J-05~08 | 4 | 반지 (에테르→기사인장→원소5종 중 2) |
| ICON-J-09~11 | 3 | 귀걸이/팔찌/브로치 |
| ICON-J-12~15 | 4 | 시간회중시계/레테눈물/원소반지3종 잔여 |

### 3.4 소비 아이템 (25종)

**디자인 지침**:
- 포션: 유리병 형태 + 액체 색상으로 즉시 구분 (빨강=HP, 파랑=MP, 녹=해독)
- 두루마리: 양피지 + 리본 색상으로 등급 구분
- 음식: 접시/그릇 위 음식 실루엣
- 재료: 날것 느낌 (광석=결정, 허브=잎사귀, 가죽=가죽 조각)

### 3.5 퀘스트/기타 아이템 (20종)

**디자인 지침**:
- 퀘스트 아이템: 금색 ❗ 표시 오버레이 + 특수 배경 톤
- 화폐: 금화/은화는 원형 실루엣 유지, 에테르 결정은 다면체
- 상자: 등급 색상 나무 상자 + 자물쇠/빛

### 7.1 아이템 아이콘 프롬프트

```

2D pixel art RPG item icon, {item_name},

{rarity} rarity tier,
64x64 pixel art, centered composition,
dark gradient background (#1A1A2E to #0D0D15),
{rarity_border} border, clean readable silhouette,
{additional_effects},
dark fantasy medieval style, game UI icon,
no text, no watermark, transparent outer area

Variables:
  rarity: common|uncommon|rare|epic|legendary|mythic
  rarity_border: gray 1px | green 1px | blue 2px | purple 2px+glow | orange 2px+glow+corners | red-gold 3px gradient+glow+corners+particles
  additional_effects: (mythic 이상) "subtle particle effects, glow emanating"
```
