## 1) 이펙트 시스템 개요

### 1.1 총괄 수량

| 카테고리 | 수량 |
|---------|------|
| 스킬 이펙트 (5클래스 × 30) | 150종 |
| 히트 이펙트 | 8종 |
| 사망 이펙트 | 5종 |
| 공통 이펙트 (회복/버프/디버프) | 12종 |
| **합계** | **175종** |

### 1.2 이펙트 카테고리 분류

| 카테고리 | 설명 | 예시 |
|---------|------|------|
| **타격 (Hit)** | 직접 접촉 공격 이펙트 | 검격, 주먹, 단검 |
| **투사체 (Projectile)** | 발사→비행→착탄 3단계 | 화살, 화염구, 빙창 |
| **범위 (AoE)** | 지정 영역 이펙트 | 메테오, 강우, 시간 정지 |
| **버프 (Buff)** | 아군 강화 이펙트 | 방어 오라, 속도 버프 |
| **디버프 (Debuff)** | 적 약화 이펙트 | 독 안개, 둔화 사슬 |
| **지속 (DoT/HoT)** | 시간당 반복 이펙트 | 출혈, 독, 재생 |
| **궁극기 (Ultimate)** | 전체 화면 연출 | 성검 소환, 시간 파괴 |

### 2.2 스킬 이펙트 시트 규격

| 이펙트 유형 | 시트 크기 | 프레임 수 | 프레임 크기 | FPS |
|-----------|----------|----------|-----------|-----|
| 소형 타격 | 256×256 | 4~6 (1행) | 64×64 | 15 |
| 중형 범위 | 512×512 | 8~12 (2행) | 128×128 | 12 |
| 대형 범위 | 1024×512 | 8~16 (2행) | 256×128 | 10 |
| 투사체 | 256×64 | 4 (1행) | 64×64 | 15 |
| 투사체 착탄 | 256×256 | 4~6 | 64×64 | 15 |
| 궁극기 전체화면 | 1024×1024 | 16~24 (4행) | 256×256 | 12 |

### 2.3 이펙트 블렌드 모드

| 용도 | 블렌드 모드 | 설명 |
|------|-----------|------|
| 화염/폭발 | Additive | 밝은 코어, 합산 발광 |
| 얼음/물 | Normal + Additive 하이라이트 | 반투명 얼음 + 반사 |
| 번개 | Additive | 순간 밝은 플래시 |
| 그림자/독 | Multiply + Normal | 어두운 안개 + 불투명 파티클 |
| 에테르/성스러움 | Additive + Screen | 부드러운 발광 |
| 궁극기 배경 | Screen (어두움 제거) | 전체 화면 밝기 부스트 |
