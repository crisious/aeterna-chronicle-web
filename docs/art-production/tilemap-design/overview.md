## 1) 레이어 시스템 상세

### 1.1 4레이어 구조

| 레이어 | Z-Order | 렌더링 | 충돌 | 설명 |
|--------|---------|--------|------|------|
| **바닥 (Ground)** | 0 | Tilemap | ❌ | 기본 지형. 오토타일 3×3 기반, 심리스 타일링 필수 |
| **장식 (Decoration)** | 1~2 | Sprite/Tilemap | 선택 | 바닥 위 장식물 + 배치 오브젝트. 캐릭터 앞뒤 정렬 기준: Y-sort |
| **충돌 (Collision)** | - | 비가시 | ✅ | 이동 불가 영역. 타일 단위 사각 충돌 + 폴리곤 보조 |
| **상호작용 (Interaction)** | - | 비가시 (아이콘 표시) | ❌ | 트리거 영역. 접근 시 ❗ 아이콘 팝업 |

### 1.2 충돌 타일 규칙

| 규칙 | 설명 |
|------|------|
| 완전 충돌 | 1타일 전체가 이동 불가 (벽, 건물, 대형 바위) |
| 절반 충돌 | 타일 상단/하단/좌/우 절반만 충돌 (울타리, 낮은 벽) |
| 경사 충돌 | 대각선 충돌면 (계단, 경사로) — 45° 고정 |
| 수중 충돌 | 이동 속도 50% 감소 + 수영 모션 전환 |
| 높이 차 충돌 | 높이 레벨 다를 시 통과 불가 (절벽, 다리) |

### 1.3 상호작용 타일 유형

| 유형 | 트리거 | 아이콘 | 예시 |
|------|--------|--------|------|
| 채집 | 접근 + 키 입력 | ⛏️ | 기억 수정, 허브, 광석 |
| 조사 | 접근 + 키 입력 | 🔍 | 우편함, 묘비, 게시판 |
| 진입 | 접근 자동 | 🚪 | 던전 입구, 포탈, 계단 |
| NPC | 접근 + 키 입력 | 💬 | NPC 대화 |
| 함정 | 접근 자동 | ⚠️ (최초 1회만) | 독침, 함정 바닥 |
| 퍼즐 | 접근 + 키 입력 | ⚙️ | 레버, 스위치, 압력판 |
