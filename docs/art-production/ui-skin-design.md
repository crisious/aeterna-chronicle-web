## 1) UI 디자인 원칙

| 원칙 | 설명 |
|------|------|
| 다크 판타지 톤 | 어두운 배경 + 금속/금색 액센트. 밝고 화사한 UI 금지 |
| 픽셀 친화 | 모든 UI 요소 1px/2px 단위. 안티앨리어싱 최소화 |
| 정보 계층 | 중요 정보 → 밝은 색/큰 글씨, 보조 정보 → 어두운/작은 글씨 |
| 등급 시각화 | 아이템/스킬 등급은 테두리+글로우로 즉시 인지 |
| 접근성 | 컬러블라인드 대응 — 색상 외 아이콘/모양으로 구분 보조 |

## 3) UI 컴포넌트 규격

### 3.1 프레임 (Frame)

| 유형 | 크기 | 코너 | 보더 | 배경 | 용도 |
|------|------|------|------|------|------|
| 기본 패널 | 가변 | 2px 라운드 | 1px `#5A5A6A` | `#16213E` 90% | 대화창, 정보 패널 |
| 강조 패널 | 가변 | 2px 라운드 | 2px `#FFD700` | `#16213E` 95% | 경고, 중요 안내 |
| 투명 패널 | 가변 | 0px | 없음 | `#000000` 60% | 오버레이, HUD 배경 |
| 모달 | 400×300+ | 4px 라운드 | 2px `#FFD700` | `#1A1A2E` 98% | 확인창, 설정 |
| 툴팁 | 자동 (패딩 8px) | 2px 라운드 | 1px `#4A4A5A` | `#0D0D1A` 95% | 아이템/스킬 설명 |

### 3.2 버튼 (Button)

| 상태 | 배경 | 보더 | 텍스트 | 추가 |
|------|------|------|--------|------|
| 기본 | `#3A3A4A` → `#2A2A3A` | 1px `#5A5A6A` + 1px 상단 하이라이트 | `#E8E8E8` | — |
| 호버 | 밝기 +15% | 동일 | `#FFFFFF` | 미세 발광 |
| 클릭 | 밝기 -10% | 1px 인셋 그림자 | `#E8E8E8` | 눌림 1px 이동 |
| 비활성 | `#2A2A2A` | 1px `#3A3A3A` | `#606060` | — |
| 위험 | `#4A1A1A` → `#3A0A0A` | 1px `#FF4444` | `#FF6666` | 삭제/파괴 전용 |
| 확인 (주요) | `#1A3A5A` → `#0A2A4A` | 1px `#89CFF0` | `#89CFF0` | 주요 행동 |

**사이즈 규격**:
- 소 (Small): 60×24 px — 닫기, 취소
- 중 (Medium): 120×32 px — 확인, 저장
- 대 (Large): 200×40 px — 메인 행동

### 3.3 슬라이더 (Slider)

| 요소 | 크기 | 색상 |
|------|------|------|
| 트랙 | 200×4 px | `#2A2A3A` |
| 채움 | 가변×4 px | 에테르 청 `#89CFF0` 그라데이션 |
| 핸들 | 12×12 px 원 | `#E8E8E8` 외곽 + `#89CFF0` 내부 |
| 핸들 (호버) | 14×14 px | 발광 추가 |
| 눈금 텍스트 | 좌우 끝 | `#A0A0A0` 8px 폰트 |

### 3.4 토글 (Toggle)

| 상태 | 트랙 | 핸들 |
|------|------|------|
| OFF | `#2A2A3A` (24×12 px) | `#606060` 좌측 (10×10 px) |
| ON | `#1A3A5A` | `#89CFF0` 우측 |
| 전환 | 200ms ease-out 슬라이드 | |

### 3.5 탭 (Tab)

| 상태 | 배경 | 보더 | 텍스트 |
|------|------|------|--------|

| 활성 | `#16213E` | 하단 2px `#FFD700` | `#FFFFFF` |
| 호버 | `#20203A` | — | `#E8E8E8` |

## 4) 화면별 UI 레이아웃

### 4.1 HUD (전투/필드)

```
┌─────────────────────────────────────────────────┐
│ [캐릭 초상] HP ████████░░ MP ████░░░░░░         │  ← 좌상단
│ Lv.45 에테르 기사         EXP ██████░░░░        │
├─────────────────────────────────────────────────┤
│                                                 │
│            [게임 화면 영역]                        │
│                                                 │
├─────────────────────────────────────────────────┤
│ [미니맵]                            [퀵슬롯 1~8] │  ← 하단
│ [채팅창]                            [스킬바]      │
│ [메뉴 아이콘 5개: 캐릭/인벤/스킬/퀘스트/설정]       │
└─────────────────────────────────────────────────┘
```

**HUD 요소 규격**:

| 요소 | 위치 | 크기 | 설명 |
|------|------|------|------|
| 캐릭 초상 | 좌상 | 48×48 px | 현재 캐릭터 얼굴 |
| HP 바 | 초상 우측 | 200×12 px | `#E74C3C` → `#2ECC71` |
| MP 바 | HP 아래 | 200×10 px | `#3498DB` → `#9B59B6` |
| EXP 바 | MP 아래 | 200×6 px | `#F39C12` |
| 미니맵 | 좌하 | 128×128 px | 원형 마스크, 반투명 배경 |
| 퀵슬롯 | 우하 | 36×36 × 8칸 | 아이템/포션 배치 |
| 스킬바 | 퀵슬롯 위 | 36×36 × 8칸 | 스킬 아이콘 + 쿨다운 오버레이 |
| 채팅창 | 좌하 (미니맵 우측) | 300×120 px | 투명 배경, 스크롤 |
| 메뉴 아이콘 | 하단 중앙 | 32×32 × 5 | 캐릭/인벤/스킬/퀘스트/설정 |

### 4.2 인벤토리

```
┌──────────────────────────────────────┐
│ [캐릭터 프리뷰]  │ [장비 슬롯 8칸]    │
│ (전신 일러스트)   │ 머리/몸/손/발      │
│                  │ 무기/방패/목걸이/반지│
├──────────────────┼───────────────────┤
│ [탭: 전체|무기|방어구|소비|재료|퀘스트]  │
│ ┌──┬──┬──┬──┬──┬──┐                 │
│ │  │  │  │  │  │  │ ← 6×6 그리드    │
│ ├──┼──┼──┼──┼──┼──┤   (36슬롯/페이지)│
│ │  │  │  │  │  │  │                 │
│ ├──┼──┼──┼──┼──┼──┤                 │
│ │  │  │  │  │  │  │                 │
│ └──┴──┴──┴──┴──┴──┘                 │
│ 골드: 1,234,567   페이지: 1/5        │
└──────────────────────────────────────┘
```

**슬롯 규격**: 36×36 px, 2px 간격, 등급 테두리 색상 적용

### 4.3 상점

```
┌──────────────────────────────────────┐
│ [NPC 초상+이름]  "어서 오세요..."      │
├──────────────────────────────────────┤
│ [탭: 구매|판매|수리]                   │
│ ┌─────────────┬──────────────────┐  │
│ │ [상품 목록]   │ [선택 아이템 상세] │  │
│ │ 아이콘+이름   │ 이름, 등급        │  │
│ │ +가격        │ 스탯, 설명        │  │
│ │ (스크롤)     │ 가격, 구매 버튼   │  │
│ └─────────────┴──────────────────┘  │
│ 보유 골드: 1,234,567                  │
└──────────────────────────────────────┘
```

### 4.4 설정 (Settings)

| 카테고리 | 항목 |
|---------|------|
| 그래픽 | 해상도 (드롭다운), 전체화면 (토글), 프레임 제한 (슬라이더), 파티클 밀도 (슬라이더) |
| 사운드 | BGM 볼륨 (슬라이더), SFX 볼륨 (슬라이더), 환경음 (슬라이더), 음소거 (토글) |
| 게임 | 언어 (드롭다운), 자막 (토글), 텍스트 속도 (슬라이더), 자동 줍기 (토글) |
| 조작 | 키 매핑 테이블 (재설정 버튼), 컨트롤러 지원 (토글) |
| 접근성 | UI 테마 (드롭다운: 기본/다크/시즌), 색약 모드 (토글), 폰트 크기 (슬라이더) |

### 4.5 채팅

```
┌─────────────────────────┐
│ [탭: 전체|파티|길드|시스템] │
│ ─────────────────────── │
│ [메시지 영역]             │
│ 시간 닉네임: 내용         │
│ (스크롤, 투명 배경)       │
│ ─────────────────────── │
│ [입력창] [전송]           │
└─────────────────────────┘
```

채팅 크기: 접이식 — 최소 300×80, 최대 500×250  
배경 불투명도: 기본 40%, 활성 시 80%

## 5) 특수 UI 요소

### 5.1 대미지 팝업

| 유형 | 색상 | 크기 | 애니메이션 |
|------|------|------|----------|
| 일반 피해 | `#FFFFFF` + 1px `#000000` 외곽 | 12px | 위로 튀고 페이드 (0.8초) |
| 크리티컬 | `#FFD700` + 2px `#000000` 외곽 | 18px | 커졌다 줄어듦 + 튀김 (1.2초) |
| 마법 피해 | `#89CFF0` + 1px 외곽 | 14px | 위로 float (1초) |
| 회복 | `#2ECC71` + 1px 외곽 | 12px | 위로 올라감 (0.8초) |
| 미스 | `#808080` | 10px | 좌우 흔들림 (0.5초) |
| 면역 | `#606060` | 10px | 투명도 깜빡임 (0.5초) |

### 5.2 쿨다운 오버레이

```
  ┌────────┐
  │ 스킬    │  ← 평소: 아이콘 100%
  │ 아이콘  │
  └────────┘
  
  ┌────────┐
  │ ██████ │  ← 쿨다운: 위에서 아래로 어두운 오버레이 시계방향 와이프
  │ █ 5.2s │     중앙에 남은 초 (소수 1자리)
  │ ██████ │
  └────────┘
```

### 5.3 등급 연출

| 등급 | 획득 연출 | 장비 시 연출 |
|------|----------|------------|
| 일반 | 없음 | 없음 |
| 고급 | 초록 테두리 플래시 | 없음 |
| 희귀 | 파란 테두리 플래시 + 이름 파란색 | 캐릭터 얇은 파란 오라 |
| 영웅 | 보라 광선 + 사운드 | 보라 오라 + 미세 파티클 |
| 전설 | 황금 광선 폭발 + 전용 사운드 + 화면 셰이크 | 금색 오라 + 파티클 |
| 신화 | 적금 방사 + 전체 화면 연출 + 전용 팡파르 | 적금 오라 + 활활 파티클 |
| 에테르 | 청백 펄스 + 화면 반전 플래시 + 특수 팡파르 | 청백 펄스 오라 + 궤적 파티클 |

## 6) 에셋 산출물 요약

| 카테고리 | 항목 수 |
|---------|--------|
| 프레임 스타일 | 5종 × 3테마 = 15 |
| 버튼 (상태별) | 6종 × 3사이즈 × 3테마 = 54 |
| 슬라이더/토글/탭/스크롤바 | 20종 |
| HUD 요소 | 12종 (바/아이콘/미니맵) |
| 인벤/상점/설정/채팅 구성 | 화면당 ~10 조각 × 4 = 40 |
| 대미지 팝업/쿨다운 | 10종 |
| **예상 UI 에셋 총 파일** | **~150 파일** |

### 파일 네이밍

```
ui_{component}_{variant}_{state}_{theme}.png

예시:
  ui_frame_panel_default_dark.png
  ui_button_medium_hover_aeterna.png
  ui_slider_handle_normal_season1.png
  ui_hud_hp_bar.png
  ui_hud_minimap_frame.png
```
