## 1) 비주얼 톤 — 다크 판타지 픽셀아트

### 1.1 핵심 키워드

| 요소 | 정의 |
|------|------|
| 장르 톤 | **다크 판타지** — 아름답지만 위험한 세계. 희망과 절망이 공존 |
| 아트 스타일 | **2D 픽셀아트** — 16/32비트 시대 정통 RPG 감성 + 현대적 색감 |
| 분위기 | **기억과 에테르** — 빛나는 기억 잔광, 안개, 고대 유적의 신비 |
| 차별화 | **한국 MMORPG 감성** — 장비 화려함, 이펙트 과잉, 등급 색상 체계 |

## 3) 라인 스타일 + 셰이딩 규칙

### 3.1 외곽선 (Outline)

| 항목 | 규격 |
|------|------|
| 기본 두께 | 2px 검정 (`#000000`) |
| 보스급 | 3px 검정 + 등급 오라 |
| UI 요소 | 1px 또는 선 없음 |
| 안티앨리어싱 | **사용 금지** — 클린 픽셀 유지 |
| 색상 외곽선 | 선택적: 머리카락/옷 경계에 해당 색상의 어두운 톤 사용 가능 |

### 3.2 셰이딩 규칙

| 규칙 | 설명 |
|------|------|
| 광원 방향 | **좌상단 45°** (모든 에셋 통일) |
| 셰이딩 단계 | 3단계 (하이라이트/미드톤/섀도) |
| 디더링 | 최소한 사용 — 그라데이션보다 색상 단계 선호 |
| 하이라이트 | 원색 + 20% 밝기 + 10% 채도 감소 |
| 미드톤 | 원색 기본 |
| 섀도 | 원색 - 30% 밝기 + 10% 색조 이동 (보라/파랑 쪽) |
| 반사광 | 반대편 하단에 약한 환경광 반사 (지역 팔레트 발광 색상) |

### 3.3 발광 이펙트 규칙

| 유형 | 처리 |
|------|------|
| 마법 발광 | 발광 중심 → 100% 밝기, 주변 2~3px → 50% 불투명도 블룸 |
| 아이템 오라 | 1px 발광 테두리 + 펄스 애니메이션 (2프레임 교대) |
| 환경 발광 | 발광균/크리스탈: 5px 반경 소프트 글로우 |
| 눈/특수 부위 | 2px 강한 발광 (캐릭터 눈, 마법 문양 등) |

## 4) UI 아트 톤

### 4.1 UI 프레임 스타일

| 요소 | 스타일 |
|------|--------|
| 기본 프레임 | 어두운 금속 (`#2A2A3A`) + 1px 밝은 테두리 (`#5A5A6A`) |
| 강조 프레임 | 금색 테두리 (`#FFD700`) + 에테르 발광 |
| 버튼 (기본) | 그라데이션 (`#3A3A4A` → `#2A2A3A`) + 1px 하이라이트 |
| 버튼 (호버) | 밝기 +15% + 미세 발광 |
| 버튼 (클릭) | 밝기 -10% + 1px 인셋 그림자 |
| 슬롯 (빈) | `#1A1A2A` + 1px 인셋 보더 |
| 슬롯 (아이템) | 등급 테두리 색상 + 아이콘 |
| 텍스트 입력 | `#0D0D1A` 배경 + 1px `#4A4A5A` 보더 |

### 4.2 폰트 가이드라인

| 용도 | 권장 폰트 | 크기 | 스타일 |
|------|----------|------|--------|
| 제목 | 픽셀 폰트 (커스텀) | 16px | Bold, 금색 |
| 본문 | 픽셀 폰트 | 8px | Regular, 밝은 회색 |
| 수치 | 모노스페이스 픽셀 | 10px | Bold, 등급 색상 |
| 대미지 팝업 | 두꺼운 픽셀 | 12~20px | Bold + 외곽선 + 바운스 |
| NPC 이름 | 픽셀 폰트 | 10px | Bold, 금색 |
| 시스템 메시지 | 픽셀 폰트 | 8px | Italic, 은색 |

### 4.3 아이콘 스타일 규칙

| 규칙 | 내용 |
|------|------|
| 크기 | 32×32 px (기본), 16×16 px (미니) |
| 배경 | 등급별 배경 그라데이션 또는 투명 |
| 외곽선 | 1px, 등급 색상 |
| 내부 | 아이템/스킬 실루엣이 16×16 축소에서도 인식 가능 |
| 스택 | 우하단 수량 텍스트 (흰 + 1px 검정 외곽) |

## 5) AI 생성 시 스타일 일관성 유지 규칙

### 5.1 필수 프롬프트 접두사 (모든 에셋)

```
2D pixel art, dark fantasy RPG, hand-painted style,
consistent art direction, [REGION] color palette,
top-left 45-degree lighting, 3-step shading,
2px black outline, no anti-aliasing, clean pixel edges,
transparent background (PNG), game asset ready
```

### 5.2 금지 프롬프트 키워드

```
Negative: 3D render, realistic, photorealistic, blurry,
watermark, text overlay, logo, low quality, deformed,
extra limbs, Western cartoon, chibi (unless specified),
noisy, grain, JPEG artifacts, soft edges, gradient shading,
anti-aliased edges, smooth lines
```

### 5.3 스타일 일관성 체크리스트

- [ ] 외곽선 2px 검정 유지?
- [ ] 광원 좌상단 45°?
- [ ] 3단계 셰이딩?
- [ ] 지역 팔레트 준수?
- [ ] 등급 시각화 규칙 준수?
- [ ] 16×16 축소 실루엣 테스트 통과?
- [ ] 투명 배경?
- [ ] 안티앨리어싱 없음?
