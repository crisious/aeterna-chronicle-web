## 1) 공통 타일 규격

### 1.1 기본 사양

| 항목 | 규격 |
|------|------|
| 기본 타일 크기 | 32×32 px |
| 고해상도 옵션 | 64×64 px |
| 타일셋 시트 크기 | 512×512 px (16×16 타일) 또는 1024×1024 px |
| 색상 모드 | RGBA 32bit |
| 뷰 방식 | 탑다운 (Top-down) |
| 조명 기준 | 좌상단 광원 (45°) |
| 이음새 | 심리스 타일링 필수 |

### 1.2 레이어 구성

| 레이어 | Z-Order | 용도 | 예시 |
|--------|---------|------|------|
| **바닥 (Ground)** | 0 | 기본 지형 | 돌바닥, 흙길, 잔디, 모래, 얼음 |
| **바닥 장식 (Ground Decor)** | 1 | 바닥 위 장식 | 균열, 이끼, 꽃, 핏자국, 발자국 |
| **오브젝트 (Objects)** | 2 | 배치 가능 오브젝트 | 나무, 바위, 건물, 가구, 상자 |
| **오브젝트 상단 (Object Top)** | 3 | 오브젝트 위 레이어 | 나뭇잎, 지붕, 현수막 |
| **충돌 (Collision)** | - | 이동 불가 영역 | 벽, 절벽, 물, 장애물 |
| **상호작용 (Interaction)** | - | 트리거 영역 | 문, NPC 대화, 아이템, 포탈 |
| **Parallax 원경** | -3 | 원경 배경 | 하늘, 산맥, 구름 |
| **Parallax 중경** | -2 | 중경 배경 | 원거리 건물, 숲 |
| **Parallax 근경** | -1 | 근경 배경 | 근거리 나무, 안개 |

### 1.3 타일 타입 분류

| 타입 | 설명 | 오토타일 |
|------|------|---------|
| 단일 타일 | 단독 배치 (풀, 꽃) | ❌ |
| 오토타일 (2×3) | 상/하/좌/우 연결 (벽, 울타리) | 47개 규칙 |
| 오토타일 (3×3) | 전 방향 연결 (길, 물) | 47개 규칙 |
| 애니메이션 타일 | 프레임 순환 (물, 용암, 불꽃) | 3~4프레임 |
| 대형 오브젝트 | 다수 타일 점유 (건물, 대형 나무) | ❌ |
