## 1) 공통 규격

| 항목 | 규격 |
|------|------|
| 기본 프레임 크기 | 64×64 px |
| 고해상도 옵션 | 128×128 px (초상화/보스용) |
| 색상 모드 | RGBA (32bit, 투명 배경) |
| 외곽선 | 2px 검정 (anti-aliasing 없음) |
| 피벗 포인트 | 하단 중앙 (x: 0.5, y: 0.9) |
| 시트 최대 크기 | 2048×2048 px |
| 프레임 간 패딩 | 2px |

### 1.1 8방향 정의

| 인덱스 | 방향 | 약어 | 미러링 |
|--------|------|------|--------|
| 0 | 아래 (정면) | D | - |
| 1 | 아래왼쪽 | DL | - |
| 2 | 왼쪽 | L | - |
| 3 | 위왼쪽 | UL | - |
| 4 | 위 (뒷면) | U | - |
| 5 | 위오른쪽 | UR | UL 수평 미러 |
| 6 | 오른쪽 | R | L 수평 미러 |
| 7 | 아래오른쪽 | DR | DL 수평 미러 |

> **미러링 규칙**: 5/6/7번 방향은 3/2/1번의 수평 반전으로 생성. 실제 제작 프레임은 5방향(0~4).

### 1.2 애니메이션 정의

| 모션 | 코드 | 프레임 수 | FPS | 루프 | 용도 |
|------|------|----------|-----|------|------|
| 아이들 | `idle` | 4 | 5 | ✅ | 대기 상태 |
| 이동 | `walk` | 6 | 10 | ✅ | 필드/전투 이동 |
| 근접 공격 | `attack_melee` | 6 | 12 | ❌ | 에테르 기사, 그림자 직조사 |
| 원거리 공격 | `attack_ranged` | 6 | 12 | ❌ | 그림자 직조사, 기억술사 |
| 스킬 시전 | `cast` | 5 | 10 | ❌ | 마법/스킬 사용 |
| 피격 | `hit` | 3 | 12 | ❌ | 피해 반응 |
| 사망 | `death` | 5 | 8 | ❌ | HP 0 시 재생 |

### 1.3 프레임 수 요약 (캐릭터 1체당)

```
방향 5개 (미러링 제외) × 모션 7종:
  idle:         5방향 × 4프레임 = 20
  walk:         5방향 × 6프레임 = 30
  attack_melee: 5방향 × 6프레임 = 30
  attack_ranged:5방향 × 6프레임 = 30
  cast:         5방향 × 5프레임 = 25
  hit:          5방향 × 3프레임 = 15
  death:        5방향 × 5프레임 = 25
  ─────────────────────────────
  합계: 175 프레임/캐릭터
```

### 6.2 시트 레이아웃 (1 모션 시트)

```
┌──────────────────────────────────────────┐
│ Row 0: Down     (프레임 0~N)             │
│ Row 1: DownLeft (프레임 0~N)             │
│ Row 2: Left     (프레임 0~N)             │
│ Row 3: UpLeft   (프레임 0~N)             │
│ Row 4: Up       (프레임 0~N)             │
└──────────────────────────────────────────┘
  * Right/DownRight/UpRight = 미러링으로 처리
  * 열 수 = 해당 모션의 프레임 수 (idle=4, walk=6 등)
```
