## 1) 공통 아이콘 규격

| 항목 | 규격 |
|------|------|
| 기본 크기 | 64×64 px |
| 소형 변형 | 32×32 px (HUD/미니맵용) |
| 색상 모드 | RGBA 32bit |
| 배경 | 어두운 그라데이션 (`#1A1A2E` → `#0D0D15`) |
| 외곽 | 등급별 테두리 (아래 참조) |
| 스타일 | 2D 픽셀아트, 다크 판타지, 아이콘그래픽 |
| 시트 구성 | 8×8 = 64 아이콘/시트 (512×512 px) |

### 1.1 등급별 테두리 색상

| 등급 | 테두리 색상 | Hex | 테두리 두께 | 추가 효과 |
|------|-----------|-----|-----------|----------|
| 일반 (Common) | 회색 | `#808080` | 1px | 없음 |
| 고급 (Uncommon) | 녹색 | `#2E7D32` | 1px | 없음 |
| 희귀 (Rare) | 파랑 | `#1565C0` | 2px | 없음 |
| 영웅 (Epic) | 보라 | `#7B1FA2` | 2px | 미세 외부 글로우 |
| 전설 (Legendary) | 주황 | `#E65100` | 2px | 외부 글로우 + 코너 장식 |
| 신화 (Mythic) | 빨강+금 | `#B71C1C`→`#FFD700` 그라디언트 | 3px | 강한 글로우 + 코너 장식 + 미세 파티클 |

### 1.2 아이콘 내부 구도 가이드

```
┌───────────────────┐
│  2px 테두리 영역    │
│ ┌───────────────┐ │
│ │               │ │
│ │  아이템 영역    │ │
│ │  (60×60 px)   │ │
│ │               │ │
│ └───────────────┘ │
│  등급 표시 (하단)   │
└───────────────────┘
```
