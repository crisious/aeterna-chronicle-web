# 에테르나 크로니클 — 프레스킷 (Press Kit)

> 최종 업데이트: 2026-03-12 | P12-16

---

## 🎮 게임 소개

**에테르나 크로니클 (Aeterna Chronicle)** 은 기억과 망각을 테마로 한 실시간 반자동 전투 MMORPG입니다.

> *"기억은 사라져도, 이야기는 남는다."*

대망각이 세계를 덮친 지 212년. 신들의 기억이 소멸하고, 에테르 결정만이 과거의 흔적을 품고 있습니다. 플레이어는 **에리언** — 잊혀진 기억을 되살릴 수 있는 마지막 기억술사로서 4개의 신성 기억 파편을 찾아 대륙을 횡단합니다.

---

## ✨ 핵심 특징

- **실시간 반자동 전투**: 스킬 타이밍과 전략적 포지셔닝이 핵심인 하이브리드 전투 시스템
- **5개 클래스**: 기억술사, 전사, 궁수, 마법사, 치유사 — 각각 고유 스킬 트리와 플레이스타일
- **9개 월드 지역**: 망각의 폐허에서 기억의 심연까지, 스토리와 연결된 광활한 세계
- **30개 던전**: 노말/하드/나이트메어 난이도 + 시즌별 특수 던전
- **길드 시스템**: 길드전, 길드 퀘스트, 공동 보스 레이드
- **PvP**: 1:1 결투, 3v3 아레나, 길드전
- **시즌 시스템**: 3개 시즌 + 시즌 패스, 주기적 콘텐츠 업데이트
- **커뮤니티**: SNS 공유, 스트리머 모드, UGC 갤러리, 레퍼럴 시스템
- **크로스 플랫폼**: PC 웹 브라우저 + UE5 데스크톱
