## 핵심 시스템

### 전투
- 실시간 반자동 전투 (오토 기본공격 + 수동 스킬 사용)
- 스킬 타이밍, 포지셔닝, 버프/디버프 관리가 전략 핵심
- 클래스별 고유 스킬 트리 + 궁극기 시스템

### 소셜
- 친구/파티/길드 시스템
- 실시간 채팅 + 우편 시스템
- SNS 공유 (Open Graph) + 스트리머 모드 (관전/오버레이)
- UGC 갤러리 + 레퍼럴 시스템
- 활동 피드 (친구/길드 활동 실시간 스트림)

### 경제
- 듀얼 통화 (골드 + 다이아)
- 경매장 (유저 간 거래)
- 시즌 패스 + 코스메틱 상점
- 가챠 없음 — 시즌패스 + 직접 구매 모델

### 기술
- TypeScript strict mode 전체 적용
- 번들 최적화 (tree-shaking, code-splitting, lazy load)
- DB 쿼리 최적화 (N+1 제거, Prisma select/include)
- Redis 캐시 전략 (핫 데이터 TTL 관리)
- k6 로드 테스트 (동시접속 1,000~10,000 시나리오)
- Grafana 모니터링 + SLO/SLI 정의
