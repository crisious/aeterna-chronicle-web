## 🎨 로고 사용 가이드

### 기본 로고

- **메인 로고**: 가로형 (텍스트 + 심볼) — 가장 많이 사용
- **심볼 로고**: 에테르 결정 아이콘 단독 — 앱 아이콘, 파비콘, 소셜 프로필
- **텍스트 로고**: "AETERNA CHRONICLE" 텍스트만 — 좁은 공간

### 사용 규칙

1. **최소 크기**: 가로형 120px 이상, 심볼 32px 이상
2. **여백**: 로고 주변 최소 여백 = 심볼 높이의 50%
3. **배경**: 밝은 배경에는 다크 로고, 어두운 배경에는 라이트 로고
4. **금지 사항**:
   - 로고 회전, 기울이기, 찌그러뜨리기
   - 로고 색상 임의 변경 (지정 색상만 사용)
   - 로고 위에 텍스트 오버레이
   - 저해상도 로고 사용 (래스터 최소 2x)
