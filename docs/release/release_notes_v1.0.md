# 에테르나 크로니클 v1.0 릴리즈 노트

> 버전: v1.0.0-rc.1 | 날짜: 2026-03-31

---

## 게임 소개

**에테르나 크로니클(Aeterna Chronicle)**은 기억과 망각을 테마로 한 16비트 픽셀아트 웹 RPG입니다.

대망각이 세계를 덮친 지 212년 후, 마지막 기억술사 **에리언**이 4개의 신성 기억 파편을 찾아 10개 지역을 횡단하는 여정을 그립니다. 크로노 트리거와 파이널 판타지 6의 황금기 JRPG에서 영감을 받아, 현대 웹 기술로 구현한 브라우저 RPG입니다.

"기억은 사라져도, 이야기는 남는다."

---

## 핵심 특징

- **1인 풀스택 개발**: 기획, 코드(301 TS 파일), 아트(1,582장), 오디오(137개) 전 과정을 단독 수행
- **브라우저 즉시 플레이**: 설치 없이 웹 브라우저에서 바로 플레이 (Phaser.js 기반)
- **FHD 지원**: 1920x1080 풀HD 해상도, Scale.FIT 반응형 대응
- **AI 에셋 파이프라인**: SDXL + Pixel Art XL LoRA(아트), MusicGen(BGM), TTS(보이스)
- **크로노 트리거 스타일**: 16비트 픽셀아트 통일, 노스탤지어와 현대성의 조화
- **4+1 멀티 엔딩**: 플레이어 선택에 따른 분기 — 212건 회귀 테스트 통과

## 기술 특징

### 클라이언트
- **Phaser.js** 기반 2D 게임 엔진, 13개 씬 구성
- FHD 1920x1080 해상도, `Scale.FIT` 반응형
- 16비트 픽셀아트 LINEAR 필터 렌더링

### 서버
- **Fastify** REST API 서버 (348 라우트)
- **Socket.io** 실시간 통신 (레이드, PvP)
- **Prisma** ORM + 110 모델 스키마

### 보안
- **bcrypt** 비밀번호 해싱
- **JWT 3중 키** 인증 (Access / Refresh / API Key)
- **2FA** AES-256-GCM 암호화 TOTP
- **P2W Guard** 과금 어뷰징 방지 시스템

### 인프라
- **Docker** 컨테이너화 + **Kubernetes** 오케스트레이션
- **GitHub Actions** CI/CD 파이프라인
- **Cloudflare Tunnel** 데모 배포

### AI 에셋 파이프라인
- **SDXL + Pixel Art XL LoRA**: 크로노 트리거 스타일 픽셀아트
- **MusicGen**: BGM 42곡 AI 생성
- **TTS**: NPC 보이스 20종
