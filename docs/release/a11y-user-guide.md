## 1. 이 문서의 쓰임

본 문서는 **플레이어** 와 **CBT 검수자** 가 한 화면에서 모든 접근성 기능을 익히고 점검할 수 있도록 작성되었사옵니다. 기술 명세는 `client/src/accessibility/README.md`에 있고, 본 문서는 **사용자 관점의 활용·확인 가이드** 입니다.

> 🪧 **표기 약속**
> - `[옵션 → 접근성]` 경로는 인게임 메뉴 위치를 뜻합니다.
> - `Ctrl+Alt+A` 같은 굵은 키 표기는 권장 기본값이며, [옵션 → 키 설정] 에서 재할당이 가능합니다.
> - 🟢 = AAA 자동 감사 통과 / 🟡 = AA 통과·AAA 진행 중 / 🔴 = 미해결 (`launch_checklist` 참조)

## 3. 시각 — 색맹 모드 / 고대비 / UI 스케일

### 3.1 색맹 시뮬 모드

> **목적**: 색약 플레이어가 자신의 시각 환경에서 게임이 식별 가능한지 즉시 확인.

| 모드 | 단축키 | 대응 색약 | 보강 표기 |
|------|--------|-----------|-----------|
| Protanopia (적색맹) | `Ctrl+Alt+1` | 1형 색각이상 | 빨강 → 텍스처 빗금 |
| Deuteranopia (녹색맹) | `Ctrl+Alt+2` | 2형 색각이상 | 초록 → 점 패턴 |
| Tritanopia (청색맹) | `Ctrl+Alt+3` | 3형 색각이상 | 파랑 → 격자 패턴 |
| Achromatopsia (전색맹) | `Ctrl+Alt+4` | 완전 색각이상 | 명도·아이콘 단독 식별 |

> ✦ **확인 포인트**: 전투 중 HP/MP/ATB 게이지가 **색상 외 2종(텍스트·패턴·아이콘) 이상** 으로 식별되는지.

### 3.2 고대비 모드

- WCAG AAA 기준 본문 대비 **7:1 / 큰 글자 4.5:1** 준수.
- 활성화 위치: `[옵션 → 접근성 → 고대비]` 또는 `Ctrl+Alt+H`.
- 다크 판타지 톤 유지를 위해 배경 `#0D0D1A` 고정, 텍스트만 명도 강화.

### 3.3 UI 스케일

- 100% / 125% / 150% / 175% / 200% 5단계.
- 200% 시에도 **레이아웃 깨짐 없이** 모든 인터랙션 요소가 화면 안에 들어옴 (자동 감사 케이스 `A11Y-UI-200`).

## 4. 입력 — 키보드 네비게이션 / 게임패드 / 키 리바인딩

### 4.1 키보드 풀-내비게이션

| 동작 | 키 |
|------|-----|
| 다음 포커스 | `Tab` |
| 이전 포커스 | `Shift+Tab` |
| 활성화 | `Enter` / `Space` |
| 취소·뒤로 | `Esc` |
| 컨텍스트 메뉴 | `Shift+F10` |
| 접근성 패널 토글 | `Ctrl+Alt+A` |

> ✦ **포커스 가시성**: 포커스 링은 `accent.ether (#89CFF0)` + 2px 외곽선으로 어떠한 배경에서도 3:1 이상 대비를 유지하옵니다.

### 4.2 게임패드 풀-내비 (🟡 진행 중)

- Xbox / DualSense / Switch Pro 자동 매핑.
- D-pad = 포커스 이동, A = 활성화, B = 취소, Start = 메뉴.
- **미해결 항목**: 인벤토리 격자 내 가로 이동 시 포커스 점프 — `BUG-A11Y-007` 추적.

### 4.3 키 리바인딩

- `[옵션 → 키 설정]` 에서 모든 입력을 재할당 가능.
- 충돌 시 메시지: `KEYBIND_CONFLICT` (→ `a11y-error-messages.md` 참조).

## 5. 청각 — 자막 / 시각화 큐 / 사운드 별칭

- **자막**: 모든 컷씬·NPC 대화에 ko/en/ja 자막 동봉. 폰트 크기 14~28pt, 배경 불투명도 0~80%.
- **사운드 → 시각 큐** (🟡 진행 중): 적 등장음·아이템 획득음을 화면 가장자리 플래시로 시각화.
- **사운드 별칭**: 단축키 음성 안내 (스크린 리더 우선)는 `AriaLabelMap.ts` SSOT 사용.

---

## 6. 인지 — 모션 감소 / 자동 진행 / 난이도 보조

- **모션 감소**: `prefers-reduced-motion` OS 설정 자동 추종 + 인게임 강제 토글 (`Ctrl+Alt+M`).
  - 화면 흔들림 ±2px 이하, 카메라 줌 0.5× 속도, 파티클 50% 감산.
- **자동 진행 속도**: 0.5× ~ 2.0× 슬라이더, 컷씬 중 일시정지 무제한.
- **난이도 보조**: '이야기 모드' 선택 시 전투 자동화 — 스토리 완주 보장.

## 7. 스크린 리더 호환성

| 리더 | OS | 상태 | 검증 시나리오 |
|------|-----|------|---------------|
| NVDA 2024.x | Windows | 🟡 | 메인 메뉴 → 새 게임 → 첫 전투 진입 (`A11Y-SR-001`) |
| JAWS 2024 | Windows | 🟡 | 인벤토리 정렬·필터 (`A11Y-SR-002`) |
| VoiceOver | macOS Sonoma | 🟡 | 옵션 메뉴 전수 낭독 (`A11Y-SR-003`) |
| Narrator | Windows 11 | ⚪ | 미테스트 — 다음 스프린트 |

- 모든 인터랙션 요소는 `aria-label` / `role` / `aria-describedby` SSOT (`AriaLabelMap.ts`) 를 통해 라벨링되옵니다.
- 동적 영역(자막·전투 로그)은 `aria-live="polite"`, 긴급 경고(저장 실패)는 `aria-live="assertive"`.

## 8. 문제 해결 (FAQ)

| 증상 | 원인 후보 | 해결 | 추적 |
|------|-----------|------|------|
| 색맹 모드가 켜지지 않음 | 옵션 미적용 / 단축키 충돌 | `Ctrl+Alt+1~4` 재시도 → [옵션 → 접근성 → 색맹 시뮬] 직접 토글 | — |
| 포커스 링이 안 보임 | 고대비 OFF + 어두운 배경 | `Ctrl+Alt+H` 로 고대비 ON, 또는 [옵션 → 접근성 → 포커스 두께] 3px 로 상향 | — |
| 자막이 가려짐 | UI 스케일 200% + 자막 큰 글자 | [옵션 → 자막 → 위치] '상단' 이동 / 배경 불투명도 60% | — |
| 게임패드 D-pad 가 인벤토리에서 점프 | 격자 포커스 매니저 회귀 | 일시 회피: 마우스/키보드 병행, 다음 패치(rc.4) 수정 예정 | `BUG-A11Y-007` |
| NVDA 가 옵션 메뉴를 낭독하지 않음 | `aria-label` 누락 / 동적 영역 미감지 | `tests/reports/a11y/contract.json` 확인 후 [도움말 → 버그 리포트] | `BUG-A11Y-011` |
| 패턴 오버레이가 게이지에서 깨짐 | UI 200% + 캔버스 스케일 충돌 | 패턴 오버레이 OFF → ON 토글 1회로 재계산 | `BUG-A11Y-013` |
| `prefers-reduced-motion` 가 무시됨 | OS 추종 OFF 상태 | [옵션 → 접근성 → 모션 감소] 'OS 설정 따르기' 체크 | — |
| 컷씬 자막이 음성보다 늦음 | 자동 진행 속도 0.5× 적용 중 | [옵션 → 접근성 → 자동 진행] 1.0× 로 복귀 | — |
| Tab 으로 닫힌 모달을 빠져나갈 수 없음 | 키보드 트랩 (회귀) | `Esc` 두 번 → 미해결 시 BUG 리포트 즉시 송출 | `BUG-A11Y-016` |

> 💌 버그 리포트 송출 시 자동으로 `tests/reports/a11y/summary.json` 의 `runId` 가 첨부되어 회귀 추적이 가능하옵니다.

## 9. VPAT 2.4 요약 (외부 공개용)

> 정식 VPAT 2.4 문서는 `docs/legal/vpat-2.4.md` (작성 예정)에서 발간하며, 본 절은 사용자에게 보여줄 핵심 요약입니다.

| WCAG 2.1 원칙 | 준수 수준 | 비고 |
|---------------|-----------|------|
| 1. 인식 가능 (Perceivable) | AAA | 색상 단독 의존 0건 (자동 감사) |
| 2. 운용 가능 (Operable) | AAA | 키보드 풀-내비 100% |
| 3. 이해 가능 (Understandable) | AA+ | 일관된 내비·예측 가능한 변경 |
| 4. 견고함 (Robust) | AA | 스크린 리더 검증 진행 중 |

> 갱신 트리거: `tests/reports/a11y/summary.json` 의 `wcagLevel` 필드 변경 시 본 표를 자동 갱신합니다 (`scripts/a11y/update-vpat.ts`, Build 단계 인계).
