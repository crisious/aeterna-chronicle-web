## 3. 오프닝 시네마틱 (0~3분)

### 3.1 구성

| 컷 | 시간 | 비주얼 | 카피 (1줄) |
|---|-----|------|---------|
| 1 | 0:00~0:20 | 어둠 속 신상이 무너지는 잔영 | "신들의 기억이 사라진 지, 212년." |

| 4 | 1:30~2:30 | 4개 신성 기억 파편 깜빡임 | "네 개의 기억을, 다시 빛으로." |

### 3.2 스킵·재시청

- **스킵**: `Esc` 또는 화면 우상단 [Skip ▶▶] 버튼 — 1회 클릭으로 즉시 비트 ②로

- **자막**: ko/en 동시, WCAG AAA 대비비 (DESIGN.md §7 참조)

### 3.3 접근성

- 자막 16/20/24px 3단계
- 화면 흔들림 효과 [설정 → 접근성 → 모션 감소]에서 OFF
- 컷 1·3 어둠 컷 — 광민감 경고 0:00에 3초 노출


> 코칭 카피는 `tutorial-onboarding-error-messages.md §1·§2` SSOT 참조.

### 4.2 비트 ③ 첫 전투 — 슬라임 (7~12분)

#### 한 화면 1개념 — ATB 게이지

```
┌──────────────────────────────────────┐
│  에리언            HP 50/50  MP 10/10 │
│  ATB ████████████░░░░░░  (75%)        │  ← coach.battle 화살표 깜빡임
│                                      │
│           슬라임          HP 8/8      │
│           ATB ░░░░░░░░░░  (5%)        │
│                                      │
│  [공격]  [스킬]  [아이템]  [도망]     │  ← 「공격」만 활성, 나머지 비활성
└──────────────────────────────────────┘
```

- ATB 게이지 100% 도달 시 「공격」 버튼 깜빡임 (≤ 800ms 응답)
- 첫 처치 시 **+1 EXP, +5 골드, +1 회복약** 자동 지급
- 실패 시 자동 부활 (튜토리얼 한정) + 재시도 카피 노출

> 자세한 ATB 동작은 `atb-battle-system-guide.md` 참조.

## 9. FAQ 7건

**Q1. 회차 플레이(NG+)에서도 30분이 강제되나?**
A. 아니옵니다. 회차에서는 메인 메뉴 → [튜토리얼 스킵 ▶▶] 단일 클릭으로 비트 ⑦로 점프하옵니다. 단, 첫 회차에서는 스킵 불가하옵니다 (학습 보장 정책).

**Q2. 스킬을 안 쓰고 기본 공격만으로 보스를 잡으면?**
A. P2(HP 50%) 진입 시 「아이템」 버튼이 깜빡이옵고, 30초 안에 회복약을 쓰지 않으면 코칭 카피 `coach.skill.warn.idle`가 노출되옵니다. 5종 모두 만지지 않으면 보스 처치가 불가능하도록 설계됩니다.

**Q3. 자막을 끄면 어떻게 되나?**
A. WCAG AAA 약속에 따라 자막은 기본 ON이옵니다. 끄려면 [설정 → 접근성 → 자막 OFF] — 단, 시작 시 광민감 경고는 자막 끔과 무관하게 항상 노출되옵니다.

**Q4. 튜토리얼 중 일시정지·종료가 가능한가?**
A. 일시정지(Esc)는 비트 ②~⑥ 모두 가능하옵니다. 종료는 강제 세이브 후만 가능 — 비트 ⑤ 통과 전에는 [메인 메뉴로] 버튼이 비활성화되옵니다.

**Q5. 색약·시각장애 사용자는 ATB 게이지를 어떻게 보나?**
A. ATB는 ① 색(금→청 그라디언트), ② 형태(채움 패턴 — 빈 / 점선 / 사선 / 가득), ③ 음(100% 도달 시 1초 'ding') 3중으로 신호하옵니다. `a11y-ingame-copy.md §3.2` 참조.

**Q6. 다국어 지원 범위는?**
A. v1.0은 ko/en 2개 국어 동시. 일본어·중국어(간체)는 Phase 2(Unity 빌드)에서 추가 예정. 카피 SSOT 키 규약은 `coach.<gate>.<state>.<reason>`이옵니다.

**Q7. 튜토리얼이 30분을 초과하면?**
A. 비트별 시간을 `.ac/tutorial-perf.json`에 기록하옵고, 누적 5회 평균이 32분을 넘으면 정경패 Reviewer가 비트 ⑥ 보스 HP를 5% 하향 조정하옵니다. 자동 조정 룰은 `tutorial-onboarding-error-messages.md §6` 참조.
