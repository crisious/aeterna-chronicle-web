# 에테르나 크로니클 — 청소년 보호 정책

> 시행일: 2026년 3월 12일  
> 버전: 1.0  
> 근거 법령: 게임산업진흥에 관한 법률, 청소년 보호법, 아동·청소년의 성보호에 관한 법률

---

## 1. 목적

본 정책은 에테르나 크로니클 서비스 이용 시 청소년의 건전한 게임 이용 환경을 조성하고, 관련 법령을 준수하기 위한 보호 조치를 규정합니다.

## 2. 게임물 등급

| 항목 | 내용 |
|------|------|
| 게임명 | 에테르나 크로니클 (Aeterna Chronicle) |
| 등급 | **12세 이용가** (예정) |
| 플랫폼 | PC 웹 브라우저, Unity WebGL |
| 심의 | 게임물관리위원회 심의 예정 |

## 3. 연령 확인

1. 회원가입 시 **생년월일 입력**을 통해 연령을 확인합니다.
2. 만 14세 미만 이용자는 **법정대리인의 동의**를 받아야 합니다.
3. 연령 확인은 서비스 가입, 결제, 채팅 기능 등에 적용됩니다.

## 4. 이용 시간 제한 (셧다운제 대응)

### 4.1 자율 규제 (선택적 이용 시간 관리)

| 대상 | 기본 설정 | 변경 가능 |
|------|----------|----------|
| 만 18세 미만 | 1일 4시간 알림 | 법정대리인 설정 |
| 만 18세 이상 | 제한 없음 | 자발적 설정 가능 |

### 4.2 피로도 알림

- **2시간 연속 플레이** 시 휴식 권고 알림
- **4시간 연속 플레이** 시 강제 휴식 권고 팝업
- 알림은 해제 가능하나, 미성년자의 경우 법정대리인만 해제 가능

## 5. 결제 한도 제한

### 만 18세 미만 이용자

| 연령대 | 월간 결제 한도 | 비고 |
|--------|--------------|------|
| 만 14세 미만 | 50,000원 | 법정대리인 동의 필수 |
| 만 14세 ~ 만 16세 미만 | 70,000원 | 법정대리인 동의 필수 |
| 만 16세 ~ 만 18세 미만 | 100,000원 | 법정대리인 동의 권장 |

- 한도 초과 시 결제가 차단됩니다.
- 법정대리인은 인게임 설정에서 한도를 하향 조정할 수 있습니다.

## 6. 유해 콘텐츠 차단

### 6.1 채팅 필터
- 욕설, 비속어, 성적 표현 자동 필터링
- 개인정보(전화번호, 주소 등) 입력 차단
- 외부 링크 공유 제한 (화이트리스트 방식)

### 6.2 신고 시스템
- 부적절한 채팅/행위 즉시 신고 기능
- 신고 접수 후 24시간 이내 검토
- 반복 위반자 자동 채팅 제한

### 6.3 콘텐츠 등급 표시
- 게임 내 콘텐츠에 등급 표시 (전체이용가/12세이용가)
- 등급 초과 이벤트 콘텐츠는 연령 확인 후 접근 허용

## 7. 개인정보 보호 (청소년)

1. 만 14세 미만 아동의 개인정보 수집 시 법정대리인 동의 획득
2. 수집 항목 최소화 (이메일, 닉네임, 생년월일)
3. 법정대리인의 열람/수정/삭제 요청권 보장
4. 상세 사항은 개인정보처리방침 참조

## 8. 법정대리인 권한

법정대리인은 다음 기능을 이용할 수 있습니다:

1. **이용 시간 설정**: 일일/주간 플레이 시간 제한
2. **결제 한도 조정**: 월간 결제 한도 하향 조정
3. **채팅 제한**: 채팅 기능 비활성화
4. **계정 삭제**: 자녀의 계정 삭제 요청
5. **활동 보고서**: 자녀의 게임 활동 요약 (주간)

설정 방법: 인게임 설정 > 보호자 관리 (법정대리인 인증 필요)

## 9. 과몰입 예방

1. **게임 시작 시 주의 문구**: "게임은 적당히 즐기세요."
2. **정기 알림**: 장시간 플레이 시 휴식 권고
3. **과몰입 자가진단**: 인게임 설정에서 자가진단 테스트 제공
4. **상담 안내**: 한국콘텐츠진흥원 게임과몰입상담센터 (1577-0075)
