# 에테르나 크로니클 — 오픈 베타 가이드

## 참여 방법

1. **초대 코드 받기** — 공식 Discord/SNS 채널에서 코드를 배포합니다
2. **게임 접속** — 로그인 화면에서 초대 코드 입력
3. **플레이** — 최대 5,000명까지 동시 참여 가능

### 초대 코드 형식
```
ETHB-XXXXXXXX
```
- 코드는 1회만 사용 가능
- 발급 후 30일 내 사용 필요
- 대소문자 구분 없음

## 피드백 채널

### 인게임 피드백
- 게임 내 `F8` 키 → 피드백 폼
- 버그/기능/밸런스/UX 분류 선택
- 스크린샷 자동 첨부 지원

### 외부 채널
- **Discord**: #beta-feedback 채널
- **이메일**: beta@etherna.io

## 피드백 작성 팁

### 버그 리포트
```
제목: [영역] 간단한 설명
설명:
  - 재현 방법 (step-by-step)
  - 기대 동작
  - 실제 동작
  - 스크린샷/영상 (가능하면)
```

### 밸런스 피드백
```
제목: [직업/스킬/아이템] 밸런스 이슈
설명:
  - 현재 상태
  - 문제점
  - 개선 제안
```

## API (개발자 참고)

| 엔드포인트 | 설명 |
|------------|------|
| POST /beta/validate-code | 초대 코드 검증 + 사용 |
| POST /beta/feedback | 피드백 접수 |
| GET /admin/beta/codes | 초대 코드 관리 |
| POST /admin/beta/generate | 초대 코드 일괄 생성 |
| GET /admin/beta/feedback | 피드백 조회 |
| PATCH /admin/beta/feedback/:id | 피드백 상태 변경 |
| GET /admin/beta/stats | 베타 통계 |
