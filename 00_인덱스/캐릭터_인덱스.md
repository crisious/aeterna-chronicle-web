## 엔딩별 생존/이탈 영향표

> 기준 문서: [[멀티엔딩_플래그_설계]]
> 대상 동료: 세라핀 / 벤자민 / 이그나 / 우르그롬 / 레이나 / 마에스트로 크리오

### 1) 엔딩별 동료 상태 영향 (요약)

| 엔딩 | 동료 생존 요구 | 이탈/손실 허용 | 에필로그 영향 |
|---|---|---|---|
| **A 기억의 수호자** | **전원 생존 필수** (`party_alive == 63`) | 사실상 불가 (1명이라도 이탈 시 A 불가) | 전원 참여형 회복 에필로그 |
| **B 마지막 증인의 선택** | 최소 파편 조건 중심 (`fragment_count >= 3`) | 허용 (생존 인원에 따라 분량 변화) | 생존 동료 수에 따라 기념비/독백 톤 변화 |
| **C 망각의 선택** | 동료 생존은 필수 조건 아님 | 허용 (분기 선택 누적에 따라 이탈 발생 가능) | 레이나 생존 시 후일담 강화 |
| **D 신들의 귀환** | 핵심은 유물/유적 조건 | 허용 (단, 동료 생존도 내러티브 풍부도에 영향) | 신화 축 중심 에필로그, 동료 생존 시 서브 장면 확장 |

### 2) 동료별 이탈 트리거와 엔딩 영향

| 동료 | 이탈 트리거 (핵심) | 엔딩 A 영향 | 엔딩 B 영향 | 엔딩 C 영향 | 엔딩 D 영향 |
|---|---|---|---|---|---|
| [[캐릭터/세라핀|세라핀]] | Ch2 2-A C(강제 채취) 후 신뢰도 0 | **A 즉시 불가** | 생존 시 감정선 강화 | 이탈해도 C 진행 가능 | 생존 시 실반헤임 후일담 강화 |
| [[캐릭터/벤자민_크로스|벤자민]] | Ch4 4-A B 선택 + 화해 미완료 | **A 불가** | 생존 여부에 따라 에필로그 톤 변화 | C 진행에는 직접 제약 적음 | 생존 시 제국 후일담 보강 |
| [[캐릭터/이그나|이그나]] | Ch3 신뢰도 20 미만 (Ch4 동행 불가) | **A 불가** (전원 생존 실패) | 동행 불가 시 분량 축소 | C 진행 가능 | 생존 시 사막/이프리타 후일담 확장 |
| [[캐릭터/우르그롬|우르그롬]] | Ch4 북방 기억석 사원 미방문 (합류 불가) | **A 불가** | 합류 여부 따라 북방 에필로그 변화 | C 진행 가능 | 생존/합류 시 히든 감정선 강화 |
| [[캐릭터/레이나|레이나]] | Ch4 4-C C 선택 시 이탈 | **A 불가** | 이탈 시 교단 내부 후일담 축소 | **생존 시 C 에필로그 핵심 장면 강화** | 생존 시 교단/신화 연결 보강 |
| [[캐릭터/마에스트로_크리오|크리오]] | Ch4 제국 침투 시 배신 선택 | **A 불가** | 이탈 시 진실 전달 후일담 축소 | C 진행 가능 | 생존 시 에레보스 재건 서사 강화 |

### 3) 운영용 판정 메모

- 엔딩 A는 `party_alive == 63` 조건 때문에 **동료 이탈 1회 = 탈락**.
- 엔딩 B/C/D는 동료 생존이 절대 조건은 아니지만, **에필로그 밀도와 감정선 품질**에 직접 영향.
- 엔딩 C는 레이나 생존 여부가 후일담 상징성에 큰 차이를 만든다.

## 보완 메모

- `확인 필요` 표시는 각 개별 캐릭터 문서의 최신 설정 확정 후 갱신.
- 위 등장표/영향표는 현재 챕터 문서 기준의 운영용 버전이며, 스크립트 잠금 전 최종 검수 필요.
- 시나리오 원문 수정 시 `시나리오_인덱스`와 함께 동기화 권장.
