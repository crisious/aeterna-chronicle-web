10. 접근성
  • ✅ prefers-reduced-motion: pulse/float/shake + 칩 잔여턴바 애니메이션 정지
  • ✅ prefers-contrast: more: 대미지 8방향 2px 아웃라인 · 상태칩 테두리 강화(+1px)
  • ✅ 색약 3중 잉여: 대미지 형태 마커(★▲▼⊘✚✗) + 상태이상 글리프(☠✦❄ 등) + 카테고리 테두리 패턴(점선/실선/이중선)
  • ✅ WCAG 보정: blind #444→#8A8A9E · curse #606→#9B4DC4 — 구 대비 FAIL → v1 AA 달성
  • ✅ 대미지 8방향 픽셀 아웃라인 → 밝은 VFX 위에서도 숫자 분리
  • ✅ 반응형: 1920 / 1440 / 1280 / 1024 px 4단계 · 모바일 칩 18→16px
