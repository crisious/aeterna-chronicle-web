# 에테르나 크로니클 — 자막 + 청각 접근성 설계

> **작성일:** 2026-03-13 | **버전:** v1.0 | **분류:** 클라이언트 — 접근성/청각  
> **티켓:** P17-18 (#326) | **참조:** accessibility_design.md §3

---

## 1. 개요

청각 장애 유저 및 음소거 환경 유저를 위한 자막 시스템, 효과음 시각화, 볼륨 세분화를 설계한다.

---

## 2. 자막 옵션 정의

### 2.1 자막 크기

| 단계 | 포인트 | px 환산 | 설명 |
|------|--------|---------|------|
| 1단계 (소) | 12pt | 16px | 최소 — 화면 점유 최소화 |
| 2단계 (중) | 16pt | 21px | 기본값 |
| 3단계 (대) | 20pt | 27px | 가독성 우선 |
| 4단계 (특대) | 28pt | 37px | 시력 저하 유저용 |
| 5단계 (최대) | 36pt | 48px | 최대 — 고령/저시력 |

### 2.2 자막 색상

| 색상 | Hex | 용도 |
|------|-----|------|
| 흰색 (기본) | `#FFFFFF` | 범용 |
| 노랑 | `#FFD700` | 고대비 배경 대응 |
| 청록 | `#00CED1` | 색약 친화 |
| 초록 | `#00FF7F` | 색약 친화 |
| 분홍 | `#FF69B4` | 화자 구분용 |
| 하늘 | `#87CEEB` | 화자 구분용 |
| 주황 | `#FFA500` | 효과음 자막용 |
| 빨강 | `#FF4444` | 경고/위험 자막용 |

### 2.3 자막 배경 불투명도

| 설정 | 불투명도 | CSS |
|------|---------|-----|
| 없음 | 0% | `background: transparent` |
| 반투명 (기본) | 60% | `background: rgba(0,0,0,0.6)` |
| 진한 반투명 | 80% | `background: rgba(0,0,0,0.8)` |
| 불투명 | 100% | `background: rgba(0,0,0,1.0)` |

### 2.4 자막 위치

| 위치 | 화면 좌표 | 비고 |
|------|-----------|------|
| 하단 중앙 (기본) | bottom: 10%, center | 일반적 |
| 상단 중앙 | top: 5%, center | 하단 UI 간섭 방지 |
| 사용자 지정 | 드래그 이동 | 임의 위치 고정 |

### 2.5 자막 표시 규칙

| 항목 | 설정 |
|------|------|
| 대사 자막 | 기본 ON, NPC 이름 + 대사 텍스트 |
| 화자 이름 색상 | 화자별 자동 배정 (8색 순환) |
| 효과음 자막 | 기본 OFF, 토글 가능 |
| 자막 큐 | 최대 3줄 동시 표시, FIFO |
| 표시 시간 | 기본 3초, 범위 1~10초 |
| 페이드 | 0.3초 페이드인/아웃 |
| 자막 로그 | 최근 50줄 스크롤 가능 |

## 3. 효과음 시각화

### 3.1 피격 방향 인디케이터

| 요소 | 사양 |
|------|------|
| 형태 | 화면 가장자리 반원형 빨간 번짐 |
| 방향 | 8방향 (N/NE/E/SE/S/SW/W/NW) |
| 색상 | `#FF0000` → `transparent` 그라디언트 |
| 지속 | 0.5초 페이드아웃 |
| 중첩 | 동시 다방향 피격 시 각 방향 독립 표시 |
| 강도 | 데미지 비율(최대HP 대비)에 비례해 불투명도 조절 |

### 3.2 스킬 시전 진동 패턴 (게임패드)

| 스킬 타입 | 진동 패턴 | 강도 | 지속 |
|-----------|----------|------|------|
| 근접 타격 | 단발 펄스 | 0.8 | 100ms |
| 원거리 발사 | 이중 펄스 | 0.5 | 150ms |
| 지속 채널링 | 저주파 연속 | 0.3 | 시전 시간 |
| 폭발/범위기 | 강한 단발 → 약한 여진 | 1.0→0.4 | 300ms |
| 힐/버프 | 부드러운 웨이브 | 0.2 | 500ms |
| 피격 | 방향별 비대칭 진동 (좌/우) | 0.6 | 200ms |

### 3.3 환경 사운드 자막

효과음 자막 ON 시 환경음을 대괄호로 표기:

| 카테고리 | 예시 | 표시 색상 |
|----------|------|-----------|
| 전투 | `[검의 충돌]` `[폭발]` `[화살 빗발]` | 주황 `#FFA500` |
| 환경 | `[바람소리]` `[물 흐르는 소리]` `[비]` | 하늘 `#87CEEB` |
| 생물 | `[발소리 — 왼쪽]` `[몬스터 울음]` `[새소리]` | 초록 `#00FF7F` |
| 시스템 | `[퀘스트 완료 효과음]` `[레벨업]` `[알림]` | 노랑 `#FFD700` |
| 위험 | `[함정 작동]` `[보스 포효]` `[균열 소리]` | 빨강 `#FF4444` |

### 3.4 방향 정보 포함

공간 오디오 정보를 자막에 방향 태그로 포함:

```
[왼쪽에서 발소리]
[오른쪽 뒤에서 검 소리]
[위에서 돌 떨어지는 소리]
[멀리서 폭발]
```

## 4. 볼륨 구조도

```
🔊 오디오 설정
├── 마스터 볼륨    ████████░░ 80%    [음소거 □]
├── BGM           ██████░░░░ 60%    [음소거 □]
├── SFX (효과음)   ████████░░ 80%    [음소거 □]
├── 보이스 (대사)   ██████████ 100%   [음소거 □]
├── 환경음         █████░░░░░ 50%    [음소거 □]
├── UI 효과음      ██████░░░░ 60%    [음소거 □]
│
├── 🔧 고급
│   ├── 모노 오디오  [OFF ○ | ● ON]
│   ├── 다이나믹 레인지 압축 [OFF ● | ○ ON]  (작은 소리 증폭)
│   └── 음성 강조   [OFF ○ | ● ON]  (음성 볼륨 자동 부스트)
│
└── 🎧 출력
    ├── 출력 장치    [시스템 기본 ▼]
    └── 공간 오디오  [OFF ○ | ● 스테레오 | ○ 서라운드]
```

### 4.1 채널별 음소거 독립 동작

| 채널 | 음소거 시 동작 |
|------|---------------|
| 마스터 | 전체 음소거 (모든 하위 채널) |
| BGM | BGM만 정지, SFX/보이스 유지 |
| SFX | 효과음 정지, 시각 인디케이터 자동 활성화 |
| 보이스 | 대사 음성 정지, 자막 자동 ON |
| 환경음 | 환경 사운드 정지 |
| UI | UI 클릭/호버 사운드 정지 |

### 4.2 다이나믹 레인지 압축

| 설정 | 동작 | 대상 유저 |
|------|------|-----------|
| OFF | 원본 오디오 그대로 | 일반 |
| ON | 작은 소리 증폭 + 큰 소리 억제 (리미터) | 난청/야간 플레이 |

## 5. 접근성 연동

| 접근성 기능 | 청각 접근성 연동 |
|------------|----------------|
| 색약 모드 | 피격 인디케이터 색상을 색약 팔레트로 변환 |
| 고대비 모드 | 자막 배경 자동 불투명 전환 |
| 모션 감소 | 피격 인디케이터 페이드 → 즉시 표시/즉시 소멸 |
| 화면 낭독기 | 자막 텍스트를 aria-live 리전으로 출력 |
