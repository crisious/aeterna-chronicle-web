# 에테르나 크로니클 — 모션 감소 모드 설계

> **작성일:** 2026-03-13 | **버전:** v1.0 | **분류:** 클라이언트 — 접근성/모션  
> **티켓:** P17-19 (#327) | **참조:** accessibility_design.md §2.4

---

## 1. 개요

모션 민감(전정기관 장애, 편두통, 광과민성 등) 유저를 위한 시각 효과 감소 옵션을 설계한다. OS `prefers-reduced-motion` 미디어 쿼리와 자동 연동한다.

---

## 2. 감소 대상 목록

### 2.1 카메라

| 대상 | 기본 동작 | 모션 감소 시 | 파라미터 |
|------|-----------|-------------|----------|
| 카메라 흔들림 (히트쉐이크) | 타격/폭발 시 0.5~2.0 강도 흔들림 | 0으로 고정 (완전 비활성) | `cameraShake: 0~100` |
| 카메라 줌 전환 | 0.3초 이징 애니메이션 | 0.8초로 감속 + 선형 보간 | `zoomDuration: 300~1000ms` |
| 급격한 카메라 이동 | 순간 이동 (텔레포트/존 전환) | 0.5초 페이드 블랙 전환 | `cameraSnapMode: instant/fade` |
| 보스 연출 카메라 | 줌인/패닝/회전 시퀀스 | 고정 앵글 + 자막으로 대체 | `bossCamera: cinematic/static` |
| 사망 카메라 | 슬로모션 + 줌아웃 | 즉시 사망 화면 전환 | `deathCamera: slow/instant` |

### 2.2 파티클

| 레벨 | 밀도 | 적용 범위 | 성능 영향 |
|------|------|-----------|-----------|
| 4 (100%) | 원본 그대로 | — | 기준 |
| 3 (50%) | 파티클 수 반감, 수명 유지 | 스킬 이펙트, 환경 파티클 | ~5% FPS 향상 |
| 2 (25%) | 파티클 수 1/4, 크기 보정(×1.3) | 스킬+환경+날씨 | ~10% FPS 향상 |
| 1 (Off) | 파티클 전면 비활성, 단색 오버레이 대체 | 전체 | ~15% FPS 향상 |

### 2.3 화면 번쩍임

| 대상 | 기본 | 모션 감소 시 |
|------|------|-------------|
| 스킬 시전 플래시 | 0.1초 백색 플래시 | 제거 (밝기 변화 없음) |
| 크리티컬 히트 플래시 | 0.05초 황색 플래시 | 제거 |
| 레벨업 이펙트 | 화면 전체 빛줄기 | 캐릭터 주변 하이라이트만 |
| 보스 페이즈 전환 | 화면 전체 플래시 | 텍스트 오버레이 대체 |
| 번개/폭발 환경 | 화면 밝기 급변 | 부드러운 밝기 전환 (0.5초) |

**WCAG 2.3.1 준수:** 모션 감소 ON 시 3회/초 이상 번쩍임은 0으로 보장.

### 2.4 UI 전환 애니메이션

| 대상 | 기본 | 모션 감소 시 |
|------|------|-------------|
| 화면 전환 | 0.3초 페이드 + 슬라이드 | 즉시 전환 (0ms) |
| 메뉴 열기/닫기 | 0.2초 스케일 + 페이드 | 즉시 표시/숨김 |
| 스크롤 | 부드러운 관성 스크롤 | 즉시 점프 |
| 알림 팝업 | 0.3초 슬라이드인 | 즉시 표시 |
| 로딩 스피너 | 회전 애니메이션 | 정적 프로그레스 바 |
| 인벤토리 아이템 호버 | 0.1초 확대 | 테두리 강조만 |
| 탭 전환 | 0.2초 슬라이드 | 즉시 전환 |

## 3. 4단계 파라미터 테이블

### 3.1 통합 설정 모드

| 설정 | 설명 | 기본값 |
|------|------|--------|
| **모션 감소 OFF** | 모든 효과 원본 | ✅ 기본 |
| **모션 감소 MILD** | 카메라 흔들림 50%, 파티클 50%, 번쩍임 제거 | — |
| **모션 감소 MODERATE** | 카메라 흔들림 0%, 파티클 25%, 번쩍임 제거, UI 전환 축소 | — |
| **모션 감소 FULL** | 위 전부 + 파티클 OFF + UI 즉시 전환 + 보스 카메라 정적 | — |

### 3.2 상세 파라미터 매트릭스

| 파라미터 | OFF | MILD | MODERATE | FULL |
|----------|-----|------|----------|------|
| `cameraShake` | 100 | 50 | 0 | 0 |
| `zoomDuration` | 300ms | 500ms | 800ms | 1000ms |
| `cameraSnapMode` | instant | fade(0.3s) | fade(0.5s) | fade(0.5s) |
| `bossCamera` | cinematic | cinematic | static | static |
| `deathCamera` | slow | slow | instant | instant |
| `particleDensity` | 100% | 50% | 25% | 0% |
| `screenFlash` | ON | OFF | OFF | OFF |
| `uiTransition` | 300ms | 200ms | 100ms | 0ms |
| `scrollSmooth` | ON | ON | OFF | OFF |
| `hoverAnimation` | ON | ON | OFF | OFF |

## 4. OS 자동 감지 연동

### 4.1 `prefers-reduced-motion` 반영

```typescript
// 웹 클라이언트 — 초기화 시 OS 설정 감지
const mediaQuery = window.matchMedia('(prefers-reduced-motion: reduce)');

function applyOSMotionPreference(matches: boolean): void {
  if (matches && !userExplicitlySetMotion) {
    // OS에서 모션 감소 활성화 → 게임 MODERATE로 자동 설정
    setMotionReduction('MODERATE');
    showNotification('OS 설정에 따라 모션 감소 모드가 활성화되었습니다.');
  }
}

// 실시간 변경 감지
mediaQuery.addEventListener('change', (e) => applyOSMotionPreference(e.matches));

// 초기 로드 시 적용
applyOSMotionPreference(mediaQuery.matches);
```

### 4.2 자동 감지 정책

| 상황 | 동작 |
|------|------|
| OS 모션 감소 ON + 게임 설정 없음 | MODERATE 자동 적용 + 안내 팝업 |
| OS 모션 감소 ON + 게임에서 수동 변경 | 게임 설정 우선 (OS 무시) |
| OS 모션 감소 OFF | 게임 설정 그대로 유지 |
| OS 설정 실시간 변경 | 게임 수동 변경 이력 없으면 실시간 반영 |

## 5. 접근성 연동

| 접근성 기능 | 모션 감소 연동 |
|------------|---------------|
| 색약 모드 | 파티클 OFF 시 색상 대신 형태 오버레이로 속성 표시 |
| 자막 | 보스 카메라 연출 비활성 시 → 자막으로 연출 정보 대체 |
| 고대비 | 파티클 OFF 시 스킬 범위를 고대비 격자 패턴으로 대체 |
| 난이도(스토리) | 보스 텔레그래프 확장 + 모션 감소 시 → 텔레그래프 정적 표시 |

---

## 6. 성능 영향 테스트 매트릭스

| 모션 설정 | 최소 사양 FPS | 권장 사양 FPS | 고사양 FPS |
|-----------|-------------|-------------|-----------|
| OFF (원본) | 30 | 60 | 144+ |
| MILD | 32 (+7%) | 62 (+3%) | 144+ |
| MODERATE | 35 (+17%) | 64 (+7%) | 144+ |
| FULL | 38 (+27%) | 66 (+10%) | 144+ |

> 모션 감소는 접근성 목적이나, 저사양 환경에서 성능 향상 부수효과도 있다.
