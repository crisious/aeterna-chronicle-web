# 벨투스 (Veltus)

> 기억 파괴자 훈련사

| 항목 | 내용 |
|------|------|
| 이름 | 벨투스 |
| 역할 | 전직 훈련사 |
| 위치 | 안개해 — 기억의 틈새 |
| 종족 | 인간 (봉인 적응자) |
| 나이 | 45 |

## 성격
침착하고 냉철함. 봉인의 힘에 적응한 극소수 중 하나. 기억 파괴자 클래스의 위험성을 경고하면서도 가르친다.

## 주요 대사
- "봉인의 힘을 쓴다는 건 네 기억의 일부를 대가로 치른다는 뜻이야."
- "준비됐나? 이 훈련에서 돌아오지 못한 자가 셋이나 있어."

## 기능
기억 파괴자 전직 퀘스트 시작 NPC, 스킬 튜토리얼 제공
