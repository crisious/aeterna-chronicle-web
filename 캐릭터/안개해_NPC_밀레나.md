# 밀레나 (Milena)

> 안개해 유랑 약초사

| 항목 | 내용 |
|------|------|
| 이름 | 밀레나 |
| 역할 | 상인 (물약/약초) |
| 위치 | 안개해 — 산호 광장 (이동) |
| 종족 | 엘프 |
| 나이 | 127 |

## 성격
명랑하고 수다스러움. 안개해 약초의 효능을 과장되게 설명하지만 실제로 효과가 있다.

## 주요 대사
- "이 약초는 기억을 되살려주기도 하고… 아, 그건 아직 검증 안 됐나."
- "안개해 생물이 독이 강해서 해독제는 필수야!"

## 판매 품목
안개해 해독제, 심해 회복약, 기억 정화수
