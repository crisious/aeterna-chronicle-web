# 세이렌 (Seiren)

> 심해의 목소리

| 항목 | 내용 |
|------|------|
| 이름 | 세이렌 |
| 역할 | 필드 보스 / 이벤트 NPC |
| 위치 | 안개해 — 심해 콜로세움 입구 |
| 종족 | 심해 정령 |
| 나이 | 불명 |

## 성격
신비롭고 양면적. 적대와 우호 사이를 오간다. 플레이어의 선택에 따라 아군 또는 적이 된다.

## 주요 대사
- "노래를 들으러 왔니, 아니면 침묵시키러 왔니?"
- "이 바다의 기억은 내가 지키고 있어. 네가 그걸 감당할 수 있을까?"

## 역할
시즌 2 이벤트 '심해의 노래' 트리거 NPC, 조건부 필드 보스
