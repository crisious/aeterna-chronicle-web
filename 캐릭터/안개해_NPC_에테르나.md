# 에테르나 (Aetherna)

> 봉인의 수호자

| 항목 | 내용 |
|------|------|
| 이름 | 에테르나 |
| 역할 | 핵심 스토리 NPC |
| 위치 | 안개해 — 봉인 제단 |
| 종족 | 불명 (에테르 존재) |
| 나이 | 불명 |

## 성격
초월적 존재. 인간의 감정을 이해하려 애쓰지만 어딘가 어색하다. 봉인을 유지하는 것이 존재 이유.

## 주요 대사
- "봉인을 풀겠다는 거야? …그건 내 존재를 지우겠다는 뜻이야."
- "기억이란 건… 잊혀져야 비로소 완성되는 건지도 모르겠어."

## 역할
챕터 6 핵심 NPC — 봉인 해제 여부 선택지에서 분기 트리거
