# 토르가 (Torga)

> 안개해 등대지기

| 항목 | 내용 |
|------|------|
| 이름 | 토르가 |
| 역할 | 퀘스트 NPC |
| 위치 | 안개해 — 침묵의 등대 |
| 종족 | 반인반어 |
| 나이 | 불명 (추정 200+) |

## 성격
느릿한 말투, 수수께끼 같은 조언을 건넨다. 안개해가 생기기 전부터 이 등대를 지켜왔다고 주장.

## 주요 대사
- "안개가 걷힌 적은 한 번도 없어. 그건 걷히는 게 아니라, 사라지는 거야."
- "등불은 길을 비추지 않아. 돌아올 곳을 알려줄 뿐."

## 퀘스트
챕터 6 메인 퀘스트 '등대의 기억' 시작 NPC
