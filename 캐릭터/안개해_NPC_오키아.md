# 오키아 (Okia)

> 안개해 술집 주인

| 항목 | 내용 |
|------|------|
| 이름 | 오키아 |
| 역할 | 주민 NPC / 정보상 |
| 위치 | 안개해 — 표류자의 술집 |
| 종족 | 인간 |
| 나이 | 52 |

## 성격
밝고 사교적. 안개해의 모든 소문을 알고 있다. 술 한 잔에 정보를 흘리는 타입.

## 주요 대사
- "이 동네 소식은 다 여기 모이지. 한 잔 할래?"
- "최근 심해에서 이상한 울음소리가 들린다더라. 나라면 안 가겠지만."

## 기능
퀘스트 힌트 제공, 안개해 소문 (일일 갱신 텍스트)
