# 하쉬르 (Hashir)

> 안개해 경비대장

| 항목 | 내용 |
|------|------|
| 이름 | 하쉬르 |
| 역할 | 경비 NPC |
| 위치 | 안개해 — 외곽 초소 |
| 종족 | 인간 |
| 나이 | 41 |

## 성격
엄격하고 규율을 중시. 안개해에 유입되는 외부인을 경계하지만 실력을 인정하면 태도가 바뀐다.

## 주요 대사
- "안개해는 관광지가 아니야. 여긴 살아서 나가는 게 목표인 곳이지."
- "실력이 있군. 좋아, 통과시켜 주지."

## 기능
안개해 진입 게이트 — 일정 레벨/퀘스트 진행도 미충족 시 진입 차단
