# 루미아 (Lumia)

> 심해 연구자

| 항목 | 내용 |
|------|------|
| 이름 | 루미아 |
| 역할 | 퀘스트 NPC / 코덱스 해설 |
| 위치 | 안개해 — 심해 관측소 |
| 종족 | 인간 |
| 나이 | 29 |

## 성격
학구적이고 호기심 왕성. 안개해 생물과 에테르 결정의 관계를 연구 중. 위험을 감수하면서도 데이터를 수집한다.

## 주요 대사
- "이 생물의 발광 패턴이 에테르 결정의 파장과 일치해! 놀랍지 않아?"
- "데이터가 더 필요해. 저 던전 안에 샘플이 있을 거야."

## 퀘스트
심해 생물 조사 연쇄 퀘스트 (3단계), 코덱스 해금 힌트 제공
