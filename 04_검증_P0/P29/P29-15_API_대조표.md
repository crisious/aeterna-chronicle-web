## 자동 언래핑 (P29-01)

`get<T>()`과 `post<T>()` 메서드에 `_unwrapResponse()` 추가:
```typescript
private async _unwrapResponse<T>(response: Response): Promise<T> {
  const json = await response.json();
  if (json && typeof json === 'object' && 'success' in json && 'data' in json) {
    return json.data as T;
  }
  return json as T;
}
```

- 서버가 `{ success: true, data: ... }` 래핑하는 47곳 전부 자동 처리
- Auth 응답(래핑 안 함)은 원본 반환
- 개별 메서드 47개 수정 대신 2곳(get/post) 수정으로 해결
