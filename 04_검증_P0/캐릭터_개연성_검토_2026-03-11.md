## 1. 나이/수명 정합성

### 🔴 K-01. 우르그롬 나이 불일치 (150세 vs 180세)

| 문서 | 나이 |
|------|------|
| 캐릭터 마스터 (동료 표) | **150세** |
| 챕터5 시나리오 | **150세** ("150세의 곰 유사종") |
| 우르그롬 개별 프로필 | **150세** |
| **외전 백스토리** | **180세** ("180년 전 태어남") |

**판정**: 본편 3곳이 150세, 외전만 180세. 외전이 오기.
**조치**: 외전의 `180세` / `180년 전` → `150세` / `150년 전`으로 수정.

### 🔴 K-02. 우르그롬 "대망각의 봉인된 기억 보유" — 개연성 부재

- 우르그롬 150세, 대망각은 212년 전 → **대망각 당시 아직 태어나지 않음**
- 그런데 프로필: "200년 전 대망각에 대한 봉인된 기억을 보유"
- 챕터5에서도 "3대에 걸쳐 기억 소멸의 피해"라고 하며 직접 목격이 아닌 세대 전승 뉘앙스

**판정**: "봉인된 기억"은 **직접 기억이 아니라 기억석에 저장된 선대의 기억**으로 해석 가능. 우르살리스는 기억석 사원에 구술 역사를 보관하는 종족이므로, 기억석을 통해 선대의 대망각 기억을 "보유"한다는 설정은 성립 가능.
**조치**: 우르그롬.md에서 "대망각에 대한 봉인된 기억을 보유" → "기억석 사원에 보관된 선대의 대망각 기억을 계승" 으로 표현 수정. 직접 목격자라는 오해를 방지.

### 🟡 K-03. 크리오 나이 — 마스터 문서 미갱신

| 문서 | 표기 |

| GDD | `212세 (외형 60대)` ← **수정 완료** |
| 캐릭터 마스터 | `인간 (실제 200세+, 외견 60세)` ← **미갱신** |
| 개별 프로필 | `인간 (200세 이상)` / `외견 나이: 60세` |

**판정**: GDD SSOT에서 212세로 확정. 마스터·개별 프로필은 "200세+"로 애매하게 표기.
**조치**: 마스터 + 개별 프로필에서 `200세+` → `212세 (에테르 장수)` 로 정확한 수치 반영.

## 3. 합류 시점 정합성

### 🟡 K-12. 세라핀 관련 지역 표기 — "실반헤임 → 에레보스" 역순

- 캐릭터 마스터 동료 표: `실반헤임 → 에레보스 | 챕터 1`
- 실제 시나리오: 챕터 1(에레보스)에서 첫 만남, 챕터 2에서 실반헤임 방문
- `실반헤임 → 에레보스`는 세라핀의 **출신지 → 합류 지역** 순서로 읽을 수 있으나, 게임 진행 순서와 역순이라 혼란

**조치**: `에레보스(합류) ← 실반헤임(출신)` 또는 `에레보스 | 챕터 1`로 명확화.
