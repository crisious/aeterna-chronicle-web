> 상태: In progress

---

## 1) 목표

- 엔딩 분기 우선순위 `D → C → A → B` 자동 회귀 검증
- 패배 루트(`DEFEAT`) 분기 검증 포함

---

## 2) 자동화 스크립트

- 경로: `tools/regression/ending_regression_runner.py`
- 입력: 케이스 내 플래그 조합
- 출력: `04_검증_P0/P2/P2-06_regression_result.json`

---

## 3) 케이스 구성

- `REG-D-001`: D 우선 적용 검증
- `REG-C-001`: C 분기 검증
- `REG-A-001`: A 분기 검증
- `REG-B-001`: B 분기 검증
- `REG-FAIL-001`: 패배 루트 강제 검증

---

## 4) 판정 기준

- 전체 케이스 PASS 시 회귀 통과
- 단 1건이라도 FAIL이면 회귀 실패
