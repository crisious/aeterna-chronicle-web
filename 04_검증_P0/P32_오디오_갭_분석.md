## 6. 디렉터리 구조 검증

### 현재 구조
```
audio/
└── bgm/          ← 15개 OGG 파일 (23MB)
```

### 권장 구조
```
audio/
├── bgm/
│   ├── field/     ← 지역 필드 테마
│   ├── battle/    ← 전투 BGM
│   ├── boss/      ← 보스 전용 테마
│   ├── event/     ← 이벤트/시즌 (현재 보유)
│   ├── ending/    ← 엔딩 BGM (현재 보유)
│   └── system/    ← 메뉴/승리/대화
├── sfx/
│   ├── battle/    ← 전투 SFX (BTL-001~025)
│   ├── ui/        ← UI SFX (UI-001~016)
│   ├── memory/    ← 기억 메카닉 SFX (MEM-001~009)
│   └── environment/  ← 환경 SFX (지역별)
└── voice/         ← Phase 2+ (미착수)
```

## 8. 우선순위별 생성 로드맵

### P0 — 즉시 필요 (게임 플레이 필수)
- 전투 SFX 12종 (기본 타격/피격/사망)
- UI SFX 7종 (클릭/창/레벨업/퀘스트)
- 기억 SFX 5종 (파편 감지~소멸)
- 지역 필드 BGM 6종
- **소계: 30종**

### P1 — 차기 필요 (게임 경험 보강)
- 전투 SFX 9종 (스킬별 특화)
- UI SFX 7종 (호버/장착/강화)
- 기억 SFX 4종 (소환/화살/폭발)
- 전투/보스 BGM 9종
- 환경 SFX 18종
- **소계: 47종**

### P2 — 후기 (폴리싱)
- UI SFX 2종 (거래/전직)
- 보이스 (Phase 2 착수 시)
- 시스템 BGM 추가
- **소계: ~45종 (보이스 포함)**
