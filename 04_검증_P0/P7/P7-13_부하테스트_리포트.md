## 1. 테스트 도구

### k6 시나리오 (`tools/loadtest/k6_scenario.js`)

- **실행 명령**: `k6 run tools/loadtest/k6_scenario.js`
- **시나리오 흐름**: 로그인 → 존 이동 → 전투 진입 → 스킬 사용(3회) → 인벤토리 → 거래소 → 채팅 → 헬스체크
- **부하 프로파일**:
  - Ramp-up: 1분 → 200 VU, 2분 → 500 VU, 2분 → 1000 VU
  - 유지: 10분 × 1000 VU
  - Ramp-down: 2분 → 500 VU, 1분 → 0
- **SLA 임계값**:
  | 지표 | 임계값 |
  |------|--------|
  | p95 응답 시간 | < 500ms |
  | 에러율 | < 1% |
  | RPS | > 5,000 |
  | 로그인 p95 | < 1,000ms |
  | HTTP 실패율 | < 1% |

### Artillery 시나리오 (`tools/loadtest/artillery_config.yml`)

- **실행 명령**: `npx artillery run tools/loadtest/artillery_config.yml`
- **시나리오 구성**:
  - Full Game Flow (가중치 60%): 로그인 → 존이동 → 던전 → 스킬 → 인벤토리 → 채팅
  - Auction Heavy (가중치 20%): 로그인 → 거래소 목록 5회 → 검색
  - Health Check (가중치 20%): 헬스 엔드포인트
- **부하 프로파일**:
  - 워밍업: 1분 × 10 req/s
  - Ramp-up: 3분 × 10→100 req/s
  - 유지: 10분 × 100 req/s
  - 스파이크: 1분 × 200 req/s
  - 쿨다운: 1분 × 100→10 req/s
