## 3. Swagger UI 설정

```typescript
// server/src/routes/ 내 Fastify Swagger 플러그인 등록
import swagger from '@fastify/swagger';
import swaggerUi from '@fastify/swagger-ui';

await fastify.register(swagger, {
  openapi: {
    info: { title: '에테르나 크로니클 API', version: '1.0.0' },
  },
});

await fastify.register(swaggerUi, {
  routePrefix: '/docs',
  uiConfig: { docExpansion: 'list' },
});
```

프로덕션: `/docs` 경로에서 Swagger UI 접근 가능.  
정적 spec: `docs/openapi.yaml` 파일 직접 참조.
