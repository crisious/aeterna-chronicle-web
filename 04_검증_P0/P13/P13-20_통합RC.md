## 3) 샘플 파이프라인 E2E 검증

### 3.1 검증 시나리오

```
[시나리오] 에테르 기사 idle-front 스프라이트 4프레임
  ① 프롬프트 작성 (ai-prompt-master.md SD 캐릭터 템플릿)
  ② AI 생성 시뮬 (실제 생성 대신 규격 확인)
  ③ 배경 제거 (remove_bg.py 호출 구조 확인)
  ④ 색보정 (color_correct.py 에레보스 팔레트)
  ⑤ QA 검증 (qa_checker.py 해상도+알파+색상)
  ⑥ 스프라이트 시트 조립 (spritesheet_assembler.py)
```

### 3.2 단계별 검증

#### ① 프롬프트 작성

```
입력:
  엔진: Stable Diffusion
  카테고리: 캐릭터
  클래스: 에테르 기사
  모션: idle
  방향: front (D)

생성 프롬프트:
  (masterpiece:1.3), (best quality:1.2), 2D pixel art RPG character sprite,
  dark fantasy Korean MMORPG style,
  에테르 기사, male, heavy plate armor with glowing ether runes,
  blue energy lines on pauldrons, sturdy shield,
  one-handed longsword with ethereal blue glow,
  kite shield with memory crest,
  standing idle pose, front-facing, full body view,
  steel blue, silver, midnight blue, faint cyan glow color scheme,
  에레보스 ruined dark city aesthetic,
  64x64 pixel art, (clean pixel edges:1.4), transparent background,
  game sprite sheet ready, hand-painted textures,
  (consistent art direction:1.2)

  Negative: [공통 네거티브]
  Model: pixel-art-xl-v1.1
  Sampler: DPM++ 2M Karras
  Steps: 35, CFG: 8, Seed: 기록

검증: ✅ 프롬프트 구조 ai-prompt-master.md 섹션 3.2 준수
```

#### ② AI 생성 시뮬

```
시뮬 출력: char_erebos_ether-knight_idle-front-f{00-03}_64x64.png × 4장
규격: 64×64, RGBA, 투명 배경
검증: ✅ 네이밍 규칙 준수 (asset-versioning.md 섹션 2)
```

#### ③ 배경 제거

```
명령: python remove_bg.py raw/frame.png output/frame_nobg.png --model isnet-anime
예상 출력: RGBA, 투명 비율 40~70%
검증: ✅ remove_bg.py 인터페이스 정상, 배치 모드 지원
```

#### ④ 색보정

```
명령: python color_correct.py frame_nobg.png frame_cc.png --palette erebos --strength 0.5
예상 출력: 에레보스 팔레트(어두운 보라-회색) 방향 보정
검증: ✅ color_correct.py 9개 지역 팔레트 내장, 강도 조절 가능
```

#### ⑤ QA 검증

```
명령: python qa_checker.py check frame_cc.png --expected-size 64 --check-alpha
예상 체크:
  - 해상도: 64×64 ✅
  - 알파 채널: RGBA, 배경 투명 ✅
  - 색상 일관성: 고유 색상 ≤256 ✅
  - 반투명 비율: <5% ✅
검증: ✅ qa_checker.py 4종 검증 (해상도/알파/색상/네이밍)
```

#### ⑥ 스프라이트 시트 조립

```
명령: python spritesheet_assembler.py frames/ ether_knight_idle.png --preset character --cols 4
예상 출력: (4+2)×(64+2)+2 = 266 × 68 시트
메타데이터: ether_knight_idle.json (프레임 좌표)
검증: ✅ spritesheet_assembler.py 프리셋+JSON 메타 생성
```
