# P26 계획 분석 — 게임플레이 루프 완성

> 작성일: 2026-03-14 | Phase: P26 | 순서: 489~508

---

## 1. 목표

퀘스트 수락 → NPC 대화 → 전투 → 보상 → 레벨업 → 장비 강화의 풀 게임플레이 루프를 완성한다. 인벤토리, 상점, 스킬 트리, 맵, 던전 관련 UI를 모두 구현한다.

## 2. 현황

- 서버 API: questRoutes, npcRoutes, inventoryRoutes, craftRoutes, dungeonRoutes 등 이미 존재
- 클라이언트: 씬 구조 존재하나 실제 게임 루프 미연결
- P24 전투 엔진 + P25 연동 완료 전제

## 3. 구현 계획

### 3.1 게임플레이 풀 루프
- 퀘스트 수락 → 목표 추적 → NPC 대화 → 전투 트리거 → 전투 → 보상 수령 → 퀘스트 완료
- 레벨업 시 스탯 분배 UI → 스킬 해금 → 장비 강화/교체

### 3.2 인벤토리 UI
- 아이템 목록 그리드 (가방 탭/장비 탭)
- 아이템 상세 정보 팝업
- 장착/해제, 드래그 앤 드롭
- 정렬 (이름/등급/종류)

### 3.3 상점 UI
- NPC 상점: 구매/판매 인터페이스
- 가격 표시, 보유 골드 확인
- 구매 확인 다이얼로그

### 3.4 스킬 트리 UI
- 6클래스별 스킬 트리 시각화
- 스킬 포인트 투자, 전직 분기 표시
- 스킬 상세 (데미지, 쿨다운, 마나, 설명)

### 3.5 맵 시스템
- 미니맵 (HUD 우상단)
- 월드맵 (풀스크린, 존 선택)
- 존 이동 + 텔레포트 포인트

### 3.6 던전 시스템 UI
- 던전 입장 조건 확인 (레벨, 키 아이템)
- 던전 내부 진행률 표시
- 클리어 판정 + 보상 화면

### 3.7 퀘스트 추적 HUD
- 활성 퀘스트 목록 (화면 우측)
- 목표 진행률 실시간 갱신
- 퀘스트 완료 알림

## 4. 리스크

| 리스크 | 대응 |
|--------|------|
| UI 요소 과다로 성능 저하 | DOM 풀링 + 가상 스크롤 |
| 게임 루프 분기 복잡도 | 상태 머신 기반 플로우 관리 |
| 존 이동 시 데이터 로딩 지연 | 프리로딩 + 로딩 화면 |
