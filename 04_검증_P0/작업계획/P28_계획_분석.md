# P28 계획 분석 — 폴리싱 + 플레이테스트

> 작성일: 2026-03-14 | Phase: P28 | 순서: 529~548

---

## 1. 목표

게임 전체의 밸런스 조정, UX 개선, 오디오 연결, 버그 수정, 성능 최적화, 접근성 검증을 수행하고 최종 플레이테스트 리포트를 작성한다.

## 2. 현황

- BGM 42곡 + SFX 75개 에셋 존재, 실제 재생 미연결
- E2E 테스트 프레임워크 구축 완료 (vitest)
- 접근성(WCAG AA) 기본 마크업 적용됨
- 몬스터 197종, 던전 69종 수치 설정 완료

## 3. 구현 계획

### 3.1 밸런스 패스
- 몬스터 HP/ATK/DEF/보상 수치 조정 (레벨 구간별)
- 아이템 가격/성능 곡선 검증
- 경험치 곡선 스무딩
- 스킬 데미지 계수 재조정

### 3.2 UX 개선
- 씬 전환 애니메이션 (페이드/슬라이드)
- 버튼 호버/클릭 피드백
- 툴팁 통일
- 사운드 연동 (UI 효과음)

### 3.3 오디오 연결
- BGM: 42곡 → 존/던전/전투/메뉴별 매핑
- SFX: 75개 → 액션별 매핑 (공격, 스킬, 레벨업, 아이템 획득 등)
- 볼륨 조절, 페이드 인/아웃
- 오디오 매니저 싱글톤

### 3.4 버그 수정
- E2E 테스트 실행 → 발견 이슈 일괄 수정
- 엣지 케이스 처리 (빈 인벤토리, 0골드 거래, 만렙 경험치 등)

### 3.5 성능 최적화
- 텍스처 아틀라스 최적화 (메모리 사용량)
- GC 부하 감소 (오브젝트 풀링)
- 네트워크 트래픽 최적화 (delta 전송)
- 번들 크기 최적화

### 3.6 접근성 (WCAG AA)
- 색상 대비 검증
- 키보드 네비게이션
- 스크린 리더 지원
- 폰트 크기 조절

### 3.7 최종 산출물
- 플레이테스트 리포트
- README 최종 갱신 (P24~P28 포함)

## 4. 리스크

| 리스크 | 대응 |
|--------|------|

| 오디오 파일 로딩 시간 | 스트리밍 + 지연 로딩 |
