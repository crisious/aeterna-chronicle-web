## 0) 공통 규약

- 공통 네이밍: `HudStatusBar`, `HudQuickSlotBar`, `HudQuestTracker`, `HudDialogueBox`
- 단위:
  - 시간(ms)
  - 비율(0.0~1.0)
  - 색상(hex)
- 이벤트 채널: `ui.event.*`
- 데이터 갱신 주기:
  - 전투 수치(HP/MP/CD): 100ms 이하
  - 퀘스트/대화: 상태 변경 시 이벤트 기반

---

## 1) HudStatusBar 계약

### 1.1 Props

| 키 | 타입 | 필수 | 설명 |
|---|---|---|---|
| `hpCurrent` | number | Y | 현재 HP |
| `hpMax` | number | Y | 최대 HP |
| `mpCurrent` | number | Y | 현재 MP |
| `mpMax` | number | Y | 최대 MP |
| `level` | number | Y | 현재 레벨 |
| `expRatio` | number(0~1) | Y | 경험치 진행률 |
| `avatarUrl` | string | N | 아바타 이미지 URI |
| `characterName` | string | Y | 캐릭터명 |
| `dangerHpThreshold` | number(0~1) | N | HP 위험 임계치(기본 0.2) |

### 1.2 State

- `dangerBlink: boolean` (HP 비율 <= 임계치)
- `showExactValue: boolean` (hover/focus 시 숫자 상세 표시)

### 1.3 Events

- `ui.event.status.avatar_click`
- `ui.event.status.hover_change`
- `ui.event.status.hp_critical` (critical 진입 시 1회)

### 1.4 접근성

- 색상 외 텍스트 동시 표기(`248/400`)
- HP 위험 상태에서 진동/사운드 대체 피드백 옵션 제공
- 스크린리더 레이블: `"HP 248 of 400"`

---

## 2) HudQuickSlotBar 계약

### 2.1 Props

| 키 | 타입 | 필수 | 설명 |
|---|---|---|---|
| `slots` | Slot[] | Y | 슬롯 배열(기본 12칸) |
| `activePage` | number | N | 페이지 인덱스(확장 시) |
| `inputMode` | enum(`keyboard`,`gamepad`) | Y | 입력 모드 |

`Slot` 구조:
- `slotIndex: number`
- `icon: string`
- `label: string`
- `cooldownMs: number`
- `remainingCooldownMs: number`
- `stackCount: number`
- `isUsable: boolean`
- `hotkey: string` (예: `1`, `2`, `LB+X`)

### 2.2 State

- `selectedSlotIndex: number`
- `cooldownAnimating: boolean`
- `invalidPulse: boolean` (사용 불가 입력 시)

### 2.3 Events

- `ui.event.quickslot.use`
- `ui.event.quickslot.select`
- `ui.event.quickslot.swap`
- `ui.event.quickslot.invalid_use`

### 2.4 접근성

- 키보드 포커스 순환(Tab/Shift+Tab)
- 쿨다운 시각 표시 + 남은 시간 텍스트(초)
- 색약 모드에서 사용가능/불가 명도 대비 4.5:1 이상

---

## 3) HudQuestTracker 계약

### 3.1 Props

| 키 | 타입 | 필수 | 설명 |
|---|---|---|---|
| `quests` | QuestItem[] | Y | 추적 퀘스트 목록 |
| `maxVisible` | number | N | 최대 노출 개수(기본 3) |
| `collapsed` | boolean | N | 접힘 여부 |

`QuestItem` 구조:
- `questId: string`
- `title: string`
- `objectiveText: string`
- `progressCurrent: number`
- `progressTarget: number`
- `isMainQuest: boolean`
- `isCompleted: boolean`
- `distanceMeters?: number`

### 3.2 State

- `expanded: boolean`
- `highlightQuestId: string | null`

### 3.3 Events

- `ui.event.questtracker.toggle`
- `ui.event.questtracker.pin`
- `ui.event.questtracker.navigate`

### 3.4 접근성

- 메인/서브 퀘스트 구분 아이콘 + 텍스트 병기
- 진행도는 숫자(`2/5`)로 항상 표기
- 완료 퀘스트는 취소선 대신 `완료` 배지 포함

---

## 4) HudDialogueBox 계약

### 4.1 Props

| 키 | 타입 | 필수 | 설명 |
|---|---|---|---|
| `speakerName` | string | Y | 화자 이름 |
| `speakerPortrait` | string | N | 화자 초상 URI |
| `bodyText` | string | Y | 대사 본문 |
| `choices` | DialogueChoice[] | N | 선택지 목록 |
| `canSkip` | boolean | N | 스킵 가능 여부 |
| `autoAdvanceMs` | number | N | 자동 진행 시간(없으면 수동) |

`DialogueChoice` 구조:
- `choiceId: string`
- `text: string`
- `disabled: boolean`
- `hint?: string`

### 4.2 State

- `typingProgress: number` (타이핑 연출 진행률)
- `selectedChoiceIndex: number`
- `isAwaitingInput: boolean`

### 4.3 Events

- `ui.event.dialogue.next`
- `ui.event.dialogue.skip`
- `ui.event.dialogue.choice_select`
- `ui.event.dialogue.choice_confirm`

### 4.4 접근성

- 본문 영역 최소 글자 크기 18px(PC 기준)
- 선택지 포커스 이동 키보드/패드 공통 지원
- 대사 자동재생 OFF 옵션 제공

## 5) 엔진별 바인딩 규약

| 컴포넌트 | Phaser/React | Unity(UI Toolkit) | UE5(UMG/CommonUI) |
|---|---|---|---|
| StatusBar | React state + CSS vars | UXML + USS var + C# VM | Widget Blueprint + Data Asset |
| QuickSlot | key/gamepad input hook | Enhanced Input + UI Toolkit | Enhanced Input + CommonUI action |
| QuestTracker | store subscribe | ScriptableObject/VM | GameplayTag 기반 상태 바인딩 |
| DialogueBox | narrative store event bus | DialogueController + UI Document | DialogueSubsystem + WBP_Dialogue |
