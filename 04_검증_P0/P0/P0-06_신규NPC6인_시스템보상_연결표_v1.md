## 2) 보상 매핑 표 (Choice → System)

### 2.1 누아리엘 — 기억 과부하 안정화 의뢰 (C2-N2)

| 선택 | 시스템 효과 | 플래그/변수 |
|---|---|---|
| A 바로 수집 | 상태이상 저항 레시피 해금 + 전투 시작 버프(저항 +12%, 15분) | `quest.nuariel_moss=COMPLETE`, `recipe.status_resist.unlocked=true`, `trust.nuariel +10` |
| B 대체 재료 | 즉시 버프(저항 +8%, 10분) + 제작 비용 +25% | `buff.resist_temp=true`, `craft.cost.multiplier=1.25`, `trust.nuariel +5` |
| C 후순위 | 버프 없음, 공명 과부하 패널티(스킬가속 -5%) | `debuff.overload=true`, `trust.nuariel -5` |

## 3) 구현 계약 (Dev Contract)

### 3.1 공통 이벤트 키

- `npc.choice.<npc_key>.<chapter_scene> = A|B|C`
- 예: `npc.choice.nuariel.c2n2 = A`

### 3.2 적용 타이밍

1. 대화 선택 확정 직후 `choice_history` 기록
2. `reward_resolver`에서 시스템 효과 적용
3. UI 토스트: “획득/변경 효과” 1~2줄 표시
4. 저장 시 스냅샷 반영

### 3.3 롤백 규칙

- 저장 전 취소 가능, 저장 후 취소 불가
- 재시도 시 동일 선택 강제(치트/오염 방지)
