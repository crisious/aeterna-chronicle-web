### 🔴 C-01. 대망각 연도 불일치 (3,200년 vs 3,212년)

| 문서 | 표기 |
|------|------|
| GDD_final.md | "212년 전 (세계력 3,200년)" |
| 연대표_역사기록.md | "세계력 3,200년 — 대망각" |
| 시나리오_마스터.md | "대망각 — **세계력 3,212년**" |

**문제**: 현재 시점이 3,412년이면:
- "212년 전" = 3,200년 (GDD·연대표 기준) ✅
- 시나리오 마스터의 "3,212년" 표기 → "200년 전"이 되어야 함 ❌

**판정**: 시나리오 마스터의 `3,212년`은 오기. GDD·연대표의 `3,200년`이 SSOT.
**조치**: 시나리오_마스터.md 71행 `3,212년` → `3,200년`, 98행 `3,212~3,412년` → `3,200~3,412년` 수정 필요.

### 🔴 C-02. 마에스트로 크리오 나이 치명적 모순

| 문서 | 표기 |
|------|------|
| GDD_final.md | "인간, **60세**" |
| 연대표_역사기록.md | "크리오의 출생: **세계력 3,200년** 7월" |
| 연대표 7-6장 | "200년 생존의 비밀 — 에테르 생리 구조" |

**문제**: 
- 3,412 - 3,200 = **212세**
- GDD에서 "인간, 60세"라고 표기 → **152년 오차**
- 연대표에서는 "200년 생존"을 명시하고 에테르 생리 구조로 장수 설명

**판정**: 연대표의 "212세 + 에테르 장수" 설정이 서사적으로 정합. GDD의 "60세"는 **외형 나이(겉보기)** 의도로 추정되나 명시되지 않음.
**조치**: GDD에서 `인간, 60세` → `인간, 212세 (외형 60대)` 또는 `인간, 외견 60세 / 실제 212세` 로 수정. 캐릭터 프로필에도 동일 반영.

### 🟡 C-03. 신성 기억 파편 수 불일치 (4개 vs 5개)

| 문서 | 표기 |
|------|------|
| 멀티엔딩_플래그.md | `fragment_count` 최대값 = **4** |
| 챕터5_시나리오.md v1.3 | "파편 수를 **4개** 기준으로 통일" |
| 시나리오_마스터.md | "신성 기억 파편 **5개** 봉인" / "5개 수집" (6곳 이상 언급) |
| 연대표 | "파편의 위치: 실반헤임, 솔라리스, 아르겐티움, 북방 영원빙원, **망각의 고원** = 5곳" |

**문제**: 
- 플래그 SSOT(멀티엔딩 설계서)는 **4개**로 확정
- 시나리오 마스터는 아직 **5개**로 표기 (미수정)
- 연대표도 봉인 위치 5곳 표기
- 챕터5 v1.3에서 4개로 통일했지만 상위 문서에 미반영

**판정**: 챕터5 수정(v1.3)이 최신. 멀티엔딩 플래그 SSOT도 4개. 시나리오 마스터·연대표가 미갱신.
**조치**: 시나리오_마스터.md, 연대표에서 "파편 5개" → "파편 4개" 일괄 수정. 봉인 위치 5곳 중 1곳을 "히든 유물 봉인지"로 재분류하거나 삭제.

### 🟡 C-06. 세라핀 "300년을 살면서" — 종족 수명 미명시

- 챕터5: "300년을 살면서 많은 사람을 봤어"
- 엘파리스(엘프)족이므로 장수 가능 → 설정상 타당
- **그러나** 대망각이 212년 전이므로, 세라핀이 300세면 대망각 이전(~3,112년)에 태어난 것
- 챕터5 80행에서 "대망각 이후 200년. 나는 그 200년을 기억해"라고 말하는데, 300세면 대망각 이전 ~88년도 기억해야 함

**판정**: 논리적으로 성립 가능 (대망각 전후 모두 경험). 단, 엘파리스 종족 평균 수명이 문서에 명시되지 않음.
**조치**: 캐릭터/세라핀.md 또는 world_building에 엘파리스 수명(500~1,000년 등) 명시 권장.

## 3. 서사 개연성

### 🟡 C-08. 챕터 1 에레보스 → GDD 지역 순서와 불일치

| 문서 | 챕터 1 지역 |
|------|------------|
| GDD | 아르겐티움 (시작 도시, Ch.1) |
| 시나리오 마스터 | 에레보스 (유령 도시, Ch.1) |
| P6-10 chapterManager | 아르겐티움 (Lv.1, 튜토리얼 완료) |

**문제**: GDD는 아르겐티움이 Ch.1, 시나리오 마스터는 에레보스가 Ch.1.
**판정**: 시나리오 마스터의 서사가 더 구체적이고 최신(칸텔라 마을 폭풍 → 에레보스 진입). GDD 표의 "아르겐티움=Ch.1"은 시나리오 확장 전의 초기 매핑.
**조치**: GDD 지역 테이블의 챕터 배정을 시나리오 마스터 기준으로 업데이트. P6-10 chapterManager의 챕터1 지역도 `에레보스`로 변경.

### ✅ C-11. 레테의 동기 — 서사적 개연성

- **서사적으로 견고함.** 안타고니스트의 동기에 공감 가능한 여지를 줌.

## 4. 시스템-서사 연동

### 🟡 C-13. 파편 획득 챕터 — 4개 기준 불완전

현재 멀티엔딩 플래그에서 `fragment_count +1` 트리거:
- Ch.1: 에레보스 봉인 유적 → +1
- Ch.2: 실반헤임 말라투스 → +1
- Ch.3: 솔라리스 라와르 → +1 (또는 -1 가능)
- Ch.4: 아르겐티움 지하 → +1

**합계**: Ch1~4에서 최대 4개 — 멀티엔딩 SSOT와 일치 ✅
**그러나**: Ch.3에서 -1 가능(굴복 선택지) → 최소 3개 가능 → 엔딩 B 조건 확인 필요

---

### 🔵 C-14. 신성 유물 12개 (엔딩 D) 획득 경로 미문서화

- 엔딩 D 조건: `artifact_count == 12`
- 유물 12개의 **구체적 위치/획득 조건**이 어떤 문서에도 상세 정의되지 않음
- 멀티엔딩 설계서에 `artifact_count` 변화 선택지는 일부만 존재

**조치**: 유물 12개의 전체 목록 + 챕터별 획득 조건 상세 문서화 필요.
