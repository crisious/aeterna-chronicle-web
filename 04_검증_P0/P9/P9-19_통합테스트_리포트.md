## 2. 테스트 시나리오 상세

- ✅ Unity WebGL 로그인 → JWT 발급 → 세션 생성
- ✅ Phaser Web 로그인 → JWT 발급 → 세션 생성
- ✅ TOTP 2FA 플로우 → 디바이스 신뢰 등록
- ✅ 이중 로그인 → 이전 세션 킥

### 2.2 Stripe 결제 통합
- ✅ Checkout Session 생성 → Stripe URL 반환
- ✅ Webhook 결제 완료 → 크리스탈 지급 (1000개)
- ✅ 시즌패스 구독 생성 → 자동 갱신 설정
- ✅ 환불 처리 → 크리스탈 회수
- ✅ 미성년자 결제 한도 초과 → 차단

### 2.3 안티치트 통합
- ✅ 클라이언트 무결성 해시 검증
- ✅ 매크로 탐지 (confidence 0.85) → 경고
- ✅ 봇 탐지 (confidence 0.75) → 24시간 정지 (에스컬레이션)
- ✅ 서버 권위 모델: 변조 데미지 무시 + 플래그
- ✅ 속도 해킹 탐지 → 위치 리셋

- ✅ 레이트 리미팅: 5회 초과 로그인 차단

- ✅ GDPR 계정 삭제 (30일 유예기간)
- ✅ XSS 페이로드 차단 + sanitize
- ✅ JWT refresh token 갱신 + 이전 토큰 무효화

### 2.5 크로스 플랫폼 결제 + 보안
