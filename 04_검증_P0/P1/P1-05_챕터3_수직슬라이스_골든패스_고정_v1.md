## 1) 수직슬라이스 골든패스 (고정 경로)

- 대상 레벨: **Lv 22 기준 파티 1인(싱글) / 동료 AI 2인(세라핀, 이그나)**
- 예상 소요: **22~30분**
- 챕터 구간: 솔라리스 사막 진입 ~ 라와르 조우 직후 파편 확보 저장

### STEP 01 — 탐색: 솔라리스 사막 외곽 진입
- 시작 지점: `씬 3-1 솔라리스 사막 진입`
- 목표: 이프리타 캠프 경계선까지 생존 이동
- 통과 조건:
  - 열사병/지형 피해로 인한 사망 없음
  - 기억 공명 튜토리얼 팝업 정상 노출 후 해제

### STEP 02 — 대화분기(동맹): 이그나 협상 고정
- 시작 지점: `씬 3-2 이그나와의 첫 만남`
- **고정 선택:** `협상 수락(거래 성립) + 이그리스 기억 복원 조사 동행`
- 고정 이유:
  - 챕터3 핵심 동반자 플래그(이그나 동행) 안정 확보
  - 후속 유적 진입/대사 분기 변동성 최소화
- 기대 결과:
  - `party.igna_joined = true`
  - `quest.c3.desert_flame = ACTIVE`

### STEP 03 — 탐색+전투: 유적 2층 골렘 구간
- 시작 지점: `씬 3-3 고대 솔리안 유적 탐사`
- 전투 대상: `솔리안 에테르 수호 골렘 x3`
- 전투 목표:
  - 골렘 3기 정리
  - 에리언 기억 함정 1회 발동 후 복귀 연출 정상
- 실패 조건:
  - 연계 방어막 해제 실패로 전투 무한 지연
  - 기억 함정 복귀 이벤트 미발생

### STEP 04 — 보상: 유적 중층 전투 종료 보상 지급
- 고정 보상 패키지:
  - EXP/골드: Ch3 엘리트 기준 보상 테이블
  - 드롭: 에테르 결정 파편(제작재료) 1회
  - 스토리 보상: 봉인실 진입 권한 ON
- 검증 포인트:
  - 전투 종료 즉시 HUD/획득 로그 반영
  - 인벤토리 반영 누락/중복 없음

### STEP 05 — 대화분기: 라와르 조우 루트 고정
- 시작 지점: `씬 3-3 하층 봉인실`
- **고정 선택:** `기억 공명 3회 성공(기억 루트)`
- 기대 결과:
  - `story.fragment_count +1`
  - `story.seal_visited |= 4`
  - `ending.progress.A += 15`
  - `boss.ravar.phase_skip = true`

### STEP 06 — 저장: 슬라이스 완료 저장 포인트
- 저장 시점: 파편 획득 직후, 유적 외곽 안전 지대 복귀
- 저장 슬롯명(권장): `SV_C3_VERTICAL_P1_05`
- 저장 데이터 최소 포함:
  - 챕터 진행도
  - 주요 플래그(`fragment_count`, `seal_visited`, `party.igna_joined`)
  - 보상 인벤토리/퀘스트 상태

## 2) 고정 플래그 스냅샷 (완료 시점 기준)

| 키 | 기대값 |
|---|---|
| `story.fragment_count` | +1 증가 |
| `story.seal_visited` | bit2 ON (솔라리스 유적 방문) |
| `party.igna_joined` | true |
| `quest.c3.desert_flame` | COMPLETE 또는 TURN_IN |
| `ending.progress.A` | +15 이상 |
| `chapter.progress` | Ch3 중후반 완료 상태 |
