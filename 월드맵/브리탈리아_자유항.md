## 지리적 특성

어느 국가에도 속하지 않는 **독립 자치 도시**이자 에테르나 최대 무역항.
모든 종족과 세력의 상인·용병·정보상·밀수업자들이 모여드는 혼돈의 허브.
바다 안개 속에서 모습을 드러내는 유령선들과 **잊혀진 섬** 전설로 유명.

### 특수 지형 현상

| 현상 | 설명 |
|------|------|
| 무한 안개 (Endless Mist) | 항구 바깥을 둘러싼 안개. 안에서 방향 감각 상실. 특수 항법 도구 필요 |
| 유령선 출몰 | 안개 속에서 불빛 없는 선박이 간헐적 목격. 접근 시 과거 선원 기억 공명 발동 |
| 정보 밀도 | 에테르나에서 정보가 가장 빠르게 순환하는 곳. NPC들이 모든 지역 소문 보유 |

## 마에스트로 크리오 합류 조건

- 챕터 4에서 선술집 '코르도바' 방문
- 크리오는 200년을 살아남은 생존자로, [[캐릭터_정리#카일 (Kail) — 에리언의 전생|카일]]의 행적에 대한 유일한 증인
- 에리언에게 카일이 봉인 전날 남긴 기록을 전달
- 합류 조건: [[연대표_역사기록]] 7-6 섹션 참조

## 잊혀진 섬 — 히든 콘텐츠

> *"안개 너머에 섬이 있다는 걸 안다. 이 항구에서 30년을 일했는데, 안개 속에서 빛이 보인 건 세 번. 가까이 가려 하면 안개가 두꺼워진다."*
> — 브리탈리아 노선원의 증언

| 항목 | 내용 |
|------|------|
| 위치 | 무한 안개해 깊은 곳 |
| 접근 조건 | 특수 항법 도구 + 기억 공명 최고 레벨 |
| 내용 | 봉인된 11개 신 중 하나의 기억 봉인 위치 |
| 보상 | 히든 엔딩 루트 단서 |

## 비주얼 & 사운드 방향

- **색조**: 회색 안개, 갈색 목재, 에테르 파란 항구 조명
- **날씨**: 항상 촉촉한 바다 안개. 간헐적 폭우
- **사운드**: 갈매기 소리, 파도 소리, 선술집에서 흘러나오는 에테르나 민요
- **특이 효과**: 안개 속에서 유령선 실루엣 가시화. 기억 공명 시 선원의 공포 감정 전달
