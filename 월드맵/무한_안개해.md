## 지리적 특성

에테르나 대륙 서쪽 끝에 펼쳐진 무한한 안개 바다. 대망각(3,200년) 당시 12인의 봉인자가 레테를 가두기 위해 만든 **봉인 해역**이다. 에테르 에너지가 비정상적으로 응축되어 안개가 영구적으로 유지되며, 일반인이 진입하면 기억이 점진적으로 소실된다.

200년간 아무도 돌아오지 못한 금지 해역이었으나, 에리언의 신성 기억 파편 4개가 안개를 밀어내는 열쇠가 된다.

### 특수 지형 현상

| 현상 | 설명 |
|------|------|
| 기억 안개 (Memory Fog) | 에테르 농도에 따라 단기→장기 기억 순서로 소실. 파편 보유자는 저항 |
| 봉인 공명 (Seal Resonance) | 봉인석 근처에서 200년 전 봉인 의식 장면이 홀로그램으로 재현 |
| 시간 왜곡 (Time Distortion) | 안개해 내부 시간 흐름이 외부 대비 1.5배 느림 |
| 에테르 폭풍 (Ether Storm) | 주기적으로 발생하는 에테르 폭풍 — 항해 불가, 전투 필드 생성 |

## 존 상세

### Zone 1: 안개해 외곽 (Mist Sea Outskirts)

| 항목 | 내용 |
|------|------|
| 레벨 | 60~65 |
| 특성 | 안개 옅음, 초보 안개 몬스터 서식 |
| 허브 | 안개의 항구 — 보급/수리/퀘스트 NPC |
| 던전 | 침몰한 상선 (일반 던전) |
| 연결 | [[브리탈리아_자유항]] ↔ 기억의 등대 |

### Zone 2: 기억의 등대 (Memory Lighthouse)

| 항목 | 내용 |
|------|------|
| 레벨 | 65~70 |
| 특성 | 봉인자 메모리아의 에테르 등대 |
| 핵심 | 챕터 6 씬 6-3 무대, 봉인 균열 발견 |
| 던전 | 등대 지하 미궁 (일반 던전) |
| 연결 | 안개해 외곽 ↔ 표류자의 군도 |

### Zone 3: 표류자의 군도 (Drifter's Archipelago)

| 항목 | 내용 |
|------|------|
| 레벨 | 70~75 |
| 특성 | 수백 년간 안개에 갇힌 난파선·잔해 밀집 |
| 핵심 | 숨겨진 보물 + 유령선 이벤트 |
| 던전 | 유령선 잔해 (엘리트 던전), 해적왕의 무덤 (일반 던전) |
| 연결 | 기억의 등대 ↔ 봉인의 첨탑 |

### Zone 4: 봉인의 첨탑 (Seal Spire)

| 항목 | 내용 |
|------|------|
| 레벨 | 75~80 |
| 특성 | 12인의 봉인 중 핵심 봉인 3개 집중, 봉인자 후손 마을 |
| 핵심 | 챕터 6 씬 6-4~6-6 무대, 기억 파괴 수련 |
| 던전 | 봉인의 균열 (엘리트 던전) |
| 연결 | 표류자의 군도 ↔ 심연의 해구 |

### Zone 5: 심연의 해구 (Abyssal Trench)

| 항목 | 내용 |
|------|------|
| 레벨 | 80+ (레이드 전용) |
| 특성 | 안개해 최심부, 레테 잔류 의식의 핵심 |
| 핵심 | 12인 레이드 보스 — 레테의 현신 |
| 던전 | 심연 레이드 (12인 레이드) |
| 연결 | 봉인의 첨탑 → (편도, 레이드 후 귀환) |
